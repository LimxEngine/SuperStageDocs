# 第一章 编程器（Programmer）

## 1.1 什么是编程器

编程器是灯光控台中最核心的概念之一。它是一个**临时值缓冲区**，用来暂存用户正在编辑的灯具属性值。

### 编程器的特点

- 通过编码器轮调整的值会自动进入编程器
- 编程器中的值以**红色**显示在灯具表中，表示"已修改但未存储"
- 编程器中的值具有**最高优先级** — 它会覆盖 CUE 回放的输出
- 将编程器内容"存储"（Store）后，值被保存到 CUE 或预设中
- 执行 **Clear** 操作后，编程器中的值被清除，灯具回到之前的 CUE 回放状态

### 编程器的工作流

```
选择灯具 → 调整编码器 → 值进入编程器（红色） → 存储为 CUE → 清除编程器
                                    ↓
                            实时输出到灯具（最高优先级）
```

## 1.2 编程器操作

### 清除编程器（Clear）

| 操作方式 | 作用范围 |
|----------|----------|
| 点击控制栏的 **Clear** 按钮 | 清除**选中灯具**在编程器中的所有值 |
| 无灯具被选中时点击 **Clear** | 清除**所有灯具**在编程器中的值 |

清除后，灯具的属性值回到：
1. 正在运行的 CUE 回放值（如果有）
2. 默认值（如果没有 CUE 在运行）

### 激活（Activate）

点击控制栏的 **Activate** 按钮 → 将选中灯具当前的 DMX 输出值"读取"到编程器中。

**用途**：当你想基于当前 CUE 回放状态做小幅调整时，先 Activate 捕获当前值，再微调。

## 1.3 编程器与 DMX 输出的优先级

控台同时有多个"值来源"，它们的优先级从高到低为：

| 优先级 | 来源 | 说明 |
|--------|------|------|
| **1（最高）** | 编程器（Programmer） | 用户正在编辑的值 |
| **2** | 帧效果（Frame Effect） | 正在运行的动态效果 |
| **3** | CUE 回放（Playback） | 正在运行的 CUE |
| **4（最低）** | 默认值 | 灯具的出厂默认值 |

> **提示**：编程器中有值时，CUE 回放的值不会对该灯具的该属性生效。记得编程完成后 Clear 编程器。

# 第二章 编码器栏（Encoder Bar）

## 2.1 编码器栏概览

编码器栏位于控台主界面的最底部，是日常编程中使用最频繁的区域。

### 布局结构

```

│ 功能按钮区    编码器轮区域                                    快捷按钮区   │

│ │Mode  │   │Dimmer │ │  Pan  │ │ Tilt  │ │ Zoom  │      │ Full │      │
│ │Value │   │  50%  │ │ 180°  │ │  45°  │ │  30°  │      │ Zero │      │
│ ├──────┤   │  ◎   │ │  ◎   │ │  ◎   │ │  ◎   │      │Highlt│      │
│ │Prec. │   └───────┘ └───────┘ └───────┘ └───────┘      └──────┘      │
```

## 2.2 属性标签栏

属性标签栏位于编码器轮上方，用于切换当前显示的属性组。

### 属性分类

| 标签 | 包含的典型属性 | 颜色 |
|------|---------------|------|
| **Dimmer** | Dimmer（亮度） | 黄色 |
| **Color** | Red、Green、Blue、White、Amber、UV、CTC、ColorWheel | 品红色 |
| **Position** | Pan、Tilt、Pan Fine、Tilt Fine | 绿色 |
| **Gobo** | Gobo1、Gobo2、Gobo1Rotation、Gobo2Rotation | 橙色 |
| **Beam** | Zoom、Focus、Iris | 蓝色 |
| **Control** | Shutter、Strobe、Prism、Frost | 白色 |

### 动态属性发现

- 属性标签栏中**只显示选中灯具实际拥有的属性分类**
- 如果选中的灯具没有 Gobo 属性，则 Gobo 标签不会出现
- 切换选择不同类型的灯具后，标签栏会自动更新

### 切换属性组

- **单击**某个标签 → 编码器轮区域切换为该分类下的属性
- 当前激活的标签以高亮颜色显示

## 2.3 编码器轮（Encoder Wheel）

编码器轮是控台中调整灯具属性值的核心控件。

### 轮的组成

每个编码器轮从上到下包含：

| 区域 | 内容 |
|------|------|
| **属性名称** | 显示属性名称（如 "Dimmer"、"Pan"） |
| **当前值** | 显示当前属性值（如 "50%"、"180°"） |
| **编码器圆盘** | 可拖拽的圆形控件，拖拽改变值 |
| **单位** | 属性的单位（%、°等） |

### 值显示状态

| 显示样式 | 含义 |
|----------|------|
| 白色数字 | 值来自 CUE 回放或默认值 |
| **红色数字** | 值来自编程器（已修改） |
| **"Mixed"** | 选中的多个灯具该属性值不同 |
| **灰色 "--"** | 选中灯具不支持此属性 |

### 拖拽操作

- **上下拖拽**编码器圆盘 → 调整属性值
  - 向上拖拽 = 增大值
  - 向下拖拽 = 减小值
- 拖拽速度决定调整幅度
- 粗调/精调模式影响步进量

### 精确输入

- **双击**编码器的值区域 → 弹出数字输入弹窗
- 直接输入精确数值，按 Enter 确认

## 2.4 编码器模式（Mode）

编码器模式决定了编码器轮调整的是属性的什么参数。

### 模式列表

| 模式 | 简写 | 编码器轮调整内容 | 用途 |
|------|------|------------------|------|
| **Value** | Val | 属性的当前值（如亮度百分比） | 日常编程 |
| **Fade In** | FIn | 属性的入渐变时间（秒） | 设置 CUE 激活时的渐变时间 |
| **Delay In** | DIn | 属性的入延时时间（秒） | 设置 CUE 激活前的等待时间 |
| **Fade Out** | FOut | 属性的出渐变时间（秒） | 设置 CUE 释放时的渐变时间 |
| **Delay Out** | DOut | 属性的出延时时间（秒） | 设置 CUE 释放前的等待时间 |

### 切换模式

- 点击左侧功能按钮区的 **Mode** 按钮
- 每次点击在 5 种模式间循环切换
- 当前模式名称显示在按钮上

### 时间模式详解

当编码器处于 Fade In / Delay In / Fade Out / Delay Out 任一模式时：

- 编码器轮显示的不再是属性值，而是**时间值**（单位：秒）
- 拖拽编码器轮 → 调整时间
- 双击值区域 → 弹出时间输入弹窗，支持表达式输入

### 时间表达式

在时间输入弹窗中，支持以下格式：

| 表达式 | 含义 | 效果 |
|--------|------|------|
| `2` 或 `2s` | 2 秒 | 所有选中灯具使用相同的 2 秒时间 |
| `0 Thru 2` 或 `0-2` | 0 到 2 秒线性分布 | 第一个灯具 0 秒，最后一个灯具 2 秒，中间线性分布 |
| `2 Thru 0` 或 `2-0` | 2 到 0 秒线性分布 | 第一个灯具 2 秒，最后一个灯具 0 秒 |
| `0 Thru 2 Thru 0` 或 `0-2-0` | 三角分布 | 中间灯具 2 秒，两端 0 秒 |
| `2 Thru 0 Thru 2` 或 `2-0-2` | 倒三角分布 | 中间灯具 0 秒，两端 2 秒 |

**示例**：选中 5 个灯具，输入 `0 Thru 4`
```
灯具 1 → 0.0 秒
灯具 2 → 1.0 秒
灯具 3 → 2.0 秒
灯具 4 → 3.0 秒
灯具 5 → 4.0 秒
```

## 2.5 编码器精度（Precision）

| 精度模式 | 简写 | 步进量 | 用途 |
|----------|------|--------|------|
| **Coarse**（粗调） | C | 正常步进 | 快速调整到大致值 |
| **Fine**（精调） | F | 细步进（约 1/10） | 精确微调 |

### 切换精度

- 点击左侧功能按钮区的 **Precision** 按钮
- 在 Coarse 和 Fine 之间切换
- 当前精度显示在按钮上

## 2.6 发散模式（Spread）

发散模式用于给多个选中灯具设置**不同的值**，而不是统一值。这是 MA 控台中 Align 功能的对应实现。

### 发散模式列表

| 模式 | 图标 | 效果 |
|------|------|------|
| **Off** | — | 所有灯具设置相同的值（默认） |
| **Right** | > | 从第一个灯具到最后一个灯具线性递增 |
| **Left** | < | 从第一个灯具到最后一个灯具线性递减 |
| **Center Out** | >< | 从中心向两侧递增 |
| **Out Center** | <> | 从两侧向中心递增 |

### 切换发散模式

- 点击左侧功能按钮区的 **Spread** 按钮
- 每次点击在 5 种模式间循环切换
- 当前模式图标显示在按钮上

### 发散模式详解

**示例**：选中灯具 1-5，拖拽 Dimmer 编码器到 100%

**Off 模式**：
```
灯具 1: 100%    灯具 2: 100%    灯具 3: 100%    灯具 4: 100%    灯具 5: 100%
```

**Right (>) 模式**：
```
灯具 1: 0%      灯具 2: 25%     灯具 3: 50%     灯具 4: 75%     灯具 5: 100%
```

**Left (<) 模式**：
```
灯具 1: 100%    灯具 2: 75%     灯具 3: 50%     灯具 4: 25%     灯具 5: 0%
```

**Center Out (><) 模式**：
```
灯具 1: 0%      灯具 2: 50%     灯具 3: 100%    灯具 4: 50%     灯具 5: 0%
```

**Out Center (<>) 模式**：
```
灯具 1: 100%    灯具 2: 50%     灯具 3: 0%      灯具 4: 50%     灯具 5: 100%
```

> **提示**：发散模式与选择工具的 Block/Group/Wings 参数配合使用，可以创造出更复杂的值分布效果。

## 2.7 快捷按钮区（右侧）

编码器栏右侧提供以下快捷按钮：

| 按钮 | 功能 |
|------|------|
| **Full** | 将选中灯具的 Dimmer 设为 100% |
| **Zero** | 将选中灯具的 Dimmer 设为 0% |
| **Highlight** | 高亮选中灯具（临时覆盖，不写入编程器） |

# 第三章 颜色选择器面板（Color Picker）

## 3.1 面板概览

颜色选择器面板提供直观的颜色选取界面，用于快速设置灯具的 RGB 颜色属性。

## 3.2 颜色选取方式

### HSV 颜色轮

面板中心是一个 HSV 颜色轮：

- **拖拽圆环外圈** → 选择色相（Hue）
- **在圆环内部的三角/正方形区域拖拽** → 调整饱和度（Saturation）和明度（Value）
- 所选颜色实时应用到选中灯具的 Red/Green/Blue 属性

### RGB 滑条

颜色轮下方提供 R/G/B 三个独立滑条：

| 滑条 | 范围 | 说明 |
|------|------|------|
| **R（红）** | 0 - 255 | 拖拽或输入精确值 |
| **G（绿）** | 0 - 255 | 拖拽或输入精确值 |
| **B（蓝）** | 0 - 255 | 拖拽或输入精确值 |

### 颜色预设

面板底部显示颜色预设色块：

- 包含常用颜色（红、橙、黄、绿、青、蓝、紫、白、暖白等）
- **单击色块** → 直接应用该颜色
- 可自定义颜色预设

## 3.3 高级颜色属性

如果灯具支持以下属性，颜色选择器会同步控制：

| 属性 | 说明 |
|------|------|
| **Red / Green / Blue** | 基本 RGB 混色 |
| **White** | 白色 LED（部分灯具） |
| **Amber** | 琥珀色 LED |
| **UV** | 紫外线 LED |
| **CTC** | 色温校正 |
| **ColorWheel** | 色轮位置 |

> **提示**：颜色选择器修改的值同样会进入编程器，需要 Clear 后才能恢复之前状态。

# 第四章 预设系统（Preset）

## 4.1 什么是预设

预设（Preset）是一组属性值的快照，可以快速应用到选中灯具。与 CUE 不同，预设通常**只保存一类属性**，更加轻量和灵活。

## 4.2 预设分类

| 分类 | 保存的属性 | 典型用途 |
|------|-----------|----------|
| **Dimmer** | Dimmer | 保存亮度值（如"半亮"、"全亮"） |
| **Position** | Pan、Tilt | 保存灯具指向某个位置（如"舞台中心"、"乐队区"） |
| **Gobo** | Gobo1、Gobo2、GoboRot | 保存图案轮选择（如"星空"、"火焰"） |
| **Color** | Red、Green、Blue、White、Amber、UV、CTC | 保存颜色方案（如"暖白"、"深蓝"、"日落橙"） |
| **Beam** | Zoom、Iris、Prism、Frost | 保存光束参数（如"窄光"、"宽光"、"柔光"） |
| **Focus** | Focus | 保存聚焦参数 |
| **Control** | Lamp、Reset、Speed | 保存灯具控制参数 |
| **Shapers** | Blade1、Blade2、Blade3、Blade4 | 保存切割叶片位置 |

## 4.3 预设面板

### 面板布局

```

│ [All] [Dimmer] [Position] [Gobo] [Color] [Beam] [Focus] [Control] [Shapers] ← 分类标签 │
├────────────────────────────────────────────────────────────┤
│ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐  │
│ │暖白  │ │深蓝  │ │日落橙│ │      │ │      │ │      │  │
│ └──────┘ └──────┘ └──────┘ └──────┘ └──────┘ └──────┘  │
│ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐  │
│ │      │ │      │ │      │ │      │ │      │ │      │  │
│ └──────┘ └──────┘ └──────┘ └──────┘ └──────┘ └──────┘  │
│                                             ← 预设网格    │
└────────────────────────────────────────────────────────────┘
```

### 分类标签

- 点击不同的分类标签切换显示对应类型的预设
- **All** 标签显示所有分类的预设
- 分类对应关系：Dimmer（亮度）、Position（位置）、Gobo（图案）、Color（颜色）、Beam（光束）、Focus（聚焦）、Control（控制）、Shapers（切割）

### 预设按钮

- 已保存的预设显示为带名称的彩色按钮
- 空槽位显示为深灰色空白按钮

## 4.4 存储预设

### 操作步骤

1. **选择灯具**并将属性调整到期望状态（值进入编程器）
2. 打开预设面板
3. 切换到目标分类标签（如 Color）
4. **右键点击**一个空槽位
5. 选择 **Store**（存储）
6. 在弹出对话框中输入预设名称
7. 点击确认

### 存储规则

- 预设只存储当前分类对应的属性值
  - 例如：在 Color 分类下存储，只会保存 Red/Green/Blue/White 等颜色属性
- 预设存储的是编程器中**所有选中灯具**的属性值

## 4.5 应用预设

### 基本应用

1. 先选中目标灯具
2. **单击**预设按钮
3. 预设中保存的属性值被应用到选中灯具
4. 值进入编程器（显示为红色）

### 应用规则

- 预设只会修改它所包含的属性
  - 例如：应用 Color 预设不会影响灯具的 Position 或 Dimmer
- 如果选中的灯具没有预设中的某个属性，该属性被自动忽略

## 4.6 更新预设

### 操作步骤

1. 将灯具属性调整到新的期望状态
2. **右键点击**要更新的预设按钮
3. 选择 **Update**（更新）
4. 预设内容被替换为当前编程器中的值

## 4.7 删除预设

1. **右键点击**预设按钮
2. 选择 **Delete**（删除）
3. 在确认对话框中点击确认
4. 预设被删除，槽位变回空白

## 4.8 重命名预设

1. **右键点击**预设按钮
2. 选择 **Rename**（重命名）
3. 输入新名称
4. 按 Enter 确认

## 4.9 复制预设

1. **右键点击**预设按钮
2. 选择 **Copy**（复制）
3. **右键点击**目标空槽位
4. 选择 **Paste**（粘贴）
5. 预设被复制到新槽位

## 4.10 预设使用技巧

### 颜色预设库建设

建议在项目初期建立一套标准颜色预设库：

| 预设名称 | RGB 参考值 | 用途 |
|----------|-----------|------|
| Warm White | R255 G200 B150 | 暖白基调 |
| Cool White | R200 G220 B255 | 冷白基调 |
| Deep Blue | R0 G50 B255 | 深蓝夜景 |
| Sunset Orange | R255 G100 B30 | 日落暖调 |
| Forest Green | R30 G200 B50 | 森林绿 |
| Rose Pink | R255 G100 B150 | 玫瑰粉 |
| Congo Blue | R0 G0 B180 | 刚果蓝 |

### 位置预设库建设

建议按演区划分位置预设：

| 预设名称 | 说明 |
|----------|------|
| Center Stage | 舞台中心 |
| Down Stage Left | 台口左侧 |
| Down Stage Right | 台口右侧 |
| Up Stage Center | 台口后方中心 |
| Band Area | 乐队区域 |
| Audience | 观众席 |
