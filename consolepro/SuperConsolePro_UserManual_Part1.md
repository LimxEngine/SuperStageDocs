# SuperConsolePro 灯光控台 — 用户手册（一）

## 概述 · 快速入门 · 界面布局 · 窗口管理

---

# 第一章 概述

## 1.1 什么是 SuperConsolePro

SuperConsolePro 是一套内置于 Unreal Engine 编辑器的 **专业级灯光控台系统**。它让灯光设计师能够在虚幻引擎的 3D 场景中，像操作真实物理控台（如 MA3、ETC Eos）一样完成：

- 灯具配接（Patch）
- 灯具选择与分组
- 属性编程（亮度、颜色、位置、光束等）
- CUE（场景记忆）的录制与回放
- 帧效果（追光、彩虹跑马灯等动态效果）编辑
- 时间线编排与音乐同步
- DMX 实时输出
- 演出文件的保存与管理

所有操作均通过鼠标和键盘在编辑器窗口内完成，无需外部硬件控台。

## 1.2 适用场景

| 场景 | 说明 |
|------|------|
| 虚拟制作 | 在 VP 阶段预演灯光效果，提前验证灯位与编程 |
| 演出预可视化 | 在 3D 环境中模拟真实演出灯光，与舞美方案联动 |
| 灯光教学 | 零成本学习专业控台操作流程 |
| 影视灯光预览 | 配合虚幻引擎的物理光照系统预览灯光效果 |
| 实时演出 | 通过 DMX 输出驱动真实灯具进行现场演出控制 |

## 1.3 核心概念速览

在开始使用之前，请先了解以下核心概念：

### Patch（配接）
将场景中的 SuperDMX 灯具"导入"到控台中，为每个灯具分配一个 **Fixture ID**。只有被配接的灯具才能被控台选择和控制。

### Selection（选择）
当前被选中的灯具集合。编码器和编程器只对选中的灯具生效。

### Programmer（编程器）
一个"临时暂存区"。通过编码器调整的值会先进入编程器，显示为红色高亮。编程器中的值尚未"保存"，关闭控台后会丢失。

### CUE（场景记忆）
将编程器中的内容"快照"保存下来，就形成了一个 CUE。CUE 包含灯具的属性值以及渐变/延时时间参数。多个 CUE 可组织在一个 **CUE List** 中按顺序回放。

### Preset（预设）
类似 CUE，但更轻量。预设通常只保存某一类属性（如颜色、位置），用于快速调用常用设置。

### Frame Effect（帧效果）
基于波形的动态效果引擎（对标 MA3 Phaser）。可以生成追光、彩虹、频闪等周期性效果。

### DMX
灯光行业标准通信协议。控台通过 DMX 将控制信号发送给灯具。每个 DMX **Universe**（域）包含 512 个通道。

### Show File（演出文件）
保存所有配接、CUE、预设、时间线、窗口布局等数据的文件（.ssshow 格式），相当于控台的"项目文件"。

# 第二章 快速入门

## 2.1 打开控台

1. 在 Unreal Editor 中，点击顶部工具栏的 **SuperConsolePro** 按钮（或通过菜单 **Window → SuperConsolePro**）
2. 控台主窗口将以独立面板形式打开

## 2.2 五分钟上手流程

以下是从零开始完成一次完整灯光编程的最短路径：

### 第 1 步：配接灯具

1. 在控台窗口中打开 **设置面板**（Settings）
2. 切换到 **Patch** 标签页
3. 点击 **扫描场景**（Scan Scene）按钮
4. 场景中所有 SuperDMX 灯具将出现在列表中
5. 勾选想要控制的灯具，点击 **导入选中**（Import Selected）
6. 灯具出现在控台中，可以被选择和控制了

### 第 2 步：选择灯具

1. 打开 **灯具表面板**（Fixture Sheet）
2. 单击灯具行 → 选中单个灯具
3. 按住 Ctrl 单击 → 多选
4. 按住 Shift 单击 → 范围选择
5. 选中的灯具会以蓝色高亮显示

### 第 3 步：调整属性

1. 查看底部的 **编码器栏**（Encoder Bar）
2. 编码器栏会自动显示选中灯具的可用属性
3. 上下拖拽编码器轮 → 调整属性值
4. 属性标签栏中点击不同分类（Dimmer、Color、Position 等）可切换显示的属性组

### 第 4 步：存储 CUE

1. 打开 **回放面板**（Playback）
2. 右键点击一个空的执行器插槽
3. 选择 **Store**（存储）
4. 编程器中的值被保存为一个 CUE

### 第 5 步：回放 CUE

1. 在回放面板中，点击已存储的 CUE 按钮
2. CUE 被激活，灯具输出设定的效果
3. 再次点击可关闭（Toggle 模式下）

### 第 6 步：保存演出

1. 打开设置面板 → **Show** 标签页
2. 点击 **保存**（Save）
3. 所有数据被保存到 .ssshow 文件
4. 下次打开控台时会自动加载上次的演出文件

# 第三章 界面布局

## 3.1 主界面结构

控台主界面由以下区域组成（从外到内）：

```
┌────────────────────────────────────────────────────────────────┐
│  控制栏 (Control Bar)                        标题栏 (Title)    │
│  ┌──────┐                                              ┌────┐ │
│  │      │                                              │视图│ │
│  │ 控制 │         网格窗口管理器                          │ 栏 │ │
│  │  栏  │      (Grid Window Manager)                   │    │ │
│  │      │         16列 × 8行 网格                       │View│ │
│  │      │       各种面板在此显示                          │Bar │ │
│  │      │                                              │    │ │
│  └──────┘                                              └────┘ │
│  ┌──────────────────────────────────────────────────────────┐ │
│  │  命令行 (Command Line)                                    │ │
│  ├──────────────────────────────────────────────────────────┤ │
│  │  编码器栏 (Encoder Bar)                                   │ │
│  └──────────────────────────────────────────────────────────┘ │
└────────────────────────────────────────────────────────────────┘
```

### 3.1.1 控制栏（Control Bar）— 左侧

位于主界面的最左侧，可以折叠/展开。提供以下快捷操作按钮：

| 按钮 | 功能 |
|------|------|
| **Clear** | 清除选中灯具在编程器中的所有值。不影响正在运行的 CUE 输出 |
| **Highlight** | 将选中灯具临时设为高亮值（Dimmer=100%，其他属性为高亮预设值），用于在场景中快速定位灯具。再次点击取消高亮 |
| **Full** | 将选中灯具的 Dimmer 设为 100%（写入编程器） |
| **Zero** | 将选中灯具的 Dimmer 设为 0%（写入编程器） |
| **Default** | 将选中灯具的所有属性恢复为出厂默认值（写入编程器） |
| **Activate** | 将选中灯具当前的 DMX 输出值全部读入编程器（相当于"捕获当前状态"） |

**使用提示**：
- **Clear** 是最常用的操作。编程完一个 CUE 后，记得 Clear 编程器，再选择其他灯具进行下一个 CUE 的编程
- **Highlight** 不会影响编程器，是非破坏性操作。松开后灯具恢复原来的状态

### 3.1.2 标题栏（Title Bar）— 顶部

显示当前演出文件名称和系统状态信息。

### 3.1.3 网格窗口管理器（Grid Window Manager）— 中央

控台界面的核心区域。采用 **16 列 × 8 行** 的网格布局，各种功能面板以窗口形式摆放在网格中。

详细操作见本章 **3.2 窗口管理**。

### 3.1.4 视图栏（View Bar）— 右侧

位于主界面的最右侧，可以折叠/展开。用于管理**视图布局预设**，即记住当前窗口的排列方式，一键切换不同的工作界面。

详细操作见本章 **3.3 视图管理**。

### 3.1.5 命令行（Command Line）— 底部

位于编码器栏上方。用于输入文本命令（高级功能，日常使用中不常用到）。

### 3.1.6 编码器栏（Encoder Bar）— 最底部

控台的核心交互区域，详见 **用户手册（三）编程器与编码器** 章节。

## 3.2 窗口管理

### 3.2.1 添加窗口

在网格窗口管理器中，点击任意**空白格子**，将弹出"添加窗口"对话框。对话框以 3 列网格展示所有可用的面板类型：

| 面板类型 | 说明 |
|----------|------|
| **Fixture Sheet**（灯具表） | 以表格形式显示所有已配接灯具及其属性值 |
| **Playback**（回放） | CUE 执行器按钮网格，用于触发和管理 CUE |
| **Preset**（预设） | 预设池，按分类显示和管理预设 |
| **Groups**（灯具组） | 灯具组管理面板 |
| **Color Picker**（颜色选择器） | HSV 颜色轮，快速设置灯具颜色 |
| **Layout**（布局视图） | 2D 灯具布局编辑器 |
| **Frame Editor**（帧效果编辑器） | 波形效果编辑器 |
| **Frame Presets**（帧预设） | 帧效果预设库 |
| **Timecode**（时间线） | 多轨时间线编辑器 |
| **Running Playbacks**（运行中的回放） | 显示当前正在运行的所有 CUE/效果 |
| **DMX**（DMX 输出） | DMX Universe 管理和输出开关 |
| **Settings**（设置） | 系统设置（演出文件、配接、DMX 配置） |
| **Show File**（演出文件） | 演出文件管理（打开、保存、新建） |
| **Patch**（配接） | 灯具配接管理面板 |

选择面板类型后，窗口将被创建在你点击的格子位置。

### 3.2.2 移动窗口

- 将鼠标放在窗口的**标题栏**上
- 按住左键拖拽，窗口跟随鼠标移动
- 松开鼠标，窗口吸附到新的网格位置

### 3.2.3 调整窗口大小

- 将鼠标放在窗口的**右下角边缘**
- 光标变为调整大小形状
- 按住左键拖拽，改变窗口占据的网格数量
- 窗口最小为 1×1 格，最大可占满整个网格（16×8）

### 3.2.4 关闭窗口

- 点击窗口标题栏右侧的 **× 按钮**
- 窗口被关闭，对应的网格区域变回空白

### 3.2.5 窗口层级

- 当多个窗口重叠时，点击某个窗口会将它提升到最前面
- 建议合理规划窗口大小和位置，避免重叠

## 3.3 视图管理（View Bar）

视图栏位于主界面右侧，提供视图布局的快速管理功能。

### 3.3.1 内置预设视图

系统内置了以下预设视图布局：

| 预设名称 | 包含面板 | 适用场景 |
|----------|----------|----------|
| **Default** | 灯具表 + 回放 + 编码器栏 | 通用编程 |
| **Programming** | 灯具表 + 颜色选择器 + 预设 + 回放 | 深度编程 |
| **Playback** | 回放（大面积）+ 运行中的回放 | 演出回放 |
| **Timeline** | 时间线（大面积）+ 回放 | 时间线编排 |

### 3.3.2 加载视图

- 在视图栏中**单击**某个视图按钮
- 窗口管理器立即切换到该视图的窗口布局

### 3.3.3 保存自定义视图

1. 先将窗口排列成想要的布局
2. 在视图栏中**右键点击**一个视图槽位
3. 选择 **Save**（保存）
4. 当前窗口布局被保存到该槽位
5. 可选择重命名视图

### 3.3.4 删除 / 重命名视图

- **右键点击**已保存的视图按钮
- 选择 **Rename**（重命名）输入新名称
- 选择 **Delete**（删除）清空该槽位

### 3.3.5 清空屏幕

- 点击视图栏底部的 **Clear Screen** 按钮
- 关闭所有窗口，回到空白网格

## 3.4 可用面板类型详细说明

以下是控台中所有可用面板的功能概要。每个面板的详细操作方法将在后续手册章节中逐一说明。

### 灯具表面板（Fixture Sheet）
以电子表格形式展示所有已配接灯具。每行一个灯具（矩阵灯具可展开为子灯具行），列显示灯具的各个属性值。支持点击选择灯具、排序、搜索筛选。

### 回放面板（Playback）
以按钮网格形式显示 CUE 执行器。每个按钮对应一个 CUE。左键点击触发 CUE，右键点击打开菜单（编辑、删除、绑定快捷键等）。

### 预设面板（Preset）
预设池。顶部有分类标签（All / Position / Color / Beam / General），点击切换显示对应类型的预设。左键点击应用预设，右键保存到空槽位。

### 灯具组面板（Groups）
以按钮网格展示灯具组。点击灯组按钮 → 选中该灯组中的所有灯具。右键创建新灯组或管理现有灯组。

### 颜色选择器面板（Color Picker）
HSV 颜色轮 + 色条 + 数值输入。拖拽颜色轮选取色相和饱和度，拖拽色条调整明度。所选颜色实时应用到选中灯具的 RGB 属性。底部显示颜色预设色块。

### 布局视图面板（Layout）
2D 俯视图，展示灯具在舞台上的物理位置。支持拖拽灯具调整位置、框选灯具、缩放平移画布。

### 帧效果编辑器面板（Frame Editor）
波形效果的可视化编辑器。左侧工具栏选择波形类型、调整参数；中间显示实时波形预览；右侧快捷参数面板。

### 帧预设面板（Frame Presets）
帧效果预设库。包含系统内置预设和用户自定义预设。点击预设即可应用到当前选中灯具。

### 时间线面板（Timecode）
多轨时间线编辑器。支持 CUE 轨道和音频轨道。可将 CUE 编排到时间线上按时间自动播放。

### 运行中的回放面板（Running Playbacks）
实时显示当前所有正在运行的 CUE 和效果，包括名称、类型、渐变进度、强度等信息。

### DMX 面板（DMX）
DMX 输出管理。包含全局 DMX 输出开关和各 Universe 的独立启用/禁用控制。

### 设置面板（Settings）
系统设置，包含三个标签页：
- **Show**（演出）：新建/保存/另存为演出文件
- **Patch**（配接）：灯具配接管理
- **DMX**：DMX 输出配置

### 演出文件面板（Show File）
显示演出文件列表，支持新建、打开、保存、另存为、删除、重命名演出文件。

### 配接面板（Patch）
灯具配接专用面板。显示已导入和未导入的灯具列表，支持扫描场景、导入、移除、筛选、搜索。

# 第四章 通用 UI 控件说明

## 4.1 控台按钮（Console Button）

控台中的按钮有多种样式变体：

| 变体 | 外观 | 用途 |
|------|------|------|
| **Default**（默认） | 深灰色背景 | 普通操作按钮 |
| **Primary**（主要） | 蓝色背景 | 主要操作（如确认、应用） |
| **Secondary**（次要） | 暗灰色背景，灰色文字 | 辅助操作 |
| **Success**（成功） | 绿色背景 | 正在激活/运行的状态 |
| **Warning**（警告） | 黄色背景，黑色文字 | 需要注意的操作 |
| **Danger**（危险） | 红色背景 | 删除、清除等危险操作 |
| **Ghost**（透明） | 透明背景 | 工具栏中的小按钮 |

按钮有三种尺寸：
- **Small**（小号）：用于工具栏中的紧凑按钮
- **Medium**（中号）：默认尺寸
- **Large**（大号）：用于主要操作按钮

## 4.2 数字输入弹窗

双击编码器的数值区域，会弹出数字输入弹窗。支持：

- 直接输入数字（如 `50`）
- 输入百分比（如 `50%`）
- 输入时间值（如 `2s` 表示 2 秒）
- 输入时间表达式（如 `0 Thru 2` 表示从 0 秒到 2 秒的线性分布）
- 按 **Enter** 确认，按 **Esc** 取消

## 4.3 右键菜单

控台中许多元素支持右键菜单操作。常见的右键菜单场景：

- **回放面板的空槽位** → Store（存储 CUE）
- **回放面板的已有 CUE** → Edit / Delete / Rename / 设置触发模式 / 绑定快捷键
- **预设面板的空槽位** → Store（存储预设）
- **预设面板的已有预设** → Update / Delete / Rename
- **灯具组面板的空槽位** → Store Group（保存当前选择为灯组）
- **灯具组面板的已有灯组** → Update / Delete / Rename / 设置颜色
- **视图栏的槽位** → Save / Rename / Delete

## 4.4 确认对话框

进行危险操作（如删除 CUE、新建演出、覆盖文件）时，系统会弹出确认对话框：

- **标题** 描述操作内容
- **消息** 说明操作的后果
- **确定** 按钮执行操作
- **取消** 按钮放弃操作
