# SuperConsolePro 灯光控台 — 用户手册（二）

## 灯具配接 · 灯具选择 · 灯具组 · 灯具表

# 第一章 灯具配接（Patch）

## 1.1 什么是配接

"配接"（Patch）是将场景中放置的 SuperDMX 灯具**导入到控台**中的过程。只有被配接的灯具才能被控台选中、编程和控制。

配接过程会为每个灯具分配以下信息：

| 信息项 | 说明 |
|--------|------|
| **Fixture ID** | 灯具编号，控台内唯一标识。例如 `1`、`2`、`101` |
| **灯具名称** | 灯具的显示名称，通常自动读取场景中 Actor 的名称 |
| **灯具类型** | 灯具型号（如 "Wash 600"、"Spot 400 Pro" 等） |
| **Universe** | DMX 域编号（1-64），表示灯具所在的 DMX 总线 |
| **起始通道** | 灯具在该 Universe 中的起始 DMX 通道（1-512） |
| **通道跨度** | 灯具占用的 DMX 通道数量 |
| **模块数量** | 矩阵灯具的子灯具数量。单灯具为 1 |

## 1.2 打开配接面板

有两种方式进入配接面板：

1. **设置面板** → 切换到 **Patch** 标签页
2. 在网格窗口管理器中直接添加 **Patch** 面板

## 1.3 扫描场景

### 操作步骤

1. 打开配接面板
2. 点击 **扫描场景**（Scan Scene）按钮
3. 系统会自动搜索当前关卡中所有继承自 `ASuperDmxActorBase` 的灯具 Actor
4. 扫描完成后，**未配接灯具列表** 会显示所有发现的灯具

### 扫描结果列表

扫描结果分为两个区域：

- **未配接灯具**：场景中存在但尚未导入控台的灯具
- **已配接灯具**：已经导入控台的灯具

每个灯具条目显示：
- 灯具名称（场景中的 Actor 名称）
- 灯具类型
- Universe / 起始通道
- 模块数量

## 1.4 导入灯具

### 导入选中灯具

1. 在 **未配接灯具** 列表中，勾选想要导入的灯具
2. 点击 **导入选中**（Import Selected）按钮
3. 系统自动为灯具分配 Fixture ID
4. 灯具出现在 **已配接灯具** 列表中

### 导入全部灯具

1. 点击 **全部导入**（Import All）按钮
2. 所有未配接灯具一次性导入控台

### Fixture ID 分配规则

- 系统自动分配从 **1** 开始的连续编号
- 如果已有灯具占用了某些编号，新灯具会自动跳过已占用的编号
- 导入后可手动修改 Fixture ID

## 1.5 移除灯具

### 移除选中灯具

1. 在 **已配接灯具** 列表中，勾选想要移除的灯具
2. 点击 **移除选中**（Remove Selected）按钮
3. 系统弹出确认对话框
4. 确认后，灯具从控台中移除

> **注意**：移除灯具不会删除场景中的 Actor，仅仅是从控台的配接列表中移除。

### 移除全部灯具

1. 点击 **全部移除**（Remove All）按钮
2. 确认后，清空所有配接数据

> **警告**：移除灯具会同时清除该灯具相关的所有编程器数据、CUE 数据和预设数据。此操作不可撤销。

## 1.6 场景同步

当场景中的灯具发生变化（添加、删除、修改属性）时，配接数据需要同步更新：

### 自动同步

系统会在以下时机自动检测场景变化：
- 打开配接面板时
- 手动点击 **刷新**（Refresh）按钮时

### 同步状态指示

| 状态 | 颜色 | 说明 |
|------|------|------|
| **已同步** | 绿色 | 灯具数据与场景一致 |
| **已变更** | 黄色 | 灯具属性发生了变化，需要重新同步 |
| **已丢失** | 红色 | 场景中找不到该灯具（Actor 已被删除） |

### 处理已变更的灯具

1. 在已配接灯具列表中，黄色标记的灯具表示其场景属性发生了变化
2. 点击 **同步变更**（Sync Changes）按钮
3. 控台自动更新配接数据

### 处理已丢失的灯具

1. 红色标记的灯具表示场景中已经找不到对应的 Actor
2. 可以选择 **保留**（保持配接数据但不再输出 DMX）
3. 也可以选择 **移除**（从配接列表中删除）

## 1.7 配接信息编辑

双击已配接灯具的某个字段，可以手动编辑：

| 字段 | 可编辑 | 说明 |
|------|--------|------|
| Fixture ID | 是 | 输入新的编号。不能与现有编号冲突 |
| 名称 | 是 | 修改灯具显示名称 |
| Universe | 只读 | 由场景中 Actor 的设置决定 |
| 起始通道 | 只读 | 由场景中 Actor 的设置决定 |
| 灯具类型 | 只读 | 由场景中使用的灯具资产决定 |

## 1.8 搜索与筛选

配接面板提供以下筛选功能：

- **搜索框**：输入关键字筛选灯具名称或类型
- **状态筛选**：可单独显示"已同步"/"已变更"/"已丢失"状态的灯具
- **排序**：点击列头按 Fixture ID、名称、类型、Universe 等排序

## 1.9 配接统计

面板底部显示配接统计信息：

- **已配接灯具总数**
- **占用的 Universe 数量**
- **占用的 DMX 通道总数**

# 第二章 灯具选择

## 2.1 基本选择操作

### 单选

- 在灯具表面板中，**单击** 灯具行 → 选中该灯具
- 其他所有灯具被取消选中

### 多选（添加）

- 按住 **Ctrl** 键 + **单击** 灯具 → 将该灯具添加到当前选择
- 已选中的灯具不会被取消

### 范围选择

- 先选中一个灯具
- 按住 **Shift** 键 + **单击** 另一个灯具
- 两个灯具之间（包含两端）的所有灯具都被选中

### 切换选择

- 按住 **Ctrl** 键 + **单击** 已选中的灯具 → 取消选中该灯具

### 全选

- 按 **Ctrl + A** → 选中所有已配接灯具

### 清除选择

- 按 **Esc** 键 → 取消所有选择

### 反选

- 按 **Ctrl + I** → 选中当前未选中的灯具，取消当前已选中的灯具

> **注意**：反选功能计划在后续版本中实现，当前版本暂不可用。

## 2.2 子灯具选择（矩阵灯具）

对于矩阵灯具（包含多个子灯具/模块实例的灯具），选择行为如下：

- **选择主灯具** → 同时选中该灯具的所有子灯具
- **展开主灯具行** → 显示各子灯具行
- **单独选择子灯具** → 只选中特定的子灯具实例

> **提示**：在编码器中调整属性时，如果只选中了部分子灯具，只有被选中的子灯具会被修改。

## 2.3 通过灯具组选择

在灯具组面板中：
- **单击灯组按钮** → 选中该灯组包含的所有灯具（替换当前选择）
- **Ctrl + 单击灯组按钮** → 将灯组灯具添加到当前选择

## 2.4 通过布局视图选择

在布局视图面板中：
- **单击灯具图标** → 选中该灯具
- **框选** → 拖拽矩形框选多个灯具
- **Ctrl + 单击** → 添加到选择

## 2.5 选择顺序

灯具被选中的**顺序**非常重要！选择顺序会影响以下功能：

- **发散模式**（Spread）：编码器的发散值按选择顺序分配
- **帧效果相位**：相位角度按选择顺序分配
- **时间分布**：渐变/延时时间按选择顺序计算

例如，如果你先选灯具 1，再选灯具 5，最后选灯具 3，那么：
- 发散模式中，灯具 1 的序号为 0，灯具 5 为 1，灯具 3 为 2

# 第三章 选择工具

选择工具用于调整选中灯具的**排列顺序**和**分组方式**，从而影响发散和帧效果的表现。

## 3.1 选择工具参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| **Block** | 整数 | 1 | 块大小。将连续灯具视为一组，共享相同属性 |
| **Group** | 整数 | 1 | 分组数。将灯具平均分成若干组 |
| **Wings** | 整数 | 1 | 镜像分段数。1=无镜像，2=左右镜像 |

### Block（块）

- **Block = 1**（默认）：每个灯具独立
- **Block = 2**：每 2 个灯具被视为一组，组内灯具获得相同的值
- **Block = 3**：每 3 个灯具一组

**示例**：选中灯具 1-8，Block = 2
```
灯具:   1  2  3  4  5  6  7  8
分组:   [A  A] [B  B] [C  C] [D  D]
发散值: 0%    33%    66%    100%
```

### Group（分组）

- **Group = 1**（默认）：所有灯具在一个组中
- **Group = 2**：灯具被分成 2 个组，每组内独立分配序号

**示例**：选中灯具 1-8，Group = 2
```
灯具:   1  2  3  4  5  6  7  8
分组:   [组1: 1 2 3 4] [组2: 5 6 7 8]
发散值: 0% 33% 66% 100%  0% 33% 66% 100%
```

### Wings（镜像）

- **Wings = 1**（默认）：无镜像
- **Wings = 2**：灯具被分成两半，右半部分的顺序被翻转，形成镜像效果

**示例**：选中灯具 1-8，Wings = 2
```
灯具:   1  2  3  4  5  6  7  8
序号:   0  1  2  3  3  2  1  0  ← 右半镜像
发散值: 0% 33% 66% 100% 100% 66% 33% 0%
```

## 3.2 高级排列工具

除了 Block/Group/Wings 参数外，还提供以下排列工具：

### Shuffle（随机打乱）

将选中灯具的顺序随机打乱。适用于制造"不规则"效果。

### Reverse（反转）

将选中灯具的顺序反转。灯具 1 变成最后一个，最后一个变成第一个。

### Shift（偏移）

将选中灯具的顺序整体偏移指定步数。

- **Shift +1**：所有灯具向后移动一个位置，最后一个移到最前面
- **Shift -1**：所有灯具向前移动一个位置，第一个移到最后面

### Interleave（交错）

将选中灯具按指定步数进行交错排列。

- **Interleave 2**：奇数位灯具排在前面，偶数位灯具排在后面

**示例**：灯具 1-8，Interleave 2
```
原始顺序: 1  2  3  4  5  6  7  8
交错后:   1  3  5  7  2  4  6  8
```

# 第四章 灯具组

## 4.1 什么是灯具组

灯具组是一组灯具的"快捷方式"。将常用的灯具组合保存为灯具组后，只需点击一个按钮即可选中这些灯具，无需每次手动逐个选择。

## 4.2 灯具组面板

灯具组面板以按钮网格形式显示所有灯组：

- 每个按钮显示灯组名称和颜色标记
- 空槽位显示为灰色空白按钮

## 4.3 创建灯具组

### 方法一：通过选择创建

1. 先在灯具表中选中想要分组的灯具
2. 打开灯具组面板
3. **右键点击** 一个空槽位
4. 选择 **Store Group**（保存灯组）
5. 输入灯组名称
6. 点击确认

### 方法二：快捷操作

1. 选中灯具
2. 在灯具组面板中，**长按** 一个空槽位
3. 直接保存为灯组

## 4.4 使用灯具组

### 选中灯组

- **单击** 灯组按钮 → 选中灯组中的所有灯具（替换当前选择）
- **Ctrl + 单击** → 将灯组灯具添加到当前选择

### 快速双击

- **双击** 灯组按钮 → 选中灯组并自动切换到编程器模式

## 4.5 管理灯具组

### 更新灯组内容

1. 选中新的灯具组合
2. **右键点击** 目标灯组按钮
3. 选择 **Update**（更新）
4. 灯组内容更新为当前选择

### 重命名灯组

1. **右键点击** 灯组按钮
2. 选择 **Rename**（重命名）
3. 输入新名称
4. 按 Enter 确认

### 删除灯组

1. **右键点击** 灯组按钮
2. 选择 **Delete**（删除）
3. 在确认对话框中点击确认

### 设置灯组颜色

1. **右键点击** 灯组按钮
2. 选择 **Set Color**（设置颜色）
3. 在颜色选择器中选取颜色
4. 灯组按钮显示为新颜色，便于视觉区分

## 4.6 灯具组的用途

| 用途 | 说明 |
|------|------|
| **快速选择** | 一键选中常用灯具组合 |
| **编程分组** | 按区域/功能分组（如"面光"、"侧光"、"染色"） |
| **帧效果分组** | 将不同灯组应用不同的帧效果 |
| **CUE 编程** | 快速切换选择不同灯组进行编程 |

# 第五章 灯具表面板（Fixture Sheet）

## 5.1 面板概览

灯具表面板是查看和操作灯具最核心的面板。它以电子表格形式展示所有已配接灯具的信息和属性值。

## 5.2 表格结构

### 列定义

| 列 | 说明 |
|----|------|
| **ID** | Fixture ID 编号 |
| **名称** | 灯具名称 |
| **类型** | 灯具型号 |
| **Dimmer** | 亮度值（0-100%） |
| **Pan** | 水平旋转角度（如果灯具支持） |
| **Tilt** | 垂直旋转角度（如果灯具支持） |
| **Red / Green / Blue** | RGB 颜色值（如果灯具支持） |
| **Zoom** | 光束角度（如果灯具支持） |
| **其他属性** | 根据灯具类型动态显示 |

### 行定义

- 每个已配接灯具占一行
- 矩阵灯具可以展开，每个子灯具（模块实例）占一行，在主灯具行下方缩进显示

## 5.3 值显示规则

### 颜色编码

| 颜色 | 含义 |
|------|------|
| **白色** | 来自 CUE 回放或默认值的当前输出值 |
| **红色** | 编程器中的值（已修改但未存储到 CUE 的值） |
| **灰色** | 该灯具不支持此属性 |

### 混合值

当选中多个灯具，且它们的某个属性值不同时，编码器轮上会显示 **"Mixed"**（混合值）标记。

## 5.4 交互操作

### 选择灯具

- **单击行** → 选中该灯具
- **Ctrl + 单击** → 多选
- **Shift + 单击** → 范围选
- 选中行以蓝色背景高亮

### 排序

- **单击列头** → 按该列升序排列
- **再次单击同一列头** → 切换为降序
- 默认按 Fixture ID 升序排列

### 搜索

- 面板顶部的搜索框可输入关键字
- 实时筛选灯具名称或类型匹配的行

### 展开矩阵灯具

- 单击矩阵灯具行左侧的 **展开箭头**
- 显示该灯具的所有子灯具行
- 再次单击收起

## 5.5 灯具表面板工具栏

面板顶部提供以下功能按钮：

| 按钮 | 功能 |
|------|------|
| **搜索框** | 输入关键字筛选灯具 |
| **展开/折叠全部** | 一键展开或折叠所有矩阵灯具 |
| **DMX 值显示** | 切换显示百分比值或原始 DMX 值（0-255） |
