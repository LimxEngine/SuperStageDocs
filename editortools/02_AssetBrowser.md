# SuperStage 资产浏览器 用户手册

## 1. 概述

资产浏览器（Super Asset Browser）是 SuperStage 插件的灯具资产管理面板。它自动收集项目中所有可用的 SuperStage 灯具蓝图，按分类和制造商进行组织，支持搜索、筛选和拖放操作，方便快速将灯具放置到场景中。

---

## 2. 打开方式

**主菜单路径**：工具栏 **SuperStage** 下拉菜单 → **SuperBrowser**

---

## 3. 界面布局

资产浏览器分为三个主要区域：

```
┌─────────────────────────────────┐
│  搜索栏 + 筛选按钮              │
├──────────┬──────────────────────┤
│          │                      │
│ 分类树   │   灯具资产瓦片视图    │
│          │                      │
│          │                      │
├──────────┴──────────────────────┤
│  状态栏（资产数量统计）          │
└─────────────────────────────────┘
```

---

## 4. 功能详解

### 4.1 搜索栏

位于面板顶部，提供实时文本搜索功能。

| 操作 | 说明 |
|------|------|
| 输入关键词 | 实时过滤灯具名称，支持模糊匹配 |
| 清空搜索框 | 显示所有灯具 |

**搜索范围**：匹配灯具的资产名称。

### 4.2 分类树（左侧面板）

以树形结构展示所有灯具资产的分类层级。

**分类层级说明**：

| 层级 | 说明 | 示例 |
|------|------|------|
| 第一级 | 灯具大类（Group） | 电脑灯、常规灯、效果灯、特效设备 |
| 第二级 | 制造商（Manufacturer） | Acme、Chauvet、ClayPaky |
| 第三级 | 具体灯具型号 | XP-380Beam、Intimidator Spot |

**操作**：
- **点击分类节点**：右侧资产视图只显示该分类下的灯具
- **点击根节点"全部"**：显示所有灯具
- 分类旁的数字表示该分类下的灯具数量

### 4.3 资产瓦片视图（右侧主区域）

以瓦片（Tile）形式展示筛选后的灯具资产。

**每个瓦片显示**：
- 灯具缩略图（自动从蓝图资产获取）
- 灯具名称

**操作**：

| 操作 | 说明 |
|------|------|
| **单击** | 选中该灯具资产 |
| **拖放到视口** | 将灯具拖放到 3D 场景中的指定位置，自动创建灯具 Actor |
| **双击** | 在内容浏览器中定位到该资产 |

### 4.4 状态栏

位于面板底部，显示当前筛选结果的统计信息：
- 当前显示的资产数量
- 总资产数量

---

## 5. 使用流程

### 5.1 将灯具放入场景

1. 打开资产浏览器
2. （可选）使用左侧分类树缩小范围，或在搜索栏输入关键词
3. 在右侧瓦片视图中找到目标灯具
4. **按住鼠标左键拖动**灯具瓦片到 3D 视口中
5. 在目标位置**松开鼠标**，灯具将被放置到该位置

### 5.2 快速查找灯具

1. 在顶部搜索栏输入灯具名称的部分字符（如 "Beam"、"Wash"）
2. 视图会实时过滤，只显示名称中包含关键词的灯具
3. 清空搜索栏可恢复显示全部

---

## 6. 注意事项

- 资产浏览器只显示继承自 SuperStage 灯具基类的蓝图资产
- 新添加的灯具蓝图需要正确设置分类元数据才能出现在正确的分类下
- 如果灯具列表为空，请检查项目的 Content 目录下是否包含 SuperStage 灯具资产
- 拖放到场景时，灯具会使用默认的 DMX 配置，之后可通过 Patch 工具修改
