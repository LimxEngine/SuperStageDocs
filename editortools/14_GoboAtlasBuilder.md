# SuperStage GOBO 图集生成器 用户手册

## 1. 概述

GOBO 图集生成器（Gobo Atlas Builder）用于从 SuperStage 灯库（SuperFixtureLibrary）中提取指定属性关联的图案纹理（Gobo），自动打包生成横向排列的 GOBO 图集纹理。生成的图集被灯具的材质系统使用，用于在 3D 视口中实时模拟图案轮的投射效果。

---

## 2. 打开方式

**主菜单路径**：Unreal Engine 编辑器顶部主菜单 → **Tools** → **GOBO Atlas Builder**（位于 SuperStageTools 分区下）

## 4. 操作步骤

### 步骤 1：选择灯库

通过资产选择器（SObjectPropertyEntryBox）选择一个 `USuperFixtureLibrary` 资产。

### 步骤 2：输入属性名称

在 **"属性名称"** 输入框中输入要提取图案的属性名称，例如 `Gobo1`。

### 步骤 3：搜索图案

点击 **"搜索"** 按钮（或在输入框中按 Enter），工具会遍历灯库的所有模块（Modules）→ 属性定义（AttributeDefs）→ 子属性（SubAttributes）→ 通道集（ChannelSets），提取匹配属性中关联的 `UTexture2D*` 纹理引用。

搜索结果会在状态区域显示：
- `找到 N 个图案纹理` — 搜索成功
- `未找到属性 'XXX' 或该属性没有图案纹理` — 未找到匹配
- `请先选择灯库` / `请输入属性名称` — 参数不完整

### 步骤 4：预览图案列表

预览区域以**文本列表**方式显示所有提取到的纹理信息，包含索引、纹理名称和原始尺寸，格式如：`[0] GoboStar (256x256)`。

### 步骤 5：生成图集

点击 **"生成图集"** 按钮，工具会：
1. 读取每个 GOBO 关联的源纹理像素数据
2. 使用最近邻采样将每个纹理缩放到分配的单元格尺寸
3. 将所有纹理**横向排列**到一张 **4096×256** 像素的图集中
4. 无纹理的 GOBO 跳过（显示为黑色区域）
5. 自动保存为 UTexture2D 资产

## 5. 生成结果

### 5.1 输出路径与命名

| 项目 | 说明 |
|------|------|
| **输出目录** | 与所选灯库资产相同的目录 |
| **命名规则** | `LTA_{灯库名称}_{属性名称}` |
| **前缀处理** | 如果灯库名称以 `SL_` 开头，会自动去除该前缀 |

示例：灯库 `SL_Spot380`，属性 `Gobo1` → 输出 `LTA_Spot380_Gobo1.uasset`

### 5.2 纹理属性

| 属性 | 值 |
|------|-----|
| **尺寸** | 4096 × 256 像素（固定） |
| **格式** | BGRA8 |
| **sRGB** | 禁用（线性空间） |
| **Mipmap** | 无 |
| **压缩** | Default |
| **LOD 组** | UI |

---

## 6. 源纹理格式支持

工具在读取源纹理时支持以下格式，会自动转换为 BGRA8：

| 源格式 | 处理方式 |
|--------|----------|
| **G8（灰度）** | Gray → RGB（灰度值复制到 R/G/B），Alpha = 255 |
| **BGRA8** | 直接读取 |
| **其他 ≥ 4 字节格式** | 按 BGRA 字节序读取 |
| **不支持的格式** | 填充为白色 |

---

## 7. 使用场景

### 自动流程

与颜色图集类似，灯具放置到场景时 SuperStage 会自动检查并生成 GOBO 图集。

### 手动使用场景

- 更新了 GOBO 纹理文件后需要重新打包
- 灯库中添加了新的 GOBO 图案
- 需要为不同的图案属性（如 `Gobo2`）分别生成图集

---

## 8. 注意事项

- 需要手动输入属性名称，工具不会自动扫描所有图案属性
- 图集尺寸固定为 4096×256，无需也无法调整
- 输出路径自动确定，无需手动设置
- 重新生成图集会覆盖已有的同名纹理资产
- 如果已有同名资产存在，会先删除旧资产再创建新资产
- 图集中 GOBO 的排列顺序与灯库中的定义顺序一致
- 每个 GOBO 按比例均分图集宽度，确保完全填满

---

## 9. 常见问题

| 问题 | 解决方法 |
|------|----------|
| GOBO 列表为空 | 确认灯库中有匹配属性名称的属性，且 ChannelSet 中有纹理引用 |
| 属性名称不匹配 | 检查灯库编辑器中属性的 `AttribName` 字段，确认拼写一致 |
| 生成的图集是全黑 | 检查源纹理是否能正常读取，格式是否受支持 |
| 灯具投射的图案模糊 | 使用更高分辨率的源纹理 |
| 生成失败 | 检查灯库资产路径是否有效，磁盘是否有写入权限 |
