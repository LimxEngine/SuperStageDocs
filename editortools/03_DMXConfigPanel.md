# SuperStage DMX 配置面板 用户手册

## 1. 概述

DMX 配置面板（Super DMX Config Panel）用于设置 SuperStage 的 DMX 信号输入和输出参数。在这里你可以选择 DMX 协议类型、绑定网络接口、配置 Universe 范围，并实时查看 DMX 数据活动状态。

---

## 2. 打开方式

**打开方式**：编辑器底部状态栏 → 点击 **SuperDMX** 按钮

---

## 3. 界面说明

面板分为 **输入配置** 和 **输出配置** 两大区域，以及底部的 **活动监控** 区域。

---

## 4. 输入配置（DMX Input）

控制 SuperStage 如何接收外部 DMX 信号（例如从灯光控台发送过来的数据）。

### 4.1 协议选择

| 选项 | 说明 |
|------|------|
| **Art-Net** | 使用 Art-Net 协议接收 DMX 数据（基于 UDP，端口 6454） |
| **sACN (E1.31)** | 使用 sACN 协议接收 DMX 数据（基于 UDP 组播） |

点击下拉框选择所需的协议类型。

### 4.2 本地 IP 地址

| 参数 | 说明 |
|------|------|
| **本地 IP** | 选择用于接收 DMX 数据的网络接口 |

下拉列表会自动列出本机所有可用的网络接口 IP 地址。如果你的电脑有多个网卡（如有线网卡和无线网卡），请选择与 DMX 设备处于同一子网的网卡 IP。

> **提示**：如果列表中没有看到预期的 IP 地址，请检查网络连接后重新打开面板。

### 4.3 起始 Universe

| 参数 | 说明 | 范围 |
|------|------|------|
| **起始 Universe** | 接收 DMX 数据的起始 Universe 编号 | 1 - 256 |

SuperStage 会从此 Universe 开始监听，覆盖到所有有灯具分配的 Universe。

---

## 5. 输出配置（DMX Output）

控制 SuperStage 如何向外发送 DMX 信号（例如用于预览或控制实体灯具）。

### 5.1 协议选择

与输入配置相同，支持 **Art-Net** 和 **sACN** 两种协议。

### 5.2 目标 IP / 组播地址

| 参数 | 说明 |
|------|------|
| **目标 IP** | Art-Net 模式下的目标广播/单播地址 |
| **组播地址** | sACN 模式下的组播地址（默认自动生成） |

- **Art-Net**：通常设为 `2.255.255.255`（广播）或灯具/节点的具体 IP
- **sACN**：使用默认组播地址即可

### 5.3 起始 Universe

| 参数 | 说明 | 范围 |
|------|------|------|
| **起始 Universe** | 输出 DMX 数据的起始 Universe 编号 | 1 - 256 |

---

## 6. 活动监控

面板底部内嵌了一个简化版的 DMX 活动指示器，显示当前正在收发数据的 Universe。

---

## 7. 配置保存

- 所有配置修改会 **自动保存** 到项目设置中
- 下次打开项目时会自动恢复上次的配置
- 配置存储在项目的 `Config` 目录下

---

## 8. 典型使用场景

### 场景 1：接收灯光控台信号

1. 将电脑和灯光控台连接到同一局域网
2. 打开 DMX 配置面板
3. **输入协议** 选择与控台一致（Art-Net 或 sACN）
4. **本地 IP** 选择与控台同一子网的网卡 IP
5. **起始 Universe** 设为控台输出的第一个 Universe
6. 场景中的灯具将自动响应控台信号

### 场景 2：从 SuperStage 输出信号到实体灯具

1. 确保电脑与 DMX 节点/灯具处于同一网络
2. **输出协议** 选择 Art-Net
3. **目标 IP** 设为 DMX 节点的 IP 或广播地址
4. **起始 Universe** 设为与实体灯具匹配的 Universe

---

## 9. 故障排查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 灯具不响应控台信号 | IP 地址不在同一子网 | 检查本地 IP 选择是否正确 |
| 灯具不响应控台信号 | 协议不匹配 | 确认控台使用的是 Art-Net 还是 sACN |
| 灯具不响应控台信号 | Universe 不匹配 | 检查起始 Universe 设置 |
| IP 下拉列表为空 | 网络接口未就绪 | 检查网络连接，重新打开面板 |
| 信号有延迟 | 网络拥塞 | 使用有线网络连接 |
