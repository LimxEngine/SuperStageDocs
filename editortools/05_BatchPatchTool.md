# SuperStage 批量 Patch 工具 用户手册

## 1. 概述

批量 Patch 工具（Super Patch Tool）用于为场景中选中的多个灯具批量分配 DMX 地址。它可以自动计算每个灯具的 Universe 和起始通道地址，支持预览和一键应用，并内置撤销功能。

---

## 2. 打开方式

**主菜单路径**：工具栏 **SuperStage** 下拉菜单 → **SuperDMXTool** → **PatchTool**

---

## 4. 参数说明

### 4.1 起始参数

| 参数 | 说明 | 范围 | 默认值 |
|------|------|------|--------|
| **起始 Universe** | 第一个灯具分配到的 Universe 编号 | 1 - 256 | 1 |
| **起始地址** | 第一个灯具在该 Universe 中的起始通道地址 | 1 - 512 | 1 |

### 4.2 地址分配规则

工具按照以下规则自动计算后续灯具的地址：

1. 从指定的起始 Universe 和起始地址开始
2. 每个灯具占用的通道数量由其灯库定义的通道跨度（Channel Span）决定
3. 当一个灯具分配完成后，下一个灯具的地址 = 当前地址 + 通道跨度
4. 如果地址超过 512（一个 Universe 的最大通道数），自动跳转到下一个 Universe 的第 1 通道
5. FixtureID 自动递增分配，跳过场景中已被其他灯具占用的 ID

## 5. 灯具列表

### 5.1 列说明

| 列名 | 说明 |
|------|------|
| **灯具名称** | 灯具在场景中的标签名称 |
| **型号** | 灯具的蓝图类名 |
| **Universe** | 分配后的 Universe 编号 |
| **起始地址** | 分配后的起始通道地址 |
| **通道数** | 该灯具占用的通道数量 |
| **FixtureID** | 分配的灯具编号 |

### 5.2 排序规则

灯具列表按照自然排序（Natural Sort）显示。例如：
- Light_1, Light_2, Light_3, ..., Light_10, Light_11
- 而不是：Light_1, Light_10, Light_11, ..., Light_2, Light_3

---

## 6. 操作按钮

| 按钮 | 功能 | 说明 |
|------|------|------|
| **刷新选择** | 重新获取场景中选中的灯具 | 当你在场景中改变了选择后，点击此按钮刷新列表 |
| **预览** | 计算并显示地址分配预览 | 不会实际修改灯具，只是在列表中显示预计的分配结果 |
| **应用** | 将预览的地址分配写入灯具 | 实际修改所有列出灯具的 DMX 地址配置 |

---

## 7. 使用流程

### 步骤 1：选择灯具

在 3D 视口中框选或逐个选中需要批量 Patch 的灯具。

### 步骤 2：打开批量 Patch 工具

通过主菜单打开工具面板。

### 步骤 3：设置起始参数

- 设置 **起始 Universe**（如第 1 个 Universe）
- 设置 **起始地址**（如从第 1 通道开始）

### 步骤 4：刷新与预览

- 点击 **"刷新选择"** 确保列表中显示的是你要 Patch 的灯具
- 点击 **"预览"** 查看地址分配结果

### 步骤 5：确认并应用

- 检查预览列表中每个灯具的 Universe 和地址是否符合预期
- 点击 **"应用"** 将地址写入灯具

---

## 8. 撤销操作

应用操作支持 **Ctrl+Z 撤销**。如果发现分配有误，可以直接撤销恢复到之前的状态。

---

## 9. 注意事项

- 只有 SuperStage DMX 灯具（SuperDmxActorBase 的子类）会出现在列表中
- 普通的 UE 灯光或其他类型的 Actor 不会被识别
- 如果场景中没有选中任何灯具，列表将为空
- 通道跨度为 0 的灯具可能导致地址计算异常，请确保灯库配置正确
- 当灯具数量较多时，地址可能跨越多个 Universe，请确保你的 DMX 网络支持所需的 Universe 数量

---

## 10. 常见问题

| 问题 | 解决方法 |
|------|----------|
| 灯具列表为空 | 先在场景中选中灯具，再点击"刷新选择" |
| 预览显示的地址不对 | 检查起始 Universe 和起始地址设置是否正确 |
| 应用后灯具没响应控台 | 检查 DMX 配置面板中的输入设置是否匹配 |
| 某些灯具地址重叠 | 检查灯库中通道跨度配置是否正确 |
