# SuperStage 灯库编辑器 用户手册

## 1. 概述

灯库编辑器（Fixture Library Editor）是 SuperStage 的灯具灯库管理工具。每个 SuperStage 灯具都依赖一个灯库资产（SuperFixtureLibrary）来定义其 DMX 通道布局、属性类型、颜色轮、图案轮等详细参数。通过灯库编辑器，你可以创建和修改这些灯库定义，无需手动编辑数据文件。

---

## 2. 打开方式

在 Unreal Engine 的 **内容浏览器** 中，双击一个 `SuperFixtureLibrary` 资产即可打开灯库编辑器。

> **提示**：如果还没有灯库资产，可以在内容浏览器中右键 → 杂项 → SuperFixtureLibrary 来创建新的灯库。

## 4. 头部信息栏

显示和编辑灯具的基本信息。所有字段均可直接点击编辑。

| 字段 | 说明 | 示例 |
|------|------|------|
| **灯具名称** | 灯具的型号名称 | XP-380 Beam |
| **制造商** | 灯具制造商名称 | Acme |
| **来源** | 灯库数据来源 | Custom / GDTF / MA2 / MA3 |
| **功率 (W)** | 灯具额定功率，单位瓦特 | 380 |
| **质量 (kg)** | 灯具重量，单位千克 | 18.5 |
| **通道跨度** | 灯具占用的 DMX 通道总数（自动计算） | 24 |

---

## 5. 模块列表（左侧面板）

一个灯具可以包含一个或多个 **DMX 模块**（Module）。每个模块代表灯具的一种工作模式（如标准模式、扩展模式）。

### 5.1 模块信息

| 字段 | 说明 | 示例 |
|------|------|------|
| **模块名称** | 模块的名称，可编辑 | Standard / Extended / Basic |
| **Patch 值** | 该模块的通道总数（自动计算或手动设置） | 24 |

### 5.2 模块操作

| 操作 | 方法 |
|------|------|
| **添加模块** | 点击模块列表下方的 "+" 按钮 |
| **删除模块** | 选中模块后点击 "✕" 按钮 |
| **重命名模块** | 直接点击模块名称进行编辑 |
| **调整顺序** | 使用 "▲" "▼" 按钮上下移动 |

> **提示**：大多数灯具只需要一个模块。只有当灯具有多种 DMX 通道模式时，才需要创建多个模块。

---

## 6. 数据层级结构

灯库编辑器采用三级层级结构：

```
模块 (Module)
 └── 属性 (Attribute)           — 如 Dimmer、Pan、Tilt、ColorWheel
      └── 子属性 (SubAttribute)  — 如 ColorWheel 下的各个颜色通道范围
           └── 槽位 (ChannelSet) — 如某个颜色的具体 DMX 值范围
```

通过 **面包屑导航** 可以在各层级之间切换。点击面包屑中的层级名称可以返回上一级。

---

## 7. 属性列表（第一级）

选中一个模块后，右侧内容区域显示该模块的所有 DMX 属性。

### 7.1 属性列表字段

| 列名 | 说明 | 可编辑 |
|------|------|--------|
| **No.** | 属性序号（自动编号） | 否 |
| **属性名** | 属性的名称 | ✅ 是 |
| **分类** | 属性所属的功能分类 | ✅ 是（下拉选择） |
| **粗通道 (Coarse)** | 粗调 DMX 通道号 | ✅ 是 |
| **细通道 (Fine)** | 细调 DMX 通道号（16-bit 时使用） | ✅ 是 |
| **超细通道 (Ultra)** | 超精细 DMX 通道号（24-bit 时使用） | ✅ 是 |
| **默认值 (Default)** | 灯具启动时该属性的默认 DMX 值 | ✅ 是 |
| **高亮值 (Highlight)** | 高亮模式时的 DMX 值 | ✅ 是 |

### 7.2 属性分类说明

| 分类 | 说明 | 典型属性 |
|------|------|----------|
| **亮度 (Dimmer)** | 光输出强度控制 | Dimmer、DimmerFine |
| **位置 (Position)** | 灯具运动控制 | Pan、Tilt、PanFine、TiltFine |
| **颜色 (Color)** | 颜色控制 | ColorWheel、Red、Green、Blue、White、CTO |
| **光束 (Beam)** | 光束形状控制 | Zoom、Focus、Iris |
| **图案 (Gobo)** | 图案轮控制 | GoboWheel1、GoboWheel1Rotation |
| **棱镜 (Prism)** | 棱镜效果控制 | Prism、PrismRotation |
| **频闪 (Strobe)** | 频闪效果控制 | Shutter、Strobe |
| **控制 (Control)** | 灯具系统控制 | Reset、LampOn、LampOff |
| **效果 (Effect)** | 特殊效果 | Effect、Frost、Animation |
| **维护 (Maintenance)** | 维护功能 | DisplayControl、FanSpeed |
| **媒体 (Media)** | 媒体播放相关 | MediaFolder、MediaFile |
| **视频 (Video)** | 视频输入相关 | VideoInput、Resolution |
| **其他 (Other)** | 未分类 | 自定义属性 |

### 7.3 进入子属性

**双击** 属性行可以进入该属性的子属性列表。

---

## 8. 子属性列表（第二级）

每个属性可以包含多个子属性，用于细分该属性在不同 DMX 值范围内的行为。

### 8.1 子属性列表字段

| 列名 | 说明 | 可编辑 |
|------|------|--------|
| **No.** | 序号 | 否 |
| **名称** | 子属性名称 | ✅ 是 |
| **DMX 起始** | 该子属性生效的 DMX 起始值 | ✅ 是 |
| **DMX 结束** | 该子属性生效的 DMX 结束值 | ✅ 是 |
| **物理起始** | 对应的物理值起始（如角度、百分比） | ✅ 是 |
| **物理结束** | 对应的物理值结束 | ✅ 是 |
| **槽位数** | 该子属性下定义的槽位数量 | 否（只读） |

### 8.2 特殊分类的额外列

根据当前属性的分类，子属性列表会动态显示额外的列：

| 属性分类 | 额外列 | 说明 |
|----------|--------|------|
| **亮度 / 频闪** | 频闪模式 | 设置频闪效果的模式类型 |
| **位置** | 旋转模式 | 设置旋转的行为模式 |

### 8.3 进入槽位

**双击** 子属性行可以进入该子属性的槽位列表。

---

## 9. 槽位列表（第三级）

槽位（Channel Set）定义了子属性内更细粒度的 DMX 值到功能的映射。例如，颜色轮属性的每个颜色就是一个槽位。

### 9.1 槽位列表字段

| 列名 | 说明 | 可编辑 |
|------|------|--------|
| **No.** | 序号 | 否 |
| **名称** | 槽位名称（如 "Red"、"Open"） | ✅ 是 |
| **DMX 起始** | 该槽位的 DMX 起始值 | ✅ 是 |
| **DMX 结束** | 该槽位的 DMX 结束值 | ✅ 是 |
| **物理范围** | 对应的物理量值 | ✅ 是 |

### 9.2 特殊分类的额外列

根据当前属性的分类，槽位列表会动态显示额外的列：

| 属性分类 | 额外列 | 说明 |
|----------|--------|------|
| **图案 (Gobo)** | 图案模式 | 图案的旋转或固定模式选择 |
| **图案 (Gobo)** | 纹理 | 关联的图案纹理资产（可选择图片文件） |
| **颜色 (Color)** | 颜色 | 该槽位代表的颜色值（颜色选择器） |
| **颜色 (Color)** | 颜色索引 | 颜色在色轮中的索引编号 |
| **棱镜 (Prism)** | 棱镜面数 | 棱镜的面数（如 3 面棱镜、8 面棱镜） |
| **棱镜 (Prism)** | 棱镜半径 | 棱镜效果的扩散半径 |
| **棱镜 (Prism)** | 棱镜缩放 | 棱镜效果的缩放系数 |

## 10. 通用工具栏操作

内容区域上方的工具栏提供以下操作，适用于当前显示的任何层级（属性/子属性/槽位）：

| 按钮 | 功能 | 快捷说明 |
|------|------|----------|
| **+ 添加** | 在列表末尾添加一个新的空项目 | 新项目使用默认值 |
| **复制** | 将选中的项目复制到剪贴板 | 支持多选 |
| **粘贴** | 将剪贴板中的项目粘贴到列表末尾 | 粘贴的是独立副本 |
| **▲ 上移** | 将选中的项目向上移动一行 | 改变顺序 |
| **▼ 下移** | 将选中的项目向下移动一行 | 改变顺序 |
| **✕ 删除** | 删除选中的项目 | 支持多选批量删除 |

### 10.1 多选操作

- **Ctrl + 点击**：逐个添加/取消选择
- **Shift + 点击**：选择两次点击之间的所有项目

---

## 11. 面包屑导航

面包屑导航条显示当前所在的层级路径，例如：

```
Standard > Pan > Pan Coarse
```

- 点击 **"Standard"** → 返回到属性列表
- 点击 **"Pan"** → 返回到 Pan 属性的子属性列表
- **"Pan Coarse"** → 当前正在查看的槽位列表

## 12. 使用示例

### 示例 1：创建一个简单的 Dimmer 属性

1. 在模块列表中选中目标模块
2. 在属性列表中点击 **"+ 添加"**
3. 将属性名设为 `Dimmer`
4. 分类选择 **"亮度"**
5. 粗通道设为 `1`
6. 默认值设为 `0`
7. 高亮值设为 `255`

### 示例 2：设置颜色轮

1. 创建一个属性，名称 `ColorWheel`，分类选择 **"颜色"**，粗通道设为 `7`
2. 双击进入子属性层级
3. 添加一个子属性，名称 `Colors`，DMX 范围 `0-127`
4. 双击进入槽位层级
5. 逐个添加颜色槽位：
   - `Open` — DMX 0-9 — 颜色白色
   - `Red` — DMX 10-19 — 颜色红色
   - `Blue` — DMX 20-29 — 颜色蓝色
   - `Green` — DMX 30-39 — 颜色绿色
   - ...

### 示例 3：设置图案轮 (Gobo)

1. 创建属性 `GoboWheel1`，分类 **"图案"**，粗通道 `9`
2. 进入子属性 → 添加 `Fixed Gobos`，DMX 0-63
3. 进入槽位 → 逐个添加图案：
   - `Open` — DMX 0-7
   - `Star` — DMX 8-15，纹理选择星形图案文件
   - `Circle` — DMX 16-23，纹理选择圆形图案文件
   - ...

---

## 13. 重复名称检测

编辑器会自动检测同一模块内是否存在重复的属性名称。如果检测到重复，属性名称会以警告颜色显示，提醒你修改。

---

## 14. 保存

对灯库的所有修改会自动标记资产为"已修改"状态（资产名称旁显示 *）。使用 **Ctrl+S** 或点击编辑器的保存按钮即可保存更改。

---

## 15. 注意事项

- 修改灯库会影响所有使用该灯库的灯具实例
- 修改通道分配后，已有灯具的 DMX 行为可能发生变化
- 建议在修改前备份灯库资产
- 通道号从 1 开始，最大 512
- 一个属性可以没有细通道和超细通道（留为 0 表示不使用）
- 颜色和纹理等额外字段只在对应分类的属性下才会显示
