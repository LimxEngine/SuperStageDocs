# SuperStage 样条线灯具分布工具 用户手册

## 1. 概述

样条线灯具分布工具（Light Array Tool）可以沿一条样条线（Spline）路径均匀分布灯具。你只需要在场景中放置一条样条线，然后使用此工具指定灯具类型和数量，灯具就会自动沿路径排列。这对于弧形 Truss、圆形舞台边缘、曲线造型灯阵等场景非常实用。

---

## 2. 打开方式

**打开方式**：在编辑器左侧模式选择栏中切换到 **SuperStage 编辑模式**，然后在模式工具栏中选择 **样条线分布工具**。

> **前提条件**：场景中必须有一个包含 Spline Component 的 Actor。可以使用 UE 自带的 Spline Actor，也可以使用任何包含样条线组件的自定义 Actor。

---

## 3. 使用流程

### 步骤 1：在场景中创建样条线

在 3D 视口中放置一个 Spline Actor，并编辑其控制点来定义灯具排列的路径形状。

### 步骤 2：选中样条线

在视口中选中包含样条线的 Actor（或直接选中 Spline Component）。

### 步骤 3：打开样条线分布工具

通过主菜单打开工具面板。工具会自动检测选中的样条线。

### 步骤 4：配置灯具序列

设置要分布的灯具类型、数量和间距等参数。

### 步骤 5：预览和调整

场景中会实时显示灯具预览，调整参数直到效果满意。

### 步骤 6：创建灯具

点击 **"创建"** 按钮，将预览灯具转为正式灯具。

---

## 4. 参数详解

### 4.1 灯具序列设置（Device 分类）

灯具序列（Fixture Sequence）定义了沿样条线分布的灯具类型组合，是一个 `TArray<FLightArrayFixtureEntry>` 数组。每个条目包含：

| 参数 | 类型 | 说明 | 范围 | 默认值 |
|------|------|------|------|--------|
| **Fixture Class** | TSubclassOf\<ASuperDmxActorBase\> | 灯具蓝图类（类选择器，仅显示 ASuperDmxActorBase 的子类） | — | 无 |
| **Repeat** | int32 | 该类型灯具连续重复的次数 | 1 - 100（UI 上限 10） | 1 |
| **Rotation** | FRotator | 该类型灯具相对于样条线方向的额外旋转偏移 | 无限制 | (0, 0, 0) |

#### 序列示例

假设序列定义为：
```
[Spot380 × 2] → [Wash600 × 1] → [Spot380 × 2] → [Wash600 × 1] → ...
```

灯具会按照 `Spot, Spot, Wash, Spot, Spot, Wash, ...` 的顺序循环排列，直到填满指定数量。

#### 添加序列项

在属性面板的 Fixture Sequence 数组中点击 **"+"** 按钮添加新条目。

### 4.2 分布参数（Array 分类）

| 参数 | 类型 | 说明 | 范围 | 默认值 |
|------|------|------|------|--------|
| **Count** | int32 | 沿样条线分布的灯具总数（设为 0 时改用 Spacing 自动计算） | 0 - 500（UI 上限 100） | 10 |
| **Spacing (cm)** | float | 灯具间距（仅当 Count = 0 时生效） | ≥ 10.0 | 100.0 |
| **Start Offset %** | float | 从样条线起点偏移的百分比 | 0% - 100% | 0% |
| **End Offset %** | float | 灯具分布结束位置的百分比 | 0% - 100% | 100% |

#### Count 与 Spacing 的关系

- **Count > 0**：灯具在 StartOffset ~ EndOffset 范围内等距分布，间距自动计算
- **Count = 0**：使用 Spacing 值，灯具数量由路径长度和间距自动确定

#### 偏移百分比说明

- **Start Offset 10%**：第一个灯具从样条线总长度的 10% 处开始
- **End Offset 90%**：最后一个灯具在路径 90% 处结束
- 这样可以避免灯具放在样条线的端点上

### 4.3 变换设置（Transform 分类）

| 参数 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| **Rotation Offset** | FRotator | 所有灯具的全局旋转偏移（Pitch/Yaw/Roll） | (0, 0, 0) |
| **Position Offset** | FVector | 灯具相对于样条线切线坐标系的位置偏移（厘米） | (0, 0, 0) |
| **Follow Spline Rotation** | bool | 灯具是否自动旋转以跟随样条线的切线方向 | 开启 |

#### Follow Spline Rotation

| 状态 | 说明 |
|------|------|
| **开** | 灯具沿样条线自动旋转，正面始终朝向前进方向 |
| **关** | 所有灯具保持统一的世界空间朝向 |

#### Position Offset 坐标系

偏移值是在样条线的 **局部切线空间** 中计算的

## 5. 状态信息

工具面板顶部的 **Spline** 分类显示以下只读信息：

| 字段 | 说明 |
|------|------|
| **Status** | 当前状态提示（如 "Select an Actor with SplineComponent"） |
| **Target Spline** | 当前选中的样条线 Actor 名称 |
| **Preview Count** | 当前预览灯具的数量 |

---

## 6. 实时预览

修改任何参数时，场景中的预览灯具会实时更新。预览灯具的特征：
- 使用 `RF_Transient` 标记，不会保存到场景
- 不会发送/接收 DMX 信号
- 当修改样条线形状时，预览灯具也会自动跟随更新

### 轻量级更新

工具内部智能区分参数变化类型：
- **仅修改 Offset/Rotation**：只更新灯具的位置和旋转（`bTransformDirty`，速度更快）
- **修改 Count/Sequence/Spacing**：完全重新计算灯具分布（`bPreviewDirty`，重建预览 Actor）

---

## 7. 操作按钮与撤销

### 操作按钮

| 按钮 | 说明 |
|------|------|
| **Create Fixtures** | 销毁预览 Actor，创建正式灯具（在 Transaction 中完成，支持撤销），然后关闭工具 |
| **Done** | 完成并退出工具 |

### 创建后

1. 样条线 Actor 保留在场景中（不会被删除）
2. 创建操作支持 **Ctrl+Z 撤销**

---

## 8. 使用示例

### 示例 1：弧形 Truss 灯排

1. 在场景中创建一条弧形样条线，模拟 Truss 的弯曲形状
2. 打开样条线分布工具
3. 灯具类型选择 `Spot380`
4. 数量设为 12
5. 开启"跟随样条线方向"
6. 调整偏移使灯具位于 Truss 下方

### 示例 2：圆形舞台边灯

1. 创建一条圆形样条线
2. 灯具类型选择 `WashLED`
3. 数量设为 24
4. 起始偏移 0%，结束偏移 0%（完整圆形）
5. 开启"跟随样条线方向"

### 示例 3：交替灯具序列

1. 创建一条直线样条线
2. 在灯具序列中添加两种灯具类型：
   - Spot380 × 1
   - Wash600 × 1
3. 数量设为 20
4. 灯具会按 `Spot, Wash, Spot, Wash, ...` 交替排列

---

## 9. 注意事项

- 必须先选中一个包含 Spline Component 的 Actor
- 如果样条线的控制点太少或路径太短，可能无法放下所有灯具
- 修改样条线的形状后，预览会自动更新
- 创建后，灯具与样条线不再有关联关系（灯具是独立的 Actor）
- 如需修改排列，需要删除已创建的灯具后重新使用工具
- 序列中至少需要一种灯具类型
- 创建大量灯具（100+）可能需要几秒钟处理时间

---

## 10. 常见问题

| 问题 | 解决方法 |
|------|----------|
| 打开工具后无预览 | 确认已选中包含样条线的 Actor |
| 灯具方向不对 | 检查"跟随样条线方向"是否开启，或调整全局旋转偏移 |
| 灯具没有均匀分布 | 检查起始/结束偏移百分比设置 |
| 灯具漂浮在空中 | 调整 Z 偏移参数，或编辑样条线高度 |
| 灯具类型不可选 | 确保项目中已导入 SuperStage 灯具蓝图资产 |
| 修改样条线后灯具没更新 | 已创建的灯具不会跟随样条线更新，只有预览阶段才会实时跟随 |
