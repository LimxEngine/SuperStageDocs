# SuperStage NDI 配置面板 用户手册

## 1. 概述

NDI 配置面板（Super NDI Config Panel）用于管理 SuperStage 的 NDI（Network Device Interface）视频输入源。通过此面板，你可以创建逻辑 NDI 输入通道、发现网络上的 NDI 视频源，并将它们映射到场景中的灯具或 LED 屏幕。

---

## 2. 打开方式

**打开方式**：编辑器底部状态栏 → 点击 **SuperNDI** 按钮

---

## 3. 界面说明

```
┌──────────────────────────────────────┐
│  NDI 输入列表                        │
│  ┌──────────────────────────────┐   │
│  │ 输入 1: [NDI源下拉选择]      │   │
│  │ 输入 2: [NDI源下拉选择]      │   │
│  │ 输入 3: [NDI源下拉选择]      │   │
│  └──────────────────────────────┘   │
│                                      │
│  [+ 添加输入]  [应用到选中Actor]      │
└──────────────────────────────────────┘
```

---

## 4. 功能详解

### 4.1 NDI 输入列表

每一行代表一个逻辑 NDI 输入通道。

| 列 | 说明 |
|----|------|
| **输入编号** | 逻辑通道编号（自动分配） |
| **NDI 源选择** | 下拉框，列出当前网络上发现的所有 NDI 视频源 |
| **删除按钮** | 移除此输入通道 |

### 4.2 NDI 源下拉框

下拉框中显示的 NDI 源名称格式为：`计算机名 (源名称)`

例如：
- `LAPTOP-A1B2 (OBS Studio)`
- `WORKSTATION (Resolume Output 1)`

> **注意**：NDI 源列表会异步刷新。如果没有看到预期的源，请稍等几秒或检查 NDI 发送端是否正常工作。

### 4.3 添加输入

点击 **"+ 添加输入"** 按钮，在列表底部新增一个空的 NDI 输入通道。

### 4.4 应用到选中 Actor

选中场景中的一个或多个 Actor（如 LED 屏幕、视频面板），然后点击此按钮，将当前配置的 NDI 输入映射应用到这些 Actor 上。

---

## 5. 使用流程

### 5.1 基本配置流程

1. 打开 NDI 配置面板
2. 点击 **"+ 添加输入"** 创建一个新的 NDI 输入通道
3. 在下拉框中选择网络上的 NDI 视频源
4. 在场景中选中需要接收 NDI 视频的 Actor
5. 点击 **"应用到选中 Actor"**
6. 被选中的 Actor 将开始显示 NDI 视频画面

### 5.2 多输入配置

如果场景中有多块 LED 屏幕需要显示不同的 NDI 源：

1. 创建多个 NDI 输入通道（如输入 1、输入 2、输入 3）
2. 每个通道选择不同的 NDI 源
3. 分别选中不同的屏幕 Actor 并应用对应的输入

---

## 6. 配置保存

- NDI 输入配置会自动保存到编辑器设置中
- 下次打开项目时自动恢复

---

## 7. 故障排查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| NDI 源下拉列表为空 | 网络上没有 NDI 发送端 | 检查 NDI 发送软件是否正在运行 |
| NDI 源下拉列表为空 | 防火墙阻止 | 确保防火墙允许 NDI 流量（mDNS + NDI 端口） |
| 视频画面卡顿 | 网络带宽不足 | 使用千兆有线网络，降低 NDI 视频分辨率 |
| 看不到特定的 NDI 源 | 不在同一子网 | 确保发送端和接收端在同一网段 |
| 应用后 Actor 无画面 | Actor 类型不支持 NDI | 确保选中的 Actor 是支持 NDI 输入的类型 |
