# SuperStage DMX Patch 预览 用户手册

## 1. 概述

DMX Patch 预览面板（Super DMX Patch Preview）提供了场景中所有灯具的 DMX 地址信息的可视化列表。你可以在此面板中查看、排序和直接编辑每个灯具的 DMX 参数，无需逐个选中灯具修改。

---

## 2. 打开方式

**主菜单路径**：工具栏 **SuperStage** 下拉菜单 → **SuperDMXTool** → **PatchPreview**

---

## 4. 列表字段说明

| 列名 | 说明 | 可编辑 |
|------|------|--------|
| **型号 (Model)** | 灯具的蓝图类名 | 否（只读） |
| **名称 (Name)** | 灯具在场景中的标签 | 否（只读） |
| **Fixture ID (FID)** | 灯具编号，用于控台识别 | ✅ 是 |
| **Universe (Uni)** | DMX Universe 编号 | ✅ 是 |
| **起始地址 (Addr)** | 在该 Universe 中的起始通道 | ✅ 是 |

---

## 5. 操作说明

### 5.1 查看灯具信息

打开面板后，会自动列出当前场景中所有的 SuperStage DMX 灯具及其 Patch 信息。

### 5.2 编辑 DMX 参数

直接在列表中点击可编辑字段（Fixture ID、Universe、起始地址），输入新值后按 **Enter** 确认。修改会立即应用到对应的灯具。

| 参数 | 有效范围 |
|------|----------|
| **Fixture ID** | 1 - 9999 |
| **Universe** | 1 - 256 |
| **起始地址** | 1 - 512 |

### 5.3 排序

列表默认按以下规则排序：
1. 首先按灯具型号（类名）字母顺序
2. 同类型灯具内按标签名称中的数字部分自然排序

### 5.4 选中联动

在列表中点击某一行灯具，3D 视口中会自动选中对应的灯具 Actor，方便你快速定位。

### 5.5 刷新列表

如果在使用面板期间场景中添加或删除了灯具，点击 **"刷新列表"** 按钮更新显示。

---

## 6. 使用场景

### 场景 1：快速检查所有灯具的 Patch 状态

打开面板即可一目了然地看到所有灯具的 DMX 分配情况，快速发现地址冲突或遗漏。

### 场景 2：单独调整某个灯具的地址

在列表中找到目标灯具，直接修改其 Universe 或起始地址，无需在视口中找到并选中该灯具。

### 场景 3：配合批量 Patch 工具使用

先使用批量 Patch 工具进行初步分配，然后在 Patch 预览面板中微调个别灯具的地址。

---

## 7. 注意事项

- 面板只显示 SuperStage DMX 灯具，不显示普通 UE 灯光
- 修改是即时生效的，但支持 Ctrl+Z 撤销
- 地址冲突不会自动检测，请自行确保不同灯具不要占用相同的通道范围
- 如果灯具数量很多（数百个），刷新可能需要短暂等待
