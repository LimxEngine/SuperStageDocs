# SuperStage 用户认证与订阅 用户手册

## 1. 概述

用户认证面板（Plugin Auth）用于管理你的 SuperStage 账户和订阅状态。SuperStage 的部分高级功能（如 DMX 信号发送、MA 宏导出等）需要有效的订阅才能使用。通过此面板，你可以使用邮箱验证码登录账户、查看订阅详情、刷新授权状态。

---

## 2. 打开方式

**主菜单路径**：工具栏 **SuperStage** 下拉菜单 → **PluginAuth**

打开后会弹出一个固定尺寸的独立窗口（460 × 620 像素），标题显示 `SuperStage {版本号}`。

---

## 4. 功能详解

### 4.1 状态指示

窗口顶部品牌栏右侧有一个圆形状态指示器和文字：

| 状态文字 | 说明 |
|----------|------|
| **Logged in** | 账户已登录 |
| **Logged in (expires in Xs)** | 已登录，显示 Token 倒计时 |
| **Refreshing...** | 正在刷新用户信息 |
| **Please login** | 未登录 |
| **Logging in...** | 正在登录 |
| **Login failed** | 登录失败 |
| **Session expired** | 会话过期，需重新登录 |

### 4.2 邮箱验证码登录

SuperStage 使用**邮箱验证码**方式登录，无需设置密码。

| 步骤 | 操作 |
|------|------|
| 1 | 输入你的邮箱地址 |
| 2 | 点击 **"发送验证码"** 按钮，系统将验证码发送到你的邮箱 |
| 3 | 输入收到的验证码 |
| 4 | 点击 **"Sign In"** 按钮完成登录 |

> **自动注册**：如果你的邮箱尚未注册账户，系统会在首次登录时自动创建账户，无需单独注册。

#### 验证码发送冷却

点击"发送验证码"后，按钮会进入 **60 秒冷却倒计时**，期间无法重复发送。倒计时结束后可再次发送。

### 4.3 操作按钮

| 按钮 | 可见状态 | 功能 |
|------|----------|------|
| **发送验证码** | 未登录 | 将验证码发送到输入的邮箱 |
| **Sign In** | 未登录 | 使用邮箱和验证码登录 |
| **Refresh** | 已登录 | 重新从服务器获取最新的用户和订阅信息 |
| **Sign Out** | 已登录 | 退出当前登录的账户 |

### 4.4 账户信息区

登录成功后，面板中央区域显示用户和订阅的合并信息：

**用户信息部分**：

| 字段 | 说明 |
|------|------|
| **Username** | 用户名 |
| **Nickname** | 昵称 |
| **Email** | 注册邮箱 |
| **Registration Date** | 注册日期 |

**订阅信息部分**（分隔线下方）：

| 字段 | 说明 |
|------|------|
| **Subscription Status** | 当前订阅类型名称 |
| **Subscription Term** | 订阅起止日期 |
| **Remaining** | 剩余天数 |

| 等级 | 说明 |

| **Tier3** | 个人永久 / 企业版，所有功能完全开放 |

## 6. 使用流程

### 首次使用

1. 打开用户认证面板（SuperStage → PluginAuth）
2. 输入你的邮箱地址
3. 点击 **"发送验证码"**，到邮箱中查收验证码
4. 输入验证码，点击 **"Sign In"**
5. 如果邮箱未注册，系统会自动创建账户并登录
6. 登录成功后，用户信息和订阅状态自动显示

### 刷新订阅状态

1. 打开用户认证面板
2. 点击 **"Refresh"** 按钮
3. 系统会重新从服务器获取最新的用户和订阅信息

### 退出登录

1. 点击 **"Sign Out"** 按钮
2. 面板恢复到未登录视图
3. 本地缓存的认证信息会被清除

---

## 7. 离线与缓存

- 首次登录成功后，认证状态会缓存在本地
- 在有效订阅期内，编辑器启动时会自动加载缓存状态
- Token 有有效期，面板顶部状态栏会显示 `expires in Xs` 倒计时
- Token 过期后需要重新联网登录
- 建议定期联网以保持授权状态

---

## 8. 多设备与防踢机制

- 设备标识使用当前计算机名称（`ComputerName`）
- 同一账户在另一台设备登录时，当前设备会收到 **"Your account has logged in on another device, current session is offline"** 通知
- 被踢下线后需要重新登录

---

## 9. 注意事项

- SuperStage 使用邮箱验证码登录，**无需设置和记忆密码**
- 新邮箱首次登录自动注册，无需访问网站注册
- 更换电脑后需要重新登录
- 高级功能受限时，检查订阅是否过期或是否拥有对应等级的订阅
- 如遇到 "Session expired" 提示，重新登录即可

---

## 10. 常见问题

| 问题 | 解决方法 |
|------|----------|
| 收不到验证码 | 检查邮箱地址是否正确，查看垃圾邮件文件夹 |
| 登录失败 | 确认验证码是否过期或输入错误，确认网络连接正常 |
| 订阅状态不更新 | 点击 "Refresh" 手动刷新 |
| 高级功能不可用 | 检查订阅是否过期，或当前订阅类型是否包含该功能 |
| 被踢下线 | 同一账户在其他设备登录，当前设备需重新登录 |
| 更换电脑后无法使用 | 在新电脑上重新登录即可 |
