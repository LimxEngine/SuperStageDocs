# SuperStage 颜色图集生成器 用户手册

## 1. 概述

颜色图集生成器（Color Atlas Builder）用于从 SuperStage 灯库（SuperFixtureLibrary）中提取指定属性的颜色数据，自动生成横向排列的颜色图集纹理。生成的图集纹理被灯具的材质系统使用，用于在 3D 视口中准确模拟灯具颜色轮的实际颜色效果。

---

## 2. 打开方式

**主菜单路径**：Unreal Engine 编辑器顶部主菜单 → **Tools** → **Color Atlas Builder**（位于 SuperStageTools 分区下）

## 4. 操作步骤

### 步骤 1：选择灯库

通过资产选择器（SObjectPropertyEntryBox）选择一个 `USuperFixtureLibrary` 资产。

### 步骤 2：输入属性名称

在 **"属性名称"** 输入框中输入要提取颜色的属性名称，例如 `ColorWheel1`。

### 步骤 3：搜索颜色

点击 **"搜索"** 按钮（或在输入框中按 Enter），工具会遍历灯库的所有模块（Modules）→ 属性定义（AttributeDefs）→ 子属性（SubAttributes）→ 通道集（ChannelSets），提取匹配属性中的 `FLinearColor` 颜色值。

搜索结果会在状态区域显示：
- `找到 N 个颜色 (图集尺寸: 256x16)` — 搜索成功
- `未找到属性 'XXX' 或该属性没有颜色定义` — 未找到匹配
- `请先选择灯库` / `请输入属性名称` — 参数不完整

### 步骤 4：预览颜色

颜色预览区域以**横向滚动**方式显示所有提取到的颜色，每个颜色块大小为 32×32 像素，下方显示索引编号。

### 步骤 5：生成图集

点击 **"生成图集"** 按钮，工具会：
1. 将所有颜色**横向排列**到一张 **256×16** 像素的纹理中
2. 每个颜色按比例均分图集宽度，确保完全填满
3. 自动保存为 UTexture2D 资产

---

## 5. 生成结果

### 5.1 输出路径与命名

| 项目 | 说明 |
|------|------|
| **输出目录** | 与所选灯库资产相同的目录 |
| **命名规则** | `CLA_{灯库名称}_{属性名称}` |
| **前缀处理** | 如果灯库名称以 `SL_` 开头，会自动去除该前缀 |

示例：灯库 `SL_Spot380`，属性 `ColorWheel1` → 输出 `CLA_Spot380_ColorWheel1.uasset`

### 5.2 纹理属性

| 属性 | 值 |
|------|-----|
| **尺寸** | 256 × 16 像素（固定） |
| **格式** | BGRA8 |
| **sRGB** | 启用 |
| **Mipmap** | 无 |
| **过滤** | 最近邻（Nearest）— 避免颜色插值 |
| **压缩** | VectorDisplacementmap（无损） |
| **LOD 组** | UI |

## 6. 使用场景

### 自动流程

大多数情况下，当灯具被放置到场景中时，SuperStage 会自动检查对应灯库是否已有颜色图集，如果没有会自动生成。

### 手动使用场景

- 修改了灯库中的颜色定义后，手动重新生成图集
- 需要为不同的颜色属性（如 `ColorWheel2`）分别生成图集

---

## 7. 注意事项

- 需要手动输入属性名称，工具不会自动扫描所有颜色属性
- 图集尺寸固定为 256×16，无需也无法调整
- 输出路径自动确定，无需手动设置
- 重新生成图集会覆盖已有的同名纹理资产
- 如果已有同名资产存在，会先删除旧资产再创建新资产
- 生成的图集纹理是一个标准的 UE 纹理资产，可以在内容浏览器中查看和管理
