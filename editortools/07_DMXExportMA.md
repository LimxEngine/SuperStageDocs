# SuperStage DMX 导出 MA 宏 用户手册

## 1. 概述

DMX 导出 MA 宏工具（Super DMX To MA）用于将 SuperStage 场景中的灯具配置导出为 grandMA2 或 grandMA3 灯光控台可直接导入的宏脚本文件和灯位图 XML 文件。这让你能够在虚拟环境中完成灯具布局后，一键同步到实际演出控台，省去手动输入大量灯具参数的工作。

---

## 2. 打开方式

**主菜单路径**：工具栏 **SuperStage** 下拉菜单 → **SuperDMXTool** → **DMXToMa**

## 4. 参数说明

### 4.1 目标控台

| 选项 | 说明 |
|------|------|
| **grandMA2** | 生成适用于 grandMA2 的命令行宏脚本（.xml 格式） |
| **grandMA3** | 生成适用于 grandMA3 的 Lua 宏脚本（.lua 格式） |

> **注意**：MA2 和 MA3 的宏格式完全不同，请确保选择与你的控台版本匹配的选项。

### 4.2 导出路径

点击 **"浏览..."** 按钮选择导出文件的保存位置。默认路径为项目目录下的 `Saved/SuperStage/Export/` 文件夹。

### 4.3 灯位图选项

| 选项 | 说明 | 默认 |
|------|------|------|
| **导出灯位图 XML** | 是否同时导出灯位图文件 | ☑ 开启 |
| **包含灯具位置** | 灯位图中是否包含灯具的 XYZ 坐标 | ☑ 开启 |
| **包含灯具旋转** | 灯位图中是否包含灯具的旋转角度 | ☑ 开启 |

## 5. 灯具列表

列表中显示场景中所有 SuperStage DMX 灯具。每一行包含：

| 内容 | 说明 |
|------|------|
| **勾选框** | 是否导出该灯具（默认全部勾选） |
| **灯具名称** | 灯具在场景中的标签 |
| **Fixture ID** | 灯具编号 |
| **DMX 地址** | 格式为 `U{Universe}.{StartAddress}` |

你可以取消勾选不需要导出的灯具。

### 5.1 快捷操作

| 按钮 | 功能 |
|------|------|
| **全选** | 勾选列表中所有灯具 |
| **全不选** | 取消所有勾选 |

## 6. 导出内容说明

### 6.1 MA2 宏脚本导出

导出的 MA2 宏文件包含以下命令：

| 命令类型 | 说明 |
|----------|------|
| **灯具类型创建** | 在控台的灯库中创建对应的灯具类型 |
| **灯具 Patch** | 为每个灯具分配 Universe 和起始地址 |
| **灯具编号** | 设置 Fixture ID |
| **灯具名称** | 设置灯具的标签名称 |

### 6.2 MA3 Lua 脚本导出

导出的 MA3 Lua 文件使用 MA3 的命令行接口完成相同的操作，语法适配 MA3 环境。

### 6.3 灯位图 XML

灯位图文件包含灯具在 3D 空间中的位置和旋转信息，可以导入到 MA 软件的 3D Stage View 中，实现虚拟灯位的一比一还原。

---

## 7. 使用流程

### 步骤 1：完成灯具布局

在 SuperStage 场景中完成所有灯具的摆放和 DMX Patch 配置。

### 步骤 2：打开导出工具

通过主菜单打开 DMX 导出 MA 宏工具。

### 步骤 3：选择目标控台

选择你使用的控台版本（MA2 或 MA3）。

### 步骤 4：设置导出路径

选择一个方便访问的文件夹（如 U 盘或网络共享目录），以便传输到控台。

### 步骤 5：勾选灯具

默认所有灯具都会导出。如果某些灯具不需要导出到控台，取消勾选即可。

### 步骤 6：点击导出

点击 **"导出"** 按钮，文件将生成到指定路径。

### 步骤 7：在控台导入

- **MA2**：在控台的 Backup → Import 中导入宏文件，然后执行宏
- **MA3**：在控台的 Plugin 功能中加载 Lua 脚本并执行

---

## 8. 导出文件列表

导出完成后，在导出路径中会生成以下文件：

| 文件 | 说明 |
|------|------|
| `SuperStage_Patch_MA2.xml` 或 `SuperStage_Patch_MA3.lua` | 宏脚本文件 |
| `SuperStage_StageView.xml` | 灯位图 XML 文件（如果勾选了导出灯位图） |

---

## 9. 注意事项

- 导出前请确保所有灯具都已正确分配 DMX 地址和 Fixture ID
- 没有分配 Fixture ID 的灯具会使用自动生成的编号
- MA 宏脚本会覆盖控台中已有的同名灯具配置，请在控台上做好备份
- 灯位图中的坐标单位为米，与 SuperStage 的 Unreal 坐标自动转换
- 如果控台灯库中没有对应的灯具类型，宏脚本会尝试创建基本的灯具类型配置

---

## 10. 常见问题

| 问题 | 解决方法 |
|------|----------|
| 控台执行宏报错 | 检查目标控台版本是否匹配（MA2/MA3） |
| 灯位图位置偏移很大 | 检查 SuperStage 场景中灯具的世界坐标是否合理 |
| 导出的灯具数量不对 | 检查是否有灯具被取消勾选 |
| 控台灯具类型不匹配 | 手动在控台灯库中选择正确的灯具类型替换 |
