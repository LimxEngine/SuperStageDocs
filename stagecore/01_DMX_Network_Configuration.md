# 01 - DMX 网络配置

> **所属模块**: SuperDMX 配置面板 (SSuperDMXConfigPanel)  
> **适用对象**: 灯光编程师、技术人员  
> **前置阅读**: [00 - DMX 系统总览](/docs/stage-core/overview)  
> **最后更新**: 2026-03-06

---

## 一、打开 DMX 配置面板

在 UE 编辑器底部的 **SuperStage 工具栏** 中，点击 **DMX** 按钮即可打开 DMX 配置面板。该面板会停靠在编辑器中，你可以随时调整其位置。

配置面板分为两个主要区域：
1. **上方** — 输入/输出配置控件（一行式布局）
2. **下方** — DMX 活动监视器（实时显示通道值）

---

## 二、网络拓扑总览

```
┌──────────────────────┐          ┌──────────────────────┐
│   MA 控台 (MA3 onPC) │          │   真实灯具 / Art-Net  │
│   IP: 192.168.0.100  │          │   节点 / DMX 解码器   │
└──────────┬───────────┘          └──────────▲───────────┘
           │ Art-Net (UDP 6454)              │ Art-Net (UDP 6454)
           │ 发送 DMX 数据                    │ 接收 DMX 数据
           ▼                                 │
┌──────────────────────────────────────────────────────────┐
│                  以太网交换机 (千兆推荐)                     │
└──────────────────────────┬───────────────────────────────┘
                           │
                           ▼
              ┌────────────────────────┐
              │   UE 编辑器 (SuperStage) │
              │   IP: 192.168.0.50      │
              │                        │
              │   ┌─ 输入 ───────────┐  │
              │   │ 接收控台 DMX 信号 │  │
              │   └─────────────────┘  │
              │   ┌─ 输出 ───────────┐  │
              │   │ 发送 DMX 到灯具   │  │
              │   └─────────────────┘  │
              └────────────────────────┘
```

> **关键要求**：控台、UE 编辑器和灯具/节点必须在**同一以太网网段**（如 `192.168.0.x`，子网掩码 `255.255.255.0`）。推荐使用有线千兆连接，**避免 WiFi**。

## 三、面板布局详解

配置面板的上方是一行紧凑的控件，从左到右分为 **输入区** 和 **输出区**，中间有竖线分隔：

```
┌─ 输入区 ─────────────────────────────────────┬─── 输出区 ───────────────────────────────────────┐
│ Input │ [协议▼] │ [本地IP▼] │ ☑ Enable │ Start Universe [0] │ │ Output │ [协议▼] │ [本地IP▼] │ ☑ Enable │ Start Universe [0] │
└───────────────────────────────────────────────┴──────────────────────────────────────────────────┘
```

## 四、输入配置（接收外部 DMX 信号）

「输入」用于**接收**来自外部控台（如 MA2、MA3、grandMA3 onPC）的 DMX 信号。

### 4.1 协议选择

| 参数 | 说明 |
|------|------|
| **下拉选项** | Art-Net |
| **默认值** | Art-Net |

当前版本支持 **Art-Net** 协议。Art-Net 是最常用的 DMX over Ethernet 协议，使用 UDP 端口 6454。

> **说明**：sACN (E1.31) 协议已预留，将在未来版本中提供。

### 4.2 本地 IP 地址

| 参数 | 说明 |
|------|------|
| **下拉选项** | 当前电脑上所有可用的网卡 IP 地址 |
| **默认值** | 空（表示监听所有网卡，即 0.0.0.0） |

**如何选择正确的 IP**：

- 如果你的电脑只有一个网卡 → 留空或选择唯一的 IP 即可
- 如果你的电脑有多个网卡（例如一个连控台、一个连互联网） → **必须选择与控台在同一网段的那个网卡 IP**
- 选择错误的网卡 IP 会导致收不到 DMX 信号

**示例**：
- 控台 IP: `192.168.0.100`
- 你的电脑网卡1: `192.168.0.50`（与控台同一网段 ✅）
- 你的电脑网卡2: `10.0.0.5`（不同网段 ❌）
- → 应选择 `192.168.0.50`

### 4.3 启用开关 (Enable)

| 参数 | 说明 |
|------|------|
| **类型** | 复选框 |
| **默认值** | 已勾选（启用） |

- **勾选** → SuperStage 开始监听 UDP 端口，接收 Art-Net 数据包
- **取消勾选** → 停止接收，节省网络资源

> **提示**：如果你暂时不需要接收外部 DMX 信号（例如只使用 Sequencer 回放），可以取消勾选以减少不必要的网络开销。

### 4.4 起始 Universe (Start Universe)

| 参数 | 说明 |
|------|------|
| **类型** | 整数输入框（支持拖拽调节） |
| **范围** | 0 - 32,767 |
| **默认值** | 0 |

这是一个**偏移量**，用于将网络中的 Art-Net Universe 编号映射到 SuperStage 内部的 Universe 编号。

**映射公式**：
```
SuperStage 内部 Universe = 网络 Universe - 起始 Universe + 1
```

**常见设置**：

| 场景 | 起始 Universe | 效果 |
|------|---------------|------|
| 默认（MA2/MA3 标准） | **0** | 网络 Universe 0 → SuperStage Universe 1 |
| MA2/MA3（Universe 从 1 开始） | **1** | 网络 Universe 1 → SuperStage Universe 1 |
| 跳过前 10 个 Universe | **10** | 网络 Universe 10 → SuperStage Universe 1 |

**大多数情况下保持默认值 0 即可**。只有当你的控台 Universe 编号与 SuperStage 不对齐时才需要调整。

**如何判断是否需要调整**：
1. 在控台中将 Universe 1 的通道 1 设为 255
2. 在 SuperStage 活动监视器中查看 Universe 1 的通道 1
3. 如果显示 255 → 对齐正确，无需调整
4. 如果显示 0 → 需要调整起始 Universe 值

## 五、输出配置（发送 DMX 信号到外部设备）

「输出」用于将 SuperStage 中的 DMX 数据（通常来自 Sequencer 回放）**发送**到外部设备（如真实灯具、Art-Net 节点）。

### 5.1 协议选择

与输入配置相同，当前支持 **Art-Net** 协议。

### 5.2 本地 IP 地址

| 参数 | 说明 |
|------|------|
| **下拉选项** | 当前电脑上所有可用的网卡 IP 地址 |
| **默认值** | 空（表示使用默认网卡） |

选择用于发送 Art-Net 数据包的网卡。应选择与目标设备在同一网段的网卡。

### 5.3 启用开关 (Enable)

| 参数 | 说明 |
|------|------|
| **类型** | 复选框 |
| **默认值** | 已勾选（启用） |

- **勾选** → SuperStage 可以通过 Art-Net 发送 DMX 数据
- **取消勾选** → 不发送任何 DMX 数据到网络

> **注意**：即使启用了输出，也只有在 Sequencer 回放或其他主动发送 DMX 的操作时才会实际发送数据。仅启用开关不会产生额外的网络流量。

### 5.4 起始 Universe (Start Universe)

| 参数 | 说明 |
|------|------|
| **类型** | 整数输入框（支持拖拽调节） |
| **范围** | 0 - 32,767 |
| **默认值** | 0 |

**映射公式**（与输入方向相反）：
```
网络 Universe = SuperStage 内部 Universe + 起始 Universe - 1
```

**示例**：

| SuperStage Universe | 起始 Universe | 发送到的网络 Universe |
|---------------------|---------------|----------------------|
| 1 | 0 | 0 |
| 1 | 1 | 1 |
| 5 | 0 | 4 |
| 5 | 1 | 5 |

> **提示**：输入和输出的起始 Universe 通常设为相同的值，以确保收发一致。

## 六、远程 IP 地址（发送目标）

输出的远程 IP 地址（即 Art-Net 数据包发送到哪里）默认为 **255.255.255.255**（广播模式），表示发送到同一网段的所有设备。

| 模式 | 远程 IP | 说明 |
|------|---------|------|
| **广播**（默认） | `255.255.255.255` | 同一网段所有设备都能收到 |
| **单播** | 例如 `192.168.0.100` | 仅发送到指定设备 |

> **说明**：远程 IP 在当前版本中通过编辑器设置面板 (Project Settings > Plugins > SuperDMX) 修改。大多数场景使用默认广播即可。

---

## 七、端口设置

| 协议 | 端口 | 是否可修改 |
|------|------|-----------|
| Art-Net | **6454** | 固定（Art-Net 标准端口） |

Art-Net 协议规定使用 UDP 端口 6454，输入和输出共用此端口。

---

## 八、配置自动保存

DMX 配置面板中的所有设置会**自动保存**到项目配置文件中。你不需要手动保存。

- **保存位置**：项目的 `EditorPerProjectUserSettings` 配置
- **保存时机**：每次修改任何参数时立即保存并应用
- **加载时机**：下次打开 UE 编辑器时自动加载上次的配置

---

## 九、配置检查清单

在开始使用之前，请按以下清单检查你的配置：

- [ ] 电脑与控台连接在**同一以太网络**（同一交换机或路由器）
- [ ] 电脑与控台的 IP 在**同一网段**（例如都是 192.168.0.x，子网掩码 255.255.255.0）
- [ ] DMX 配置面板中**输入已启用**
- [ ] **本地 IP** 选择了正确的网卡（与控台同网段的那个）
- [ ] **起始 Universe** 设置正确（默认 0 通常即可）
- [ ] Windows 防火墙**允许 UE 编辑器**的 UDP 6454 端口通信
- [ ] 控台的 Art-Net 输出已开启且目标地址包含你的电脑 IP（或广播地址）

## 十、故障排除

### 问题：活动监视器没有任何数据

**依次检查**：

1. **物理连接**：确认网线已连接，网口指示灯亮起
2. **IP 配置**：在命令提示符中运行 `ipconfig`，确认你的 IP 与控台在同一网段
3. **防火墙**：临时关闭 Windows 防火墙测试。如果关闭后正常，则需要添加 UDP 6454 的防火墙放行规则
4. **控台设置**：确认控台的 Art-Net 输出已开启
5. **协议匹配**：确认控台输出的是 Art-Net 而非 sACN
6. **本地 IP 选择**：尝试切换配置面板中的本地 IP，或留空（监听所有网卡）

### 问题：收到信号但灯具不动

1. **检查 Universe**：灯具设置的 Universe 是否与活动监视器中收到信号的 Universe 一致
2. **检查地址**：灯具的 Start Address 是否与控台中的地址一致
3. **检查灯具库**：灯具是否正确关联了 Fixture Library

### 问题：信号有延迟

1. 确认使用的是**有线以太网**（不要使用 WiFi）
2. 检查是否有其他软件大量占用网络带宽
3. 确认使用千兆交换机（尤其是多 Universe 场景）

## 十一、进阶设置（Project Settings）

除了 DMX 配置面板中的快捷设置外，你还可以在 UE 编辑器的 **Project Settings > Plugins > SuperDMX** 中找到更详细的设置：

| 设置项 | 说明 |
|--------|------|
| **Protocol** | 网络协议（Art-Net） |
| **Input > bEnabled** | 输入启用开关 |
| **Input > LocalIp** | 输入监听的本地 IP |
| **Input > RemoteIp** | 输入的远程 IP（通常为广播地址） |
| **Input > Port** | 输入端口（6454） |
| **Input > StartUniverse** | 输入起始 Universe 偏移 |
| **Output > bEnabled** | 输出启用开关 |
| **Output > LocalIp** | 输出使用的本地 IP |
| **Output > RemoteIp** | 输出目标远程 IP |
| **Output > Port** | 输出端口（6454） |
| **Output > StartUniverse** | 输出起始 Universe 偏移 |

这些设置与 DMX 配置面板中的控件完全同步，修改任何一处都会自动更新另一处。
