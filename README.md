## <img src="https://media2.giphy.com/media/v1.Y2lkPTc5MGI3NjExcWx4OWk5eGFha3dscXAzbHd0Z3NqM3FwMnRhcnJjOGU5MzVpMDFwayZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/QMHoU66sBXqqLqYvGO/giphy.gif" width="35" align="center"/> 概述

<table>
<tr>
<td width="55%">

**SuperStage** 是基于 **Unreal Engine 5** 的新一代舞美灯光预演与虚拟制作系统。它超越了传统"可视化插件"的定义范畴，将灯光控台编程、激光控制、NDI 视频流、无人机编队、施工图绘制整合在统一的实时 3D 环境中，实现 **「一个引擎，全链路覆盖」**。

SuperStage 旨在为 **虚拟制片（Virtual Production）**、**大型现场演出（Live Events）** 及 **跨媒体装置艺术** 提供一个 **确定性、实时性、全链路** 的统一生产环境。

</td>
<td width="45%">

### 🎯 核心价值

| 特性 | 描述 |
|:---:|:---|
| 🎯 | **零学习曲线集成** |
| 🚀 | **工业级性能** < 16ms 延迟 |
| 🎬 | **时间轴原生** Sequencer 深度集成 |
| 🔌 | **开放协议** Art-Net/sACN/NDI/Beyond |
| 🎨 | **资产驱动** 40+ 专业灯具库 |
| 📐 | **施工图输出** DXF/PDF/PNG |
| 🌐 | **跨平台同步** SuperData 协议 |

</td>
</tr>
</table>

### 🏗️ 设计哲学

| 原则 | 说明 |
|:---:|:---|
| **Configuration over Customization** | 不为任何单一厂家写死代码，所有硬件特性通用化、配置化 |
| **Graceful Degradation** | 单台灯具离线不影响全局，Art-Net 中断保持最后有效状态 |
| **Single Source of Truth** | 所有状态判定以 SuperStage 为准，硬件端只是"影射" |
| **Zero Trust for External Systems** | 所有外部输入经过校验和容错处理 |

## <img src="https://media0.giphy.com/media/v1.Y2lkPTc5MGI3NjExZ3F1bHN4aXc0eWg3YzB1eXRpb3l4c2t4YXd6eG83NWE1MzFlM2RlYSZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/KzqU6zKL9j9MQ/giphy.gif" width="30" align="center"/> 快速开始

### 1️⃣ 安装插件

将 SuperStage 插件解压到项目的 `Plugins/` 目录，或在 UE 编辑器中启用已安装的插件。

### 2️⃣ 放置灯具

```
Window → SuperStage → SuperBrowser → 拖拽灯具到场景
```

### 3️⃣ 配置 DMX

```
选中灯具 → Details 面板 → 设置 Universe + StartAddress
```

### 4️⃣ 打开控台

```
Window → SuperStage → Console → 开始灯光编程
```

### 5️⃣ 导出施工图

```
Window → SuperStage → SuperCAD → 导出 DXF/PDF
```
