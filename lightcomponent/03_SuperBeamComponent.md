# 03 - 体积光束组件 (SuperBeamComponent)

> **所属模块**: SuperStage 运行时  
> **适用对象**: 灯光设计师、灯具蓝图制作者  
> **前置阅读**: [02 - 聚光灯组件](02_SuperSpotComponent.md)

---

## 一、概述

**SuperBeamComponent** 是在聚光灯组件基础上增加了**可见体积光束**的组件。它通过一个专用的光束静态网格和体积光材质，在场景中渲染出从灯具镜片到地面/目标的「可见光柱」效果，是 SuperStage 中最具视觉冲击力的光源组件。

在真实演出场景中，当舞台上有烟雾/Haze 时，灯光的光束会变得可见。SuperBeamComponent 正是模拟了这一效果——无论场景中是否有真实的雾效粒子，都能渲染出清晰的体积光束。

### 适用场景

- **Beam 灯** — 光束灯、光柱灯，需要清晰可见的体积光束
- **Profile 灯** — 成像灯配合体积光束展示 Gobo 图案
- **Spot 灯 + 光束** — 需要同时有光照投射和可见光束
- **作为切割组件（CuttingComponent）的基类**

---

## 二、组件结构

```
SuperBeamComponent (继承自 SuperSpotComponent)
  ├── YStaticMeshLens   — 镜片模型（继承）
  ├── YArrowComponent   — 方向箭头（继承）
  ├── YSpotLight        — 聚光灯光源（继承）
  └── YStaticMeshBeam   — 光束模型（新增，锥形/柱形体积网格）
```

### 三套材质系统

SuperBeamComponent 管理**三套**独立的动态材质，确保镜片、光斑和光束的视觉效果保持一致：

| 材质 | 用途 | 受影响的参数 |
|------|------|-------------|
| **DynamicMaterialLens** | 镜片发光效果 | 亮度、颜色、Gobo、旋转、棱镜、对焦、颜色轮 |
| **DynamicMaterialLightSpot** | SpotLight 光照函数 | 亮度、颜色、Gobo、旋转、棱镜、对焦、颜色轮 |
| **DynamicMaterialBeam** | 光束体积效果 | 亮度、颜色、变焦、雾化、大气密度、Gobo、旋转、棱镜、对焦、颜色轮 |

> **核心设计**：当你设置颜色、Gobo 图案、棱镜等参数时，三套材质会**同步更新**，确保镜片上看到的图案与光束中显示的图案以及投射到地面的光斑图案完全一致。

## 三、默认参数 (DefaultParameter)

### 3.1 光束品质参数

以下参数在灯具初始化时设置，决定光束的视觉品质：

| 参数 | 说明 | 范围 | 推荐值 |
|------|------|------|--------|
| **最大亮度** | 光束材质的最大亮度基准（与灯具最大亮度挂钩） | > 0 | 与灯具一致 |
| **大气密度 (AtmosphericDensity)** | 光束中模拟大气粒子的密度，值越高光束越「浓」 | > 0 | 根据灯具类型 |
| **镜片半径 (LensRadius)** | 光束在镜片处的初始半径大小 | > 0 | 10.0 |
| **光束品质 (BeamQuality)** | 控制光束体积效果的采样精度（0~100），值越高光束越细腻但开销越大 | 0 ~ 100 | 50 |
| **烟雾影响 (FogInfluence)** | 烟雾/Haze 对光束的影响程度，0 = 无烟雾纹理，高值 = 明显的烟雾流动 | ≥ 0 | 适中 |
| **烟雾速度 (FogSpeed)** | 光束中烟雾纹理的流动速度 | ≥ 0 | 适中 |
| **光束亮度 (BeamIntensity)** | 光束材质的亮度乘数，独立于 SpotLight 亮度 | > 0 | 1.0 |

> **示例**：
> - 高功率 Beam 灯：大气密度高、镜片半径小、光束亮度高 → 细而亮的光柱
> - 宽角 Wash 灯：大气密度低、镜片半径大、光束亮度低 → 宽而柔的光雾

### 3.2 继承的默认参数

| 参数 | 来源 | 说明 |
|------|------|------|
| **ZoomRange** | SuperSpotComponent | 变焦角度范围 |
| **bDisableLightFunction** | SuperSpotComponent | 光照函数开关 |
| **Angle** | SuperLightingComponent | 光斑角度 |
| **DimmerCurveExponent** | SuperLightingComponent | 亮度曲线指数 |
| **MaxLightIntensity** | SuperLightingComponent | 组件级亮度分控 |

> **自动初始化**：组件默认参数由 `OnRegister()` 自动初始化。组件注册时会自动调用 `SetLightingMaterial()` 和 `SetBeamDefaultValue()`，无需在 Actor 中手动调用。

## 四、DMX 可控参数

### 4.1 亮度 (Intensity / Dimmer)

| 操作 | 说明 | 参数范围 |
|------|------|----------|
| **设置亮度** | 同时控制 SpotLight、镜片材质和**光束材质**的亮度 | 0.0 ~ 1.0 |

最终亮度计算：
```
光束亮度 = Actor级Dimmer × ComponentDimmer × 频闪倍率
```

当亮度为 0 时，光束材质的 Brightness 参数归零，光束在视觉上完全消失。

### 4.2 颜色 (Color)

| 操作 | 说明 |
|------|------|
| **设置颜色** | 同时设置 SpotLight、镜片和**光束**的颜色 |

光束的颜色通过材质参数 `LightColor` 控制，与 SpotLight 颜色保持同步。

> **视觉效果**：红色光束会呈现红色的体积光柱效果，蓝色光束呈现蓝色光柱，与真实舞台效果一致。

### 4.3 变焦 (Zoom)

| 参数 | 说明 | 范围 |
|------|------|------|
| **Zoom** | 同时控制 SpotLight 锥角和光束材质的缩放 | 0.0 ~ 1.0 |

变焦对光束的影响：

| 效果 | 说明 |
|------|------|
| **SpotLight 角度** | 继承自 SuperSpotComponent 的变焦映射 |
| **光束材质 LightZoom** | 光束的扩展程度，根据 ZoomRange 线性插值 |
| **光束材质 MaxLightDistance** | 光束的最大长度 |

> **视觉效果**：Zoom 值小时光束细而集中，Zoom 值大时光束宽而散开。

### 4.4 雾化 (Frost)

| 参数 | 说明 | 范围 |
|------|------|------|
| **Frost** | 柔化光束和光斑 | 0.0 ~ 1.0 |

雾化对光束的特殊影响：
- **光束材质 LightZoom**：Frost 额外增大光束的 Zoom 值（Frost × 6.0），使光束变宽
- **光束材质 LightFrost**：直接控制光束的雾化参数

> **注意**：Frost 对光束的影响是在当前 Zoom 值基础上**叠加**的，不会改变基础 Zoom 值。

### 4.5 光圈 (Iris)

| 参数 | 说明 | 范围 |
|------|------|------|
| **Iris** | 光圈开合度 | 0.0（全关）~ 1.0（全开） |

光圈对光束的影响（**只能缩小，不能放大**）：

| 效果 | 计算方式 |
|------|----------|
| **光束 LensRadius** | `默认镜片半径 × Iris`，光圈缩小时光束根部变细 |
| **光束 LightZoom** | `当前Zoom值 × Iris`，光圈缩小时光束整体变窄 |
| **SpotLight 角度** | 继承自 SuperSpotComponent 的光圈效果 |
| **镜片/光斑 LensRadius** | 同步更新，保持一致 |

> **视觉效果**：Iris 缩小时，光束从根部到末端都会变细，模拟真实光圈收束的效果。

### 4.6 Gobo 图案 (Beam Texture)

| 参数 | 说明 |
|------|------|
| **光束纹理 (BeamTexture)** | Gobo 图案纹理，会被投射到光束体积中 |
| **Gobo 数量 (NumGobos)** | 纹理图集中包含的 Gobo 图案数量 |
| **Gobo 索引 (GoboIndex)** | 当前选中的 Gobo 图案编号 |
| **Gobo 旋转速度 (GoboSpeed)** | Gobo 图案的旋转速度 |
| **抖动速度 (ShakeSpeed)** | Gobo 图案的抖动效果速度 |

> **核心特性**：Gobo 图案会**同时**应用到光束、光斑和镜片三套材质，确保从任何角度看到的效果都一致。

### 4.7 颜色轮 (Color Wheel)

与聚光灯组件相同，支持最多 **3 组颜色轮**。每组参数（纹理、数量、索引、速度）会同步应用到光束、光斑和镜片三套材质。

### 4.8 棱镜 (Prism)

| 参数 | 说明 | 范围 |
|------|------|------|
| **棱镜面数 (PrismFacets)** | 棱镜的面数（如 3面、8面、16面等），0 = 关闭棱镜 | ≥ 0 |
| **棱镜半径 (PrismRadius)** | 棱镜效果的分裂半径 | 默认 0.3 |
| **棱镜缩放 (PrismScale)** | 每个棱镜分裂的缩放比例 | 默认 0.15 |
| **棱镜旋转角度 (PrismRotation)** | 棱镜的静态旋转角度 | 任意 |
| **棱镜旋转速度 (PrismRotationSpeed)** | 棱镜的持续旋转速度 | 任意 |

棱镜效果会同步应用到光束、光斑和镜片三套材质。

> **特殊操作**：当 PrismFacets 设为负值（如 -1）时，仅更新旋转参数而不改变棱镜面数和大小——用于「仅旋转棱镜」的控制通道。

### 4.9 对焦 (Focus)

| 参数 | 说明 | 范围 |
|------|------|------|
| **对焦 (Focus)** | 控制 Gobo 图案的清晰度 | 0.0（清晰）~ 1.0（模糊） |

对焦效果同步应用到光束、光斑和镜片三套材质的 `Focus` 参数。

> **使用场景**：当 Gobo 图案需要呈现模糊的背景效果时，增大 Focus 值。

### 4.10 图案旋转 (Rotate)

| 参数 | 说明 |
|------|------|
| **旋转角度** | 同时旋转光束、光斑和镜片上的图案 |
| **无极旋转** | 光束、光斑和镜片上的图案持续旋转 |

### 4.11 可见性控制

| 操作 | 说明 |
|------|------|
| **设置可见性** | 控制光束网格的显示/隐藏（**不影响 SpotLight 光照**） |

> **注意**：光束可见性仅控制体积光束网格的显隐。要同时关闭光照，需要通过灯具 Actor 的亮度通道将亮度设为 0，或使用 SpotLight 的可见性控制。

## 五、视觉效果调节指南

### 5.1 光束浓度

- **大气密度 (AtmosphericDensity)** — 增大使光束整体更「浓密」
- **烟雾影响 (FogInfluence)** — 增大使光束出现明显的烟雾纹理

### 5.2 光束锐度

- **光束品质 (BeamQuality)** — 增大使光束边缘更清晰，但性能开销更高
- **镜片半径 (LensRadius)** — 减小使光束根部更细，看起来更聚焦
- **对焦 (Focus)** — 减小使 Gobo 图案更清晰

### 5.3 烟雾效果

- **烟雾影响 (FogInfluence)** — 控制烟雾纹理的可见程度
- **烟雾速度 (FogSpeed)** — 控制烟雾流动的快慢

### 5.4 光束亮度

- **光束亮度 (BeamIntensity)** — 独立于 SpotLight，单独控制光束的亮度
- **ComponentDimmer** — 同时影响 SpotLight 和光束
- **DMX Dimmer** — 同时影响所有视觉元素

## 六、常见问题

### Q: 光束和光斑的颜色/图案不一致怎么办？
**A**: 正常使用情况下不会出现这个问题，因为所有参数都同步写入三套材质。如果发现不一致，检查是否有自定义代码只修改了部分材质。

### Q: 光束效果很模糊/看不清 Gobo 图案？
**A**: 调高光束品质 (BeamQuality) 参数。注意这会增加 GPU 开销。

### Q: 如何只显示光束不显示光照？
**A**: 可以将 SpotLight 的光照强度设为极低值，同时保持光束材质亮度正常。或者在灯具蓝图中不绑定光照函数材质。

### Q: 光束在移动/旋转时有明显的抖动？
**A**: 这是变焦去抖机制导致的，阈值为角度变化 0.1°、衰减半径变化 1cm。如果需要更平滑的过渡，可以在灯具 Actor 层面加入插值逻辑。
