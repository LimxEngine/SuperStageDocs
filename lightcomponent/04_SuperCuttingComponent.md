# 04 - 切割光束组件 (SuperCuttingComponent)

> **所属模块**: SuperStage 运行时  
> **适用对象**: 灯光设计师、灯具蓝图制作者  
> **前置阅读**: [03 - 体积光束组件](/docs/light-components/beam)

---

## 一、概述

**SuperCuttingComponent** 是在体积光束组件基础上增加了**四叶光闸切割**功能的组件。它模拟真实 Profile 灯（成像灯/切割灯）的光闸机构，能够将圆形光斑裁剪为任意四边形形状——从简单的矩形到复杂的梯形、平行四边形都可以实现。

在真实舞台上，Profile 灯通常配备 4 片可独立调节的金属光闸叶片，每片叶片的两端可以独立伸缩，从而在上下左右四个方向对光斑进行裁剪。SuperCuttingComponent 完美模拟了这一机构。

### 适用场景

- **Profile 灯 / 成像灯** — 需要精确控制光斑形状的灯具
- **切割灯** — 需要将光斑裁剪为特定形状（如舞台区域、建筑轮廓）
- **Follow Spot（追光灯）** — 需要通过光闸限制光斑范围

---

## 二、组件结构

```
SuperCuttingComponent (继承自 SuperBeamComponent)
  ├── YStaticMeshLens   — 镜片模型（继承，使用切割专用镜片材质）
  ├── YArrowComponent   — 方向箭头（继承）
  ├── YSpotLight        — 聚光灯光源（继承，使用切割专用光斑材质）
  └── YStaticMeshBeam   — 光束模型（继承，使用切割专用光束材质）
```

### 切割专用材质

SuperCuttingComponent 在构造时会**替换**父类的三套材质为切割专用版本：

| 材质 | 路径 | 说明 |
|------|------|------|
| **光束材质** | MI_Beam_Cutting_Inst | 支持切割参数的体积光束材质 |
| **光斑材质** | MI_CuttingLampLight_Inst | 支持切割参数的光照函数材质 |
| **镜片材质** | MI_CuttingLens_Inst | 支持切割参数的镜片材质 |

> **关键区别**：普通 Beam 组件的材质不支持切割参数，只有切割专用材质才能响应光闸叶片的调节。

## 三、光闸系统 (Cutting / Framing Shutter)

### 3.1 光闸结构

切割组件拥有 **4 片光闸叶片**，分别控制光斑的上、下、左、右四个边缘。每片叶片的**两端可以独立调节**，总共 **8 个控制点**：

```
        ┌─── TopLeftX ───┬─── TopRightX ───┐
        │                │                  │
   UpperLeftY            │             TopRightY
        │                │                  │
        │      ┌─────────┼────────┐         │
        │      │  可见光斑区域    │         │
        │      │                  │         │
        │      └─────────┼────────┘         │
        │                │                  │
   LowerLeftY            │           BottomRightY
        │                │                  │
        └── BottomLeftX ─┴── BottomRightX ──┘
```

### 3.2 八点参数详解

| 参数 | 方向 | 控制的边缘 | 说明 |
|------|------|-----------|------|
| **UpperLeftY** | 垂直 | 左侧叶片 - 上端 | 控制左侧叶片上半部分的伸缩 |
| **LowerLeftY** | 垂直 | 左侧叶片 - 下端 | 控制左侧叶片下半部分的伸缩 |
| **TopRightY** | 垂直 | 右侧叶片 - 上端 | 控制右侧叶片上半部分的伸缩 |
| **BottomRightY** | 垂直 | 右侧叶片 - 下端 | 控制右侧叶片下半部分的伸缩 |
| **TopLeftX** | 水平 | 上方叶片 - 左端 | 控制上方叶片左半部分的伸缩 |
| **BottomLeftX** | 水平 | 下方叶片 - 左端 | 控制下方叶片左半部分的伸缩 |
| **TopRightX** | 水平 | 上方叶片 - 右端 | 控制上方叶片右半部分的伸缩 |
| **BottomRightX** | 水平 | 下方叶片 - 右端 | 控制下方叶片右半部分的伸缩 |

> **参数值含义**：  
> - **0.0** = 叶片完全打开（不遮挡）  
> - 增大值 = 叶片逐渐伸入光斑，遮挡光束  
> - 最大值 = 叶片完全关闭（遮挡该方向的全部光线）

### 3.3 切割形状示例

#### 矩形切割
将所有叶片设置为相同的伸缩量：
```
UpperLeftY = LowerLeftY = 0.3      ← 左侧均匀遮挡
TopRightY = BottomRightY = 0.3     ← 右侧均匀遮挡
TopLeftX = TopRightX = 0.2         ← 上方均匀遮挡
BottomLeftX = BottomRightX = 0.2   ← 下方均匀遮挡
```
效果：产生一个居中的矩形光斑。

#### 梯形切割
左右叶片的上下端使用不同值：
```
UpperLeftY = 0.2, LowerLeftY = 0.4   ← 左侧下端比上端遮挡更多
TopRightY = 0.2, BottomRightY = 0.4  ← 右侧下端比上端遮挡更多
```
效果：产生一个上宽下窄的梯形光斑。

#### 三角形近似
一侧叶片完全关闭：
```
UpperLeftY = 0.0, LowerLeftY = 1.0   ← 左侧从全开到全关
TopRightY = 1.0, BottomRightY = 0.0  ← 右侧从全关到全开
```
效果：产生一个近似三角形的光斑。

## 四、DMX 可控参数

### 4.1 切割参数

| 参数组 | DMX 通道 | 说明 |
|--------|----------|------|
| **叶片 A（上）** | TopLeftX, TopRightX | 控制上方叶片的左右两端 |
| **叶片 B（下）** | BottomLeftX, BottomRightX | 控制下方叶片的左右两端 |
| **叶片 C（左）** | UpperLeftY, LowerLeftY | 控制左侧叶片的上下两端 |
| **叶片 D（右）** | TopRightY, BottomRightY | 控制右侧叶片的上下两端 |

> **提示**：在灯具库（Fixture Library）中，切割参数通常映射为 8 个独立的 DMX 通道，或者使用精细/粗糙双通道模式（16bit）以获得更精确的控制。

### 4.2 切割旋转 (Rotate)

| 参数 | 说明 | 范围 |
|------|------|------|
| **旋转角度** | 将整个切割形状旋转指定角度 | 0 ~ 360（度） |
| **无极旋转** | 切割形状持续旋转 | 任意浮点值 |

切割旋转会**同步**应用到光束、光斑和镜片三套材质：
- 光束中的切割形状跟随旋转
- 地面光斑的切割形状跟随旋转
- 镜片上的切割图案跟随旋转

> **与图案旋转的区别**：切割旋转控制的是光闸叶片的整体旋转，图案旋转（Gobo Rotate）控制的是 Gobo 图案的旋转。两者是独立的。

### 4.3 继承的全部参数

SuperCuttingComponent 完整继承了 SuperBeamComponent 的所有功能：

| 功能 | 说明 |
|------|------|
| **亮度 (Dimmer)** | 控制光束、光斑、镜片的亮度 |
| **颜色 (Color)** | 控制整体颜色 |
| **变焦 (Zoom)** | 控制光锥角度 |
| **雾化 (Frost)** | 柔化光斑边缘 |
| **光圈 (Iris)** | 缩小光斑 |
| **Gobo 图案** | 在切割形状内投射 Gobo 图案 |
| **棱镜 (Prism)** | 棱镜分裂效果 |
| **对焦 (Focus)** | Gobo 图案清晰度 |
| **颜色轮** | 最多 3 组颜色轮 |
| **频闪 (Strobe)** | 8 种频闪模式 |
| **可见性** | 光束显隐控制 |

## 五、使用技巧

### 5.1 切割 + Gobo 组合

切割和 Gobo 图案可以**叠加使用**：
- 先通过 Gobo 投射一个图案（如窗户形状）
- 再通过切割叶片裁剪掉不需要的部分
- 效果：精确控制图案投射的范围

### 5.2 切割 + 旋转组合

1. 先设置好切割形状（如一个窄长条）
2. 使用旋转参数将切割形状旋转到需要的角度
3. 配合无极旋转可以产生旋转光柱效果

### 5.3 精确边缘控制

- 每片叶片的两端**独立可调**，可以创建斜线边缘
- 四片叶片组合可以创建任意凸四边形
- 配合 Frost 参数可以柔化切割边缘

## 六、常见问题

### Q: 切割效果在光束中看不到？
**A**: 确认灯具蓝图使用的是 SuperCuttingComponent 而不是 SuperBeamComponent。只有切割组件使用切割专用材质，普通光束组件不支持切割参数。

### Q: 切割形状旋转后位置不对？
**A**: 切割旋转是围绕光束中心轴旋转的。确认旋转角度参数同时应用到了光束、光斑和镜片三套材质（组件会自动处理）。

### Q: 如何实现柔和的切割边缘？
**A**: 增大 Frost（雾化）参数值。Frost 会柔化光斑的所有边缘，包括切割产生的边缘。

### Q: 切割参数的精度不够？
**A**: 在灯具库中将切割通道配置为 16bit（精细+粗糙双通道），可以获得更精确的叶片控制。
