# 02 - 聚光灯组件 (SuperSpotComponent)

> **所属模块**: SuperStage 运行时  
> **适用对象**: 灯光设计师、灯具蓝图制作者  
> **前置阅读**: [01 - 基础灯光组件](/docs/light-components/lighting)

---

## 一、概述

**SuperSpotComponent** 是在基础灯光组件之上增加了**真实 SpotLight 聚光灯光源**的组件。它是 SuperStage 中最常用的光源组件之一，能够产生真实的光照效果（阴影、体积散射等），并支持变焦（Zoom）、雾化（Frost）、光圈（Iris）等专业灯具控制功能。

### 适用场景

- **Spot 灯** — 仅需要投射光斑，不需要可见光束效果
- **追光灯** — 需要精确控制光锥角度和亮度
- **辅助光源** — 作为灯具的 Spot 辅光组件
- **作为 Beam/Cutting 组件的基类** — 提供 SpotLight 相关的基础能力

---

## 二、组件结构

```
SuperSpotComponent (继承自 SuperLightingComponent)
  ├── YStaticMeshLens  — 镜片模型（继承）
  ├── YArrowComponent  — 方向箭头（继承）
  └── YSpotLight       — 聚光灯光源（SpotLight，产生真实光照）
```

### 材质系统

在基类的两套材质基础上，光斑材质会被绑定为 SpotLight 的**光照函数材质**（Light Function Material），实现 Gobo 图案投射、颜色轮模拟等效果。

| 材质 | 用途 |
|------|------|
| **DynamicMaterialLens** | 镜片发光效果（继承自基类） |
| **DynamicMaterialLightSpot** | SpotLight 的光照函数材质，控制光斑图案和效果 |

> **特殊情况**：当 `bDisableLightFunction` 开关打开时，光斑材质不会绑定到 SpotLight，频闪改由 Tick 直接控制光源强度。

## 三、默认参数 (DefaultParameter)

### 3.1 变焦范围

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| **ZoomRange** | 变焦角度范围（度），X = 最小角度，Y = 最大角度 | 二维向量 | (1.0, 10.0) |

DMX Zoom 通道值（0~1）会被映射到这个范围内：
- Zoom = 0.0 → 光束角度 = ZoomRange.X（最窄）
- Zoom = 1.0 → 光束角度 = ZoomRange.Y（最宽）

> **实际灯具示例**：  
> - 窄角 Spot 灯：ZoomRange = (3, 15)  
> - 宽角 Wash 灯：ZoomRange = (10, 50)  
> - Profile 灯：ZoomRange = (5, 35)

### 3.2 光照函数开关

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| **bDisableLightFunction** | 是否禁用光照函数 | 布尔 | false（启用） |

| 状态 | 行为 |
|------|------|
| **关闭**（默认） | SpotLight 使用光照函数材质，支持 Gobo 图案投射。频闪通过材质参数控制 |
| **开启** | SpotLight 不使用光照函数材质，频闪通过 Tick 直接控制光源强度。适合不需要图案投射的简单灯具 |

### 3.3 继承的默认参数

以下参数继承自基类 SuperLightingComponent：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| **Angle** | 光斑角度参考值 | 1.0 |
| **DimmerCurveExponent** | 亮度响应曲线指数 | 2.0 |
| **ComponentDimmer** | 组件级亮度分控 | 1.0 |
| **StaticMeshLens** | 镜片模型 | — |
| **LensTransform** | 镜片变换 | 无偏移 |

## 五、DMX 可控参数

### 5.1 亮度 (Intensity / Dimmer)

| 操作 | 说明 | 参数范围 |
|------|------|----------|
| **设置亮度** | 同时控制 SpotLight 光源和材质的亮度 | 0.0（全灭）~ 1.0（全亮） |

亮度会同时应用到：
- SpotLight 的强度（实际光照亮度）
- 镜片材质的 Brightness（镜片发光效果）
- 光斑材质的 Brightness（图案投影亮度）

### 5.2 颜色 (Color)

| 操作 | 说明 | 参数范围 |
|------|------|----------|
| **设置颜色** | 同时设置 SpotLight 颜色和材质颜色 | RGB 线性颜色 |

颜色会同时应用到：
- SpotLight 的光源颜色（影响实际光照）
- 镜片材质的 LightColor（镜片显示颜色）
- 光斑材质的 LightColor（图案投影颜色）

### 5.3 变焦 (Zoom)

| 参数 | 说明 | 范围 |
|------|------|------|
| **Zoom** | 变焦值，控制光锥角度大小 | 0.0（最窄）~ 1.0（最宽） |
| **光束遮挡模式** | 开启后根据角度计算衰减半径，防止光照穿透 | 开/关 |

#### 变焦映射机制

Zoom 值通过**分段线性插值表**映射到实际的 SpotLight 锥角：

```
DMX Zoom (0~1) → ZoomRange角度(度) → 查表插值 → SpotLight OuterConeAngle
```

插值表基于实测数据标定，确保不同角度范围内的变焦手感一致：

| 输入角度（度） | 输出锥角（度） | 说明 |
|----------------|----------------|------|
| 1.0 | 2.2 | 最窄光束 |
| 5.0 | 6.2 | — |
| 10.0 | 11.0 | — |
| 20.0 | 20.0 | — |
| 30.0 | 29.0 | — |
| 40.0 | 36.0 | — |
| 50.0 | 46.0 | 最宽光束 |

> **性能优化**：变焦映射使用「邻域搜索」算法——记住上一帧命中的分段索引，下一帧从该位置开始搜索，通常 0~2 次比较就能命中，几乎零开销。

#### 去抖机制

SpotLight 的角度和衰减半径更新带有**去抖处理**：
- 角度变化 < 0.1° 时不更新（避免渲染线程频繁刷新）
- 衰减半径变化 < 1cm 时不更新

### 5.4 雾化 (Frost)

| 参数 | 说明 | 范围 |
|------|------|------|
| **Frost** | 雾化程度，柔化光斑边缘 | 0.0（清晰锐利）~ 1.0（完全雾化） |

雾化效果通过两种方式实现：

1. **材质层面**：修改光斑材质的 `LightFrost` 参数，柔化图案边缘
2. **SpotLight 层面**：
   - **增大外锥角**：Frost 每增加 0.1，外锥角增加约 0.6°，使光束更宽散
   - **减小内锥角比例**：Frost=0 时内锥角为外锥角的 90%（硬边）；Frost=1 时内锥角为外锥角的 1%（极柔和边缘）

> **设计原理**：真实的雾化片（Frost Filter）会同时使光束变宽和边缘变柔，这里的实现模拟了这两种物理效果。

### 5.5 光圈 (Iris)

| 参数 | 说明 | 范围 |
|------|------|------|
| **Iris** | 光圈开合度，只能缩小不能放大 | 0.0（全关，最小光斑）~ 1.0（全开，最大光斑） |

光圈效果：
- **SpotLight 角度**：`最终角度 = 基础角度 × Iris 值`
- 光圈只能**缩小**光束，不能超过当前 Zoom 决定的基础角度
- 光圈会与 Frost 效果叠加

> **示例**：  
> - Zoom = 0.5 → 基础角度 = 20°  
> - Iris = 0.5 → 最终角度 = 10°（缩小到一半）  
> - Iris = 1.0 → 最终角度 = 20°（全开，等于基础角度）

### 5.6 图案旋转 (Rotate)

| 参数 | 说明 | 范围 |
|------|------|------|
| **旋转角度** | Gobo 图案的静态旋转角度 | 0 ~ 360（度） |
| **无极旋转速度** | Gobo 图案的持续旋转速度 | 任意浮点值 |

旋转参数会同时应用到 SpotLight 光斑材质和镜片材质。

### 5.7 颜色轮 (Color Wheel)

聚光灯组件支持最多 **3 组颜色轮**，模拟真实灯具的颜色轮机构：

| 颜色轮 | 说明 |
|--------|------|
| **颜色轮 1** | 主颜色轮 |
| **颜色轮 2** | 第二颜色轮（高端灯具） |
| **颜色轮 3** | 第三颜色轮（特殊灯具） |

每组颜色轮的参数：

| 参数 | 说明 |
|------|------|
| **颜色纹理** | 颜色轮的纹理贴图（包含所有颜色格） |
| **颜色数量** | 颜色轮上的颜色格数量 |
| **颜色索引** | 当前选中的颜色位置 |
| **颜色旋转速度** | 颜色轮的旋转速度（用于半色/彩虹效果） |

> **提示**：颜色轮效果会同时应用到 SpotLight 光斑材质和镜片材质，确保视觉一致性。

### 5.8 可见性控制

| 操作 | 说明 |
|------|------|
| **设置可见性** | 同时控制 SpotLight 光源的显示/隐藏 |

当可见性关闭时，SpotLight 停止渲染和光照计算，完全不消耗 GPU 资源。

### 5.9 光照通道

| 参数 | 说明 | 默认值 |
|------|------|--------|
| **通道 0** | 是否启用光照通道 0 | 开启 |
| **通道 1** | 是否启用光照通道 1 | 关闭 |
| **通道 2** | 是否启用光照通道 2 | 关闭 |

> **用途**：通过光照通道可以实现「某些灯光只照亮某些物体」的选择性照明效果。

### 5.10 光照传输

| 参数 | 说明 | 默认值 |
|------|------|--------|
| **影响半透明材质** | 控制 SpotLight 是否照亮半透明物体 | 开启 |

### 5.11 高级参数

| 参数 | 说明 | 范围 |
|------|------|------|
| **SpecularScale** | 高光反射缩放，控制灯光在光滑表面上产生的高光强度 | 0.0 ~ 1.0 |
| **SourceRadius** | 光源半径，影响阴影柔和度，值越大阴影越柔和 | ≥ 0.0 cm |

## 六、光照函数 (Light Function)

当 `bDisableLightFunction` 为 false（默认）时，SpotLight 会配置光照函数用于投射 Gobo 图案：

| 参数 | 说明 | 值 |
|------|------|-----|
| **光照函数缩放** | 控制光照函数的分辨率 | 64 |
| **光照函数衰减距离** | 光照函数的最远有效距离 | 100000 cm |
| **禁用时亮度** | 光照函数不可用时的亮度回退值 | 0.5 |
| **阴影分辨率缩放** | 阴影贴图分辨率倍率 | 0.5 |
| **阴影偏移** | 防止阴影自交叉的偏移量 | 0.1 |
| **使用反向平方衰减** | 是否使用物理精确的反向平方衰减 | 关闭 |

> **关于光照函数图集**：SuperStage 使用引擎的 Light Function Atlas（大小 32768），在此分辨率下可支持约 1000+ 灯具同时使用光照函数，保持清晰的 Gobo 图案。

## 七、常见问题

### Q: Zoom 变焦时光锥角度不线性，是正常的吗？
**A**: 是正常的。真实灯具的变焦机构也是非线性的。SuperSpotComponent 使用实测标定数据进行分段线性插值，模拟真实灯具的变焦行为。

### Q: 为什么设置了 Frost 后光束变宽了？
**A**: 这是正确的行为。真实的雾化片会散射光线，使光束同时变宽和柔化。SuperSpotComponent 通过增大外锥角和减小内锥角比例来模拟这一物理效果。

### Q: Iris 和 Zoom 有什么区别？
**A**: 
- **Zoom** 改变灯具的光学焦距，影响光束的基础角度范围
- **Iris** 是机械光圈，只能在当前 Zoom 的基础上**缩小**光束，不能放大

### Q: 什么时候应该开启 bDisableLightFunction？
**A**: 当灯具不需要投射 Gobo 图案时（如简单的 Wash 灯或补光灯），开启此选项可以略微提升性能。开启后频闪由 Tick 直接控制光源强度。
