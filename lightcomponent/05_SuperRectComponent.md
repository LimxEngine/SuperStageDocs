# 05 - 矩形面光组件 (SuperRectComponent)

> **所属模块**: SuperStage 运行时  
> **适用对象**: 灯光设计师、灯具蓝图制作者  
> **前置阅读**: [01 - 基础灯光组件](/docs/light-components/lighting)

---

## 一、概述

**SuperRectComponent** 是基于 UE RectLight（矩形面光源）的光源组件，专门用于模拟 **LED 面板灯、Wash 灯、柔光灯** 等产生均匀面状光照的灯具。与 SpotLight 的锥形光照不同，RectLight 从一个矩形区域均匀发出光线，产生更柔和、更自然的照明效果。

### 适用场景

- **LED 面板灯** — 平面 LED 阵列灯具
- **Wash 灯** — 染色灯、面光灯
- **柔光灯** — 需要大面积均匀照明的灯具
- **天幕灯** — 照亮背景幕布

### 与 SpotComponent 的区别

| 特性 | SuperSpotComponent | SuperRectComponent |
|------|-------------------|--------------------|
| **光源类型** | SpotLight（锥形） | RectLight（矩形面状） |
| **光照形态** | 圆锥形，有明确的光斑边缘 | 矩形面状，光照柔和均匀 |
| **变焦 (Zoom)** | 支持 | 不支持 |
| **雾化 (Frost)** | 支持（材质层面） | 不支持 |
| **光圈 (Iris)** | 支持 | 不支持 |
| **遮光板 (BarnDoor)** | 不支持 | 支持 |
| **光照函数** | 支持（Gobo 投射） | 不支持 |
| **适用灯具** | Spot/Beam/Profile 灯 | Wash/面板/柔光灯 |

---

## 二、组件结构

```
SuperRectComponent (继承自 SuperLightingComponent)
  ├── YStaticMeshLens  — 镜片模型（继承）
  ├── YArrowComponent  — 方向箭头（继承）
  └── YRectLight       — 矩形面光源（RectLight，产生面状光照）
```

> **注意**：RectLight 不使用光照函数材质，因此不支持 Gobo 图案投射。频闪效果通过 Tick 直接控制光源强度实现。

## 三、默认参数 (DefaultParameter)

### 3.1 面光尺寸

| 参数 | 说明 | 范围 | 默认值 | 单位 |
|------|------|------|--------|------|
| **SourceWidth** | 矩形光源的宽度 | 0 ~ 200 | 20.0 | cm |
| **SourceHeight** | 矩形光源的高度 | 0 ~ 200 | 20.0 | cm |

> **提示**：面光源的尺寸应与灯具的实际发光面积匹配。例如一个 30cm × 30cm 的 LED 面板灯，应设置 SourceWidth = 30, SourceHeight = 30。

### 3.2 遮光板 (BarnDoor)

| 参数 | 说明 | 范围 | 默认值 | 单位 |
|------|------|------|--------|------|
| **BarnDoorAngle** | 遮光板张开角度，控制光照的扩散范围 | 0 ~ 80 | 20.0 | 度 |
| **BarnDoorLength** | 遮光板叶片长度，影响光照边缘的锐利程度 | 0 ~ 1000 | 20.0 | cm |

#### 遮光板工作原理

遮光板（BarnDoor）是矩形灯具特有的光束控制机构：

```
        ┌──────────────┐ ← 遮光板叶片
       ╱                ╲
      ╱    BarnDoorAngle  ╲  ← 张开角度
     ╱                      ╲
    ╱                        ╲
   ╱     矩形光源发光面        ╲
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  │    SourceWidth × SourceHeight  │
  └────────────────────────────────┘
```

| BarnDoorAngle | 效果 |
|---------------|------|
| 0° | 遮光板完全关闭，光照范围极窄 |
| 20°（默认） | 适中的光照范围 |
| 80° | 遮光板大幅张开，光照范围很宽 |

| BarnDoorLength | 效果 |
|----------------|------|
| 小值 | 遮光板短，光照边缘较柔和 |
| 大值 | 遮光板长，光照边缘更锐利，遮光效果更明显 |

### 3.3 继承的默认参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| **DimmerCurveExponent** | 亮度响应曲线指数 | 2.0 |
| **ComponentDimmer** | 组件级亮度分控 | 1.0 |
| **StaticMeshLens** | 镜片模型 | — |
| **LensTransform** | 镜片变换 | 无偏移 |

## 四、灯光初始化参数

以下参数在灯具初始化时由灯具 Actor 设置：

| 参数 | 说明 | 默认值 | 单位 |
|------|------|--------|------|
| **最大光照距离** | RectLight 的衰减半径 | 2345.0 | cm |
| **最大亮度** | 光源最大亮度系数 | 100.0 | — |
| **体积散射强度** | RectLight 的体积雾散射强度 | — | — |
| **是否投射阴影** | RectLight 是否开启阴影投射 | — | — |
| **光斑亮度** | 光斑材质亮度乘数 | 100.0 | — |
| **镜片亮度** | 镜片材质亮度乘数 | 1.0 | — |

### 阴影配置

RectLight 初始化时会自动配置以下阴影参数：

| 参数 | 值 | 说明 |
|------|-----|------|
| 阴影分辨率缩放 | 0.25 | 较低的阴影分辨率以节省性能 |
| 阴影偏移 | 0.2 | 防止阴影自交叉 |
| 阴影斜坡偏移 | 0.2 | 防止阴影在斜面上出现条纹 |
| 光线追踪阴影 | 关闭 | 不使用 RTX 阴影 |

## 五、DMX 可控参数

### 5.1 亮度 (Intensity / Dimmer)

| 操作 | 说明 | 参数范围 |
|------|------|----------|
| **设置亮度** | 同时控制 RectLight 和材质的亮度 | 0.0（全灭）~ 1.0（全亮） |

亮度会同时应用到：
- RectLight 的强度（实际光照亮度）
- 镜片材质的 Brightness（镜片发光效果）
- 光斑材质的 Brightness

最终亮度计算：
```
RectLight 强度 = 光斑基础亮度 × DMX亮度 × ComponentDimmer × 频闪倍率
```

### 5.2 颜色 (Color)

| 操作 | 说明 | 参数范围 |
|------|------|----------|
| **设置颜色** | 同时设置 RectLight 颜色和材质颜色 | RGB 线性颜色 |

颜色会同时应用到：
- RectLight 的光源颜色（影响实际光照）
- 镜片材质的 LightColor
- 光斑材质的 LightColor

### 5.3 频闪 (Strobe)

RectLight 的频闪通过 **Tick 直接控制光源强度**实现（不使用光照函数）。支持与基类相同的 8 种频闪模式：

| 模式 | 名称 | 效果 |
|------|------|------|
| 0 | Closed | 完全关闭 |
| 1 | Open | 常亮（默认） |
| 2 | Linear | 渐亮渐灭 |
| 3 | Pulse | 方波闪烁 |
| 4 | Ramp Up | 渐亮 |
| 5 | Ramp Down | 渐灭 |
| 6 | Sine | 正弦脉动 |
| 7 | Random | 随机乱闪 |

### 5.4 可见性控制

| 操作 | 说明 |
|------|------|
| **设置可见性** | 同时控制 RectLight 和镜片的显示/隐藏 |

当可见性关闭时，RectLight 停止渲染和光照计算。

### 5.5 光照通道

| 参数 | 说明 | 默认值 |
|------|------|--------|
| **通道 0** | 是否启用光照通道 0 | 开启 |
| **通道 1** | 是否启用光照通道 1 | 关闭 |
| **通道 2** | 是否启用光照通道 2 | 关闭 |

### 5.6 光照传输

| 参数 | 说明 | 默认值 |
|------|------|--------|
| **影响半透明材质** | RectLight 是否照亮半透明物体 | 开启 |

## 六、不支持的功能

由于 RectLight 的特性限制，以下功能在矩形面光组件中**不可用**：

| 功能 | 原因 |
|------|------|
| **变焦 (Zoom)** | RectLight 通过遮光板控制光照范围，不支持锥角变焦 |
| **雾化 (Frost)** | RectLight 本身就是面状柔光，不需要额外雾化 |
| **光圈 (Iris)** | RectLight 没有光圈机构 |
| **Gobo 图案** | RectLight 不支持光照函数，无法投射图案 |
| **图案旋转** | 无图案可旋转 |
| **棱镜** | RectLight 不支持棱镜效果 |
| **对焦** | 无图案可对焦 |
| **颜色轮** | 通过 RGB 直接控制颜色 |

---

## 七、性能特点

| 特性 | 说明 |
|------|------|
| **渲染开销** | RectLight 的渲染开销略高于 SpotLight（面光计算更复杂） |
| **阴影开销** | 使用较低的阴影分辨率缩放（0.25），阴影开销可控 |
| **频闪开销** | 频闪模式 0/1 无额外开销；模式 2~7 需要 Tick 更新 |
| **无光照函数** | 不使用光照函数，省去了函数图集的开销 |

## 八、常见问题

### Q: 如何控制面光的照射范围？
**A**: 通过 **BarnDoorAngle**（遮光板角度）控制。角度越小照射范围越窄，角度越大照射范围越宽。

### Q: 面光的边缘太硬/太柔怎么调？
**A**: 调整 **BarnDoorLength**（遮光板长度）。长度越大边缘越硬（遮挡越明显），长度越小边缘越柔。

### Q: 为什么 RectLight 不能投射 Gobo 图案？
**A**: 这是 UE RectLight 的限制——矩形面光源不支持光照函数。如果需要投射图案，请使用 SuperSpotComponent 或 SuperBeamComponent。

### Q: 面光的亮度单位是什么？
**A**: RectLight 使用坎德拉（Candelas）作为光强度单位，并且关闭了反向平方衰减，以便在舞台设计中更直观地控制亮度。

### Q: 如何模拟 LED 面板灯的像素控制？
**A**: 面光组件不支持像素级控制。如果需要像素级控制，请使用 **SuperMatrixComponent**（矩阵灯组件）配合面光组件使用。
