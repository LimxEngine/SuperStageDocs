## 第一章 产品概述

### 1.1 什么是 SuperCAD

SuperCAD 是 SuperStage 插件中的专业施工图绘制模块，专为演艺行业设计。它能够将 Unreal Engine 编辑器中的三维灯光场景自动转换为标准的二维工程图纸（灯位图、立面图等），并提供完整的 CAD 级标注、线路绘制、统计报表和导出能力。

**核心价值**：从三维场景到工程交付图纸，一站式完成，无需切换到 AutoCAD 等外部软件。

### 1.2 SuperCAD 能做什么

| 功能 | 说明 |
|------|------|
| **自动灯位图** | 从 3D 场景自动生成俯视/正面/侧面/背面正交视图 |
| **灯具信息叠加** | 自动显示每个灯具的 DMX 地址、编号、类型、功率、重量 |
| **尺寸标注** | 测量并标注灯具之间的距离，支持单次和连续标注 |
| **文字标注** | 在图纸任意位置添加文字说明 |
| **引线标注** | 带指向箭头的文字注释，指向特定对象 |
| **区域标注** | 绘制多边形区域，自动计算面积 |
| **线路绘制** | 绘制电源线、信号线等各类线缆路径 |
| **图层管理** | 将不同元素分配到不同图层，独立控制显示和锁定 |
| **统计报表** | 灯具清单、Universe 分配表、电源负载表、挂点载荷表 |
| **多格式导出** | 导出为 DXF（AutoCAD 兼容）、PDF、PNG 格式 |
| **多页面文档** | 一个文档内包含多个页面（如俯视图、正面图分页） |

### 1.3 适用场景

- 演唱会/音乐节灯光设计出图
- 剧场/舞台灯光施工图交付
- 展览/活动灯光方案文档化
- 灯光工程招投标图纸制作
- 设备统计与物料清单生成

### 1.4 系统要求

- Unreal Engine 5.6–5.7 编辑器环境
- 已安装 SuperStage 插件
- 场景中已放置灯具 Actor（ASuperDmxActorBase 及其子类）

## 第二章 快速入门

### 2.1 打开 SuperCAD

SuperCAD 通过 Unreal Engine 编辑器菜单启动：

1. 在 UE 编辑器顶部菜单栏中，找到 **SuperStage** 菜单
2. 点击 **SuperCAD — 施工图绘制** 菜单项
3. SuperCAD 将在独立窗口中打开

> **提示**：也可以在控制台中输入 `SuperCAD.Open` 命令直接打开。

### 2.2 第一次使用

打开 SuperCAD 后，你将看到以下画面：

1. **视口区域**：显示当前场景的正交俯视图，场景中的灯具以描边/剪影方式呈现
2. **工具栏**：窗口顶部，包含所有操作工具按钮
3. **状态栏**：窗口底部，显示当前坐标、缩放比例、灯具统计等信息
4. **图层面板**：窗口左侧，管理图纸图层
5. **页面标签栏**：底部状态栏上方，管理多页面

### 2.3 五分钟快速上手

**步骤 1：查看灯位图**
- 打开 SuperCAD 后，默认显示俯视图（Plan View）
- 场景中所有灯具自动出现在视图中
- 每个灯具旁自动显示 DMX 地址（如 `U1.001`）和灯具编号（如 `#001`）

**步骤 2：视图导航**
- **滚轮**：缩放视图
- **中键拖拽**：平移视图
- 按 **F** 键：适应全部灯具到视口
- 按 **1/2/3/4** 键：切换 俯视/正面/侧面/背面 视图

**步骤 3：添加尺寸标注**
- 按 **D** 键或点击工具栏「标注」按钮
- 点击第一个灯具（起点）
- 点击第二个灯具（终点）
- 移动鼠标调整标注偏移位置，点击确认
- 距离数值自动计算并显示

**步骤 4：保存文档**
- 按 **Ctrl+S** 或点击工具栏「保存」按钮
- 首次保存时将弹出「另存为」对话框，选择保存路径和文件名
- 文档保存为 UE 资产（.uasset），存储在项目 Content 目录中

**步骤 5：导出图纸**
- 点击工具栏「导出」按钮，或在工具栏中直接点击 DXF/PDF/PNG 快速导出按钮
- 选择导出格式和保存位置
- 导出的 DXF 文件可直接在 AutoCAD 中打开

## 第三章 界面详解

SuperCAD 窗口采用经典的 CAD 软件布局，从上到下依次为：

```
┌──────────────────────────────────────────────────────────┐
│  工具栏 (ToolBar)                                    32px │
├────────┬─────────────────────────────────┬──────────────┤
│ 图层   │                                 │  属性面板    │
│ 面板   │        视口区域 (Viewport)       │ (可折叠)    │
│(可折叠)│     包含标尺 (Ruler)             │              │
│        │                                 │              │
├────────┴─────────────────────────────────┴──────────────┤
│  统计报表面板 (StatsPanel, 可展开/收起)             280px │
├──────────────────────────────────────────────────────────┤
│  页面标签栏 (PageTabBar)                            26px │
├──────────────────────────────────────────────────────────┤
│  状态栏 (StatusBar)                                 22px │
└──────────────────────────────────────────────────────────┘
```

### 3.2 工具栏

工具栏是 SuperCAD 的主要操作入口，从左到右排列以下功能区域：

#### 3.2.1 文件菜单

点击「**文件▼**」下拉菜单，包含以下操作：

| 菜单项 | 快捷键 | 功能 |
|--------|--------|------|
| 新建 | Ctrl+N | 创建新的空白图纸文档 |
| 打开 | Ctrl+O | 从 Content Browser 打开已有图纸 |
| 保存 | Ctrl+S | 保存当前文档 |
| 另存为 | — | 将文档另存为新资产 |

#### 3.2.2 绘图工具按钮组

| 按钮 | 快捷键 | 功能 |
|------|--------|------|
| **选择** | S | 选择和移动图纸元素 |
| **标注** | D | 添加尺寸标注（测量距离） |
| **文字** | T | 添加文字标注 |
| **线路** | L | 绘制线路（电源/信号/引线） |
| **区域** | — | 绘制区域标注（多边形+面积计算，仅工具栏按钮） |

> **提示**：当前激活的工具按钮会高亮显示。切换工具时，前一个工具自动退出。

#### 3.2.3 视图控制区

| 按钮/控件 | 功能 |
|----------|------|
| **视图下拉** | 切换视图方向：俯视/正面/侧面/背面 |
| **模式下拉** | 切换渲染模式：线框(Line)/着色(Shaded)/光照(Lit) |

#### 3.2.4 缩放控制区

| 按钮 | 功能 |
|------|------|
| **缩小 (−)** | 缩小视图 |
| **放大 (+)** | 放大视图 |
| **适应** | 缩放至全部灯具可见 (F) |

#### 3.2.5 辅助开关区

| 按钮 | 快捷键 | 功能 | 默认状态 |
|------|--------|------|---------|
| **网格** | G | 显示/隐藏参考网格 | 开启 |
| **捕捉** | — | 启用/禁用智能捕捉 | 开启 |
| **正交** | — | 启用/禁用正交约束 | 关闭 |
| **图框** | — | 显示/隐藏图纸图框 | 关闭 |
| **坐标轴** | — | 显示/隐藏原点坐标轴 | 开启 |

#### 3.2.6 右侧功能区

| 按钮/控件 | 功能 |
|----------|------|
| **报表** | 展开/收起统计报表面板 |
| **单位下拉** | 切换图纸单位：厘米(cm)/米(m)/毫米(mm)/英尺(ft)/英寸(in) |
| **导出** | 打开导出对话框 |
| **DXF/PDF/PNG** | 快速导出（直接弹出文件保存对话框） |
| **帮助** | 打开官网文档 (https://yunsio.com/docs) |

### 3.3 视口区域

视口是 SuperCAD 的核心工作区域，显示场景的正交投影视图。

**视口特点**：
- **正交投影**：无透视畸变，所有平行线保持平行，适合工程制图
- **标尺**：视口上方和左侧显示刻度标尺，随缩放自适应精度
- **网格**：可切换显示的参考网格，具有主/次网格线层级
- **比例尺**：视口内显示当前比例尺刻度条

**视口中显示的内容**：
- 场景中所有灯具的正交投影（线框或着色模式）
- 灯具信息标签（地址码、编号、类型等）
- 用户添加的标注（尺寸、文字、引线、区域）
- 用户绘制的线路（电源线、信号线）
- 参考网格和坐标轴

### 3.4 状态栏

状态栏位于窗口最底部，持续显示以下信息（从左到右）：

| 信息 | 说明 | 示例 |
|------|------|------|
| **文档名称** | 当前文档名，未保存时显示「未命名」，有修改时显示 * | `MyDrawing*` |
| **视图方向** | 当前视图方向 | `俯视图` |
| **视图模式** | 当前渲染模式 | `线框` |
| **当前工具** | 当前激活的工具 | `选择` |
| **鼠标坐标** | 鼠标在世界空间中的坐标（单位随设置变化） | `X: 120.5 Y: 350.2` |
| **灯具数量** | 场景中的灯具总数 | `灯具: 48` |
| **选中数量** | 当前选中的对象数量 | `选中: 3` |
| **总功率** | 所有灯具额定功率之和 | `功率: 28.5kW` |
| **总重量** | 所有灯具重量之和 | `重量: 450kg` |
| **比例/单位** | 当前图纸比例和单位 | `1:100 | m` |

### 3.5 图层面板

位于窗口左侧，可折叠。详见 [第九章 图层管理](#第九章-图层管理)。

### 3.6 属性面板

位于图层面板下方（或窗口右侧），显示当前选中对象的属性信息和项目统计数据。

属性面板显示的统计信息包括：
- 灯具总数、灯具类型数
- 总功率 (kW)、总重量 (kg)
- Universe 数量、总通道数
- 线路总长度 (m)
- 标注总数

### 3.7 页面标签栏

位于状态栏上方，显示当前文档中的所有页面标签。详见 [第十二章 页面管理](#第十二章-页面管理)。

## 第四章 文档管理

### 4.1 文档类型

SuperCAD 文档以 Unreal Engine 的 **UDataAsset**（`.uasset` 文件）形式保存在项目的 Content 目录中。这意味着：

- 文档天然支持版本控制（Git、Perforce 等）
- 文档可以在 UE Content Browser 中浏览和管理
- 双击 Content Browser 中的 CAD 文档资产可直接打开 SuperCAD

### 4.2 新建文档

**操作方式**（任选一种）：
1. 工具栏 → 文件菜单 → **新建**
2. 快捷键 **Ctrl+N**
3. 在 Content Browser 中右键 → 杂项 → **SuperCAD Document**

新建文档时：
- 如果当前文档有未保存的修改，系统将弹出提示「当前图纸已修改，是否保存？」
- 选择「是」保存后再新建，选择「否」放弃修改直接新建，选择「取消」返回编辑

新文档默认包含：
- 一个默认页面（「Page 1」，俯视图）
- 一个默认图层（「默认」图层）
- 默认比例 1:100
- 默认单位：毫米

### 4.3 打开文档

**操作方式**（任选一种）：
1. 工具栏 → 文件菜单 → **打开**
2. 快捷键 **Ctrl+O**
3. 在 Content Browser 中双击 SuperCAD Document 资产

打开操作会弹出资产选择器对话框，仅显示 SuperCAD Document 类型的资产。选择要打开的文档后确认即可。

### 4.4 保存文档

**操作方式**（任选一种）：
1. 工具栏 → 文件菜单 → **保存**
2. 快捷键 **Ctrl+S**

保存行为：
- **首次保存**：如果文档从未保存过（「未命名」），将自动跳转到「另存为」流程
- **后续保存**：直接覆盖保存到原有路径

### 4.5 另存为

工具栏 → 文件菜单 → **另存为**

弹出保存对话框，可选择新的保存路径和文件名。另存为会创建一个新的文档资产，原文档不受影响。

### 4.6 自动记忆上次文档

SuperCAD 会自动记住上次打开的文档路径。下次启动时将自动加载该文档，无需手动打开。

### 4.7 关闭保存提示

在以下情况下，如果当前文档有未保存的修改，系统将弹出保存提示：
- 关闭 SuperCAD 窗口
- 新建文档
- 打开其他文档

提示选项：
- **是**：保存当前文档后继续操作
- **否**：放弃修改，继续操作
- **取消**：取消操作，返回编辑

### 4.8 脏标记（修改指示器）

状态栏中的文档名称后会显示星号 `*` 表示文档有未保存的修改。以下操作会触发脏标记：
- 添加/删除/修改标注
- 添加/删除/修改线路
- 修改图层设置
- 添加/删除页面
- 修改文档设置（比例、纸张大小等）

## 第五章 视图控制

### 5.1 视图方向

SuperCAD 支持四个标准正交视图方向：

| 视图 | 快捷键 | 投影方向 | 测量平面 | 典型用途 |
|------|--------|---------|---------|---------|
| **俯视图** (Plan View) | 1 | 从上向下看 | XY 平面 | 灯位图（最常用） |
| **正立面** (Front Elevation) | 2 | 从前向后看 | XZ 平面 | 前立面图 |
| **侧立面** (Side Elevation) | 3 | 从侧面看 | YZ 平面 | 侧立面图 |
| **后立面** (Rear Elevation) | 4 | 从后向前看 | XZ 平面 | 后立面图 |

**切换方式**：
- 按数字键 **1/2/3/4**
- 工具栏视图下拉菜单选择

### 5.2 视图模式

SuperCAD 提供三种渲染模式：

| 模式 | 快捷键 | 说明 | 推荐场景 |
|------|--------|------|---------|
| **线框** (Line) | O 循环切换 | 仅显示物体轮廓线，最接近传统 CAD | 出图、标注 |
| **着色** (Shaded) | O 循环切换 | 显示物体的基础颜色和形状 | 设备辨识 |
| **光照** (Lit) | O 循环切换 | 完整的光照渲染效果 | 效果预览 |

**切换方式**：
- 按 **O** 键在三种模式间循环切换（Line → Shaded → Lit → Line）
- 工具栏模式下拉菜单选择

> **提示**：线框模式下，不同图层的物体可以显示不同的线框颜色，方便区分。

### 5.3 视图导航

#### 5.3.1 平移

| 操作 | 方式 |
|------|------|
| **中键拖拽** | 按住鼠标中键并拖拽，平移视图 |
| **右键拖拽** | 按住鼠标右键并拖拽，也可平移视图 |

#### 5.3.2 缩放

| 操作 | 方式 |
|------|------|
| **滚轮向上** | 放大（以鼠标位置为中心） |
| **滚轮向下** | 缩小（以鼠标位置为中心） |
| **工具栏 +/−** | 点击放大/缩小按钮 |

缩放范围：从极近距离（几厘米级别）到超远距离（数千米级别）

#### 5.3.3 适应视图

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| **适应全部** | F | 缩放并居中，使所有灯具都在视口中可见 |
| **适应选择** | Shift+F | 缩放并居中到当前选中的灯具 |

适应全部也可通过：
- 工具栏「适应」按钮
- 右键菜单 → 适应全部

### 5.4 网格显示

网格是 CAD 绘图的重要参考系统。SuperCAD 的网格具有以下特点：

- **自适应密度**：随缩放级别自动调整网格间距，始终保持清晰可读
- **主/次网格线**：主网格线（粗线）和次网格线（细线）交替显示
- **单位关联**：网格刻度随图纸单位自动适配

**控制方式**：
- 按 **G** 键切换网格显示/隐藏
- 工具栏「网格」按钮

### 5.5 坐标轴显示

原点坐标轴（红色 X 轴、绿色 Y 轴）可帮助确认空间方位。

**控制方式**：
- 工具栏「坐标轴」按钮

## 第六章 选择工具

### 6.1 概述

选择工具是 SuperCAD 的默认工具，用于选择和操作图纸中的对象（灯具、标注、线路等）。按 **S** 键或点击工具栏「选择」按钮即可切换到选择工具。

### 6.2 点选

**操作**：在视口中左键单击一个对象。

- 点击灯具：选中该灯具
- 点击标注：选中该标注（尺寸标注、文字标注、引线标注、区域标注）
- 点击线路：选中该线路
- 点击空白处：取消所有选择

### 6.3 修饰键

| 修饰键 | 效果 |
|--------|------|
| **Shift+点击** | 将对象添加到当前选择集（不取消已选） |
| **Ctrl+点击** | 切换对象的选中状态（已选则取消，未选则添加） |
| 无修饰键+点击 | 清除之前的选择，仅选中当前对象 |

### 6.4 框选（CAD 双模式）

SuperCAD 实现了专业 CAD 软件的双模式框选：

#### 6.4.1 Window 选择（从左向右拖拽）

- **操作**：从左向右拖拽鼠标
- **表现**：蓝色实线矩形框
- **规则**：仅选择**完全在框内**的对象

适用场景：精确选择某一区域内的所有对象，排除只部分进入框内的对象。

#### 6.4.2 Crossing 选择（从右向左拖拽）

- **操作**：从右向左拖拽鼠标
- **表现**：绿色虚线矩形框
- **规则**：选择**在框内或与框相交**的对象

适用场景：快速选择某一区域及其边界上的所有对象。

> **提示**：这是 AutoCAD 等专业 CAD 软件的标准行为，已有 CAD 使用经验的用户可以无缝上手。

### 6.5 全选

- 快捷键 **Ctrl+A**
- 工具栏 → 文件菜单 → 全选

全选会选中视图中的所有灯具。

### 6.6 删除选中

- 快捷键 **Delete**
- 右键菜单 → 删除

删除操作仅删除标注和线路等用户创建的元素，不会删除场景中的灯具 Actor。

### 6.7 选择反馈

选中对象后：
- 状态栏显示选中数量
- 选中的标注/线路以高亮颜色显示
- 选中的灯具同步到 UE 编辑器的选择系统（3D 视口中对应灯具也会被选中）

## 第七章 标注工具

### 7.1 尺寸标注工具

#### 7.1.1 功能说明

尺寸标注用于测量并标注两点之间的距离。距离值根据当前视图方向在对应平面上自动计算：

| 视图方向 | 测量公式 |
|---------|---------|
| 俯视图 | √((x₂−x₁)² + (y₂−y₁)²) — XY 平面距离 |
| 正立面 | √((x₂−x₁)² + (z₂−z₁)²) — XZ 平面距离 |
| 侧立面 | √((y₂−y₁)² + (z₂−z₁)²) — YZ 平面距离 |

距离值随图纸单位自动转换显示（如切换到米时，1500cm 显示为 15.00m）。

#### 7.1.2 操作步骤（三步交互）

1. **切换到标注工具**：按 **D** 键或点击工具栏「标注」
2. **点击起点**：在视口中点击第一个位置（系统会自动捕捉到最近的灯具锚点）
3. **点击终点**：在视口中点击第二个位置
4. **调整偏移**：移动鼠标调整标注线与测量线的距离，点击确认放置

标注放置后，系统自动计算并显示距离值。

#### 7.1.3 连续标注模式

连续标注是灯位图中最常用的标注方式 — 连续测量一排灯具之间的间距。

**操作**：
1. 在工具栏中选择「连续标注」模式（或在标注下拉菜单中选择）
2. 点击第一个灯具
3. 点击第二个灯具 → 自动标注两点间距
4. 继续点击下一个灯具 → 第二个点自动成为新的起点
5. 重复步骤 4 直到标注完成
6. 按 **ESC** 或双击结束

**特点**：
- 提交后终点自动接续为下一个起点
- 偏移距离在第一次设定后锁定，后续标注自动使用相同偏移
- 效率极高，适合标注一排灯具的间距

#### 7.1.4 标注自动更新

当被标注的灯具（或物体）在 3D 场景中移动时：
- 标注的端点如果绑定到了灯具锚点，会自动跟随移动
- 距离值自动重新计算并更新
- 标注线的几何形状自动重建

#### 7.1.5 取消/回退

- 按 **ESC**：回退一步（如已设置终点则回到起点状态）
- 再按 **ESC**：完全取消标注工具

### 7.2 文字标注工具

#### 7.2.1 功能说明

在图纸任意位置添加自由文字说明。

#### 7.2.2 操作步骤

1. **切换到文字工具**：按 **T** 键或点击工具栏「文字」
2. **点击放置位置**：在视口中点击要放置文字的位置
3. **输入文字**：直接从键盘输入文字内容
   - 光标闪烁表示输入模式激活
   - 支持字母、数字、标点符号
   - **Shift+Enter**：换行
   - **BackSpace**：删除最后一个字符
4. **确认**：按 **Enter** 键确认文字内容
5. 文字标注被写入文档

#### 7.2.3 取消

- 按 **ESC**：取消当前文字输入，清空已输入的内容

### 7.3 区域标注工具

#### 7.3.1 功能说明

绘制多边形区域，系统自动计算并标注区域面积。

#### 7.3.2 操作步骤

1. **切换到区域工具**：按 **A** 键或点击工具栏「区域」
2. **依次点击顶点**：在视口中点击放置多边形的各个顶点（至少 3 个）
   - 每次点击添加一个顶点
   - 实时预览多边形轮廓和当前鼠标位置形成的边
3. **闭合多边形**：双击或按 **Enter** 闭合多边形
   - 系统自动使用 Shoelace 公式计算面积
   - 面积值自动填入文字标注（单位：平方米）
4. **编辑面积文字**（可选）：闭合后可修改自动生成的面积文字
5. **确认**：按 **Enter** 确认

#### 7.3.3 显示效果

- 半透明填充色显示区域范围
- 轮廓线标示边界
- 顶点位置显示标记点
- 区域质心位置显示面积数值

#### 7.3.4 取消

- **ESC**：在文字编辑阶段取消回到绘制阶段
- 绘制阶段再按 **ESC**：完全取消

## 第八章 线路工具

### 8.1 概述

线路工具用于绘制各种电气线路路径，包括电源线、信号线和引线标注。SuperCAD 将引线标注和线缆绘制统一到一个工具中，通过切换类型来选择。

### 8.2 线路类型

| 类型 | 快捷键 | 线型 | 颜色 | 用途 |
|------|--------|------|------|------|
| **引线标注** | L | 实线+箭头 | 白色 | 带指向箭头的文字注释 |
| **三相电源** | L (循环切换) | 粗实线 | 红色 | 主配电线路 |
| **DMX 信号** | L (循环切换) | 虚线 | 蓝色 | DMX 信号链路 |

**类型切换**：当线路工具已激活时，再按 **L** 键将在各类型间循环切换。

### 8.3 引线标注操作

引线标注是带有指向箭头的文字注释，通常用于标注特定灯具或区域的详细说明。

1. **切换到线路工具**：按 **L** 键（默认为引线模式）
2. **放置引线起点**：在视口中点击引线的起始位置（通常靠近要标注的对象）
3. **放置拐点**（可选）：继续点击添加引线的转折点
4. **结束引线**：双击或按 **Enter** 结束引线绘制
5. **输入标注文字**：进入文字输入模式
   - 输入引线标注的文字内容
   - **Enter** 确认
   - **ESC** 取消文字输入

### 8.4 线缆绘制操作

电源线和信号线用于表示设备之间的电气连接关系。

1. **选择线缆类型**：通过 **L** 键循环切换到目标类型
2. **放置起点**：点击线缆的起始位置
   - 如果捕捉到灯具锚点，线缆会自动与灯具关联
3. **放置路径节点**：继续点击添加折线的转折点
4. **结束绘制**：双击或按 **Enter** 结束

**线缆与灯具关联**：
- 当线缆端点捕捉到灯具时，会自动建立关联
- 灯具在 3D 场景中移动时，关联的线缆端点会跟随移动

### 8.5 回退与取消

- **ESC（一次）**：回退最后一个路径节点
- **ESC（继续按）**：逐一回退节点，直到清空所有节点后取消工具
- 引线模式下文字输入中按 **ESC**：取消文字输入，回到折线状态

## 第九章 图层管理

### 9.1 概述

图层系统让您可以将图纸中的不同元素组织到不同的层次中，独立控制它们的显示、隐藏和锁定状态。这是专业 CAD 软件的标准功能。

### 9.2 图层面板

图层面板位于窗口左侧，显示当前文档的所有图层。每个图层条目显示：
- **图层名称**：可自定义
- **颜色**：图层的线框颜色（线框模式下生效）
- **可见性图标**（👁）：点击切换图层的显示/隐藏
- **锁定图标**（🔒）：点击切换图层的锁定/解锁

### 9.3 默认图层

新建文档时自动创建一个「默认」图层。所有未分配图层的灯具和标注默认属于此图层。

### 9.4 添加图层

在图层面板底部点击「+」按钮，输入图层名称即可添加新图层。

### 9.5 删除图层

选中图层后点击「−」按钮可删除图层。默认图层不可删除。

### 9.6 图层属性

| 属性 | 说明 |
|------|------|
| **名称** | 图层的显示名称 |
| **颜色** | 线框模式下该图层的显示颜色，点击可修改 |
| **可见性** | 控制图层是否在视口中显示 |
| **锁定** | 锁定的图层上的对象不可编辑 |

### 9.7 分配对象到图层

**方法 1：通过右键菜单**
1. 选中要分配的灯具或标注
2. 右键点击 → 在弹出的上下文菜单中选择「指定图层」
3. 从子菜单中选择目标图层

**自动分配**：使用标注/线路工具创建新元素时，元素会自动分配到当前活跃图层。

### 9.8 图层颜色与线框模式

在线框（Line）视图模式下：
- 不同图层的灯具以各自图层的颜色显示线框
- 修改图层颜色后，该图层的所有对象线框颜色立即更新
- 着色和光照模式下图层颜色不影响显示

### 9.9 图层可见性

- **隐藏图层**：该图层上的所有灯具和标注从视口中消失
- **显示图层**：恢复显示
- 隐藏的图层不影响导出（导出时所有图层均会包含在内，但遵循各自的可打印设置）

### 9.10 图层锁定

- **锁定图层**：该图层上的对象不可被选择、编辑或删除
- **解锁图层**：恢复可编辑状态
- 尝试在锁定图层上使用标注/线路工具时，操作会被忽略

底部合计行显示：总数量、总功率、总重量。

## 第十一章 导出功能

### 11.1 支持的导出格式

| 格式 | 文件扩展名 | 说明 | 适用场景 |
|------|-----------|------|---------|
| **DXF** | .dxf | AutoCAD 图形交换格式 (R12) | 与 CAD 软件交互，施工交付 |
| **PDF** | .pdf | 便携式文档格式 | 查阅、打印、分享 |
| **PNG** | .png | 便携式网络图形 | 快速分享、文档嵌入 |

### 11.2 导出操作

#### 方式一：导出对话框

1. 点击工具栏「**导出**」按钮
2. 在弹出的导出对话框中配置参数：
   - 选择导出格式（DXF / PDF / PNG）
   - 选择纸张大小
   - 选择比例
   - 选择分辨率倍数（PNG/PDF）
   - 选择是否包含网格、图框、图例
   - 选择是否批量导出
3. 点击「导出」按钮
4. 选择保存位置和文件名

#### 方式二：快速导出

工具栏右侧提供三个快速导出按钮（DXF / PDF / PNG），点击后直接弹出文件保存对话框，使用默认设置导出。

### 11.3 DXF 导出详情

SuperCAD 导出 DXF R12 格式（ASCII 文本），兼容所有 CAD 软件。

**图层映射**：

| CAD 图层 | DXF Layer Name | 内容 |
|---------|----------------|------|
| 灯具轮廓 | FIXTURES | 灯具外形线 |
| DMX 地址 | DMX_ADDRESS | 地址码文字 |
| 灯具编号 | FIXTURE_ID | 编号文字 |
| 尺寸标注 | DIMENSIONS | 尺寸线和文字 |
| 文字标注 | ANNOTATIONS | 文字和引线 |
| 电源线路 | POWER_CABLES | 电源线 |
| 信号线路 | SIGNAL_CABLES | 信号线 |
| 图框 | TITLEBLOCK | 图框线和标题栏 |

**图元映射**：
- 直线 → LINE
- 折线 / 引线 / 线路 / 区域边界 → LWPOLYLINE
- 文字 / 尺寸数值 → TEXT

> **注意**：未使用 DIMENSION、HATCH、MTEXT 等高级图元，以确保最广泛的 CAD 软件兼容性。

### 11.4 PDF 导出详情

PDF 导出通过视口截图方式生成，将当前视口画面截取并嵌入到 PDF 文件中。支持配置输出分辨率。

### 11.5 PNG 导出详情

PNG 截图导出直接捕获当前视口的渲染画面：
- 支持 1x, 2x, 4x 超采样分辨率
- 适合快速分享和文档嵌入

## 第十二章 页面管理

### 12.1 概述

SuperCAD 文档支持多页面，类似于一套施工图中的多张图纸。每个页面可以有独立的视图方向、视图模式、缩放级别和视口位置。

### 12.2 页面标签栏

位于状态栏上方，显示所有页面的标签。

- **点击标签**：切换到对应页面
- **当前页面**：高亮显示
- **「+」按钮**：添加新页面
- **「×」按钮**：删除页面（至少保留一个页面）

### 12.3 添加页面

点击页面标签栏的「+」按钮。新页面默认命名为「Page N」（N 为序号），采用俯视图方向。

### 12.4 切换页面

点击对应页面的标签即可切换。切换时：
- 自动保存当前页面的视图状态（方向、模式、位置、缩放）
- 自动加载目标页面的视图状态
- 工具栏和状态栏信息同步更新

### 12.5 删除页面

点击页面标签上的「×」按钮。如果只剩一个页面则无法删除。

### 12.6 页面独立设置

每个页面独立保存以下设置：
- 视图方向（俯视/正面/侧面/背面）
- 视图模式（线框/着色/光照）
- 视口中心位置
- 缩放级别
- 网格显示开关
- 图框显示开关

### 12.7 典型多页面用法

| 页面 | 视图方向 | 用途 |
|------|---------|------|
| Page 1 | 俯视图 | 灯位图（主图纸） |
| Page 2 | 正立面 | 前立面图 |
| Page 3 | 侧立面 | 侧立面图 |
| Page 4 | 俯视图 | 电源线路图（灯位图+电源线路） |

## 第十三章 辅助功能

### 13.1 智能捕捉系统

捕捉系统帮助您精确地放置标注端点和线路节点。

#### 捕捉类型与优先级（从高到低）

| 优先级 | 类型 | 图标 | 说明 |
|--------|------|------|------|
| 1 | **灯具锚点** | ⊙ | 灯具的几何中心，最常用 |
| 2 | **标注端点** | □ | 已有标注的起点/终点 |
| 3 | **网格交叉点** | + | 网格线的交叉位置 |

#### 捕捉行为

- 当鼠标靠近可捕捉对象时（10像素范围内），鼠标自动吸附到该点
- 视口中会显示捕捉图标指示当前捕捉到的类型
- 被捕捉的坐标用于标注/线路的精确定位

#### 捕捉开关

通过工具栏「捕捉」按钮可开启/关闭捕捉功能。关闭后鼠标位置不受约束，可自由放置。

### 13.2 正交约束

启用正交约束后，线路和标注的方向被限制为水平或垂直（0°/90°），类似 AutoCAD 的 ORTHO 模式。

**控制**：工具栏「正交」按钮

### 13.3 图纸图框

图框是工程图纸的标准边框和标题栏。

**控制**：工具栏「图框」按钮可切换图框的显示/隐藏。

图框包含：
- 外框线（粗实线）
- 标题栏（右下角，包含项目名称、设计者、日期等信息）

### 13.4 撤销与重做

SuperCAD 集成了 Unreal Engine 的事务系统，支持完整的撤销/重做：

| 操作 | 快捷键 |
|------|--------|
| 撤销 | Ctrl+Z |
| 重做 | Ctrl+Y |

以下操作均可撤销：
- 添加标注
- 删除标注
- 添加/删除线路
- 添加/删除图层
- 修改图层属性
- 添加/删除页面
- 指定图层

### 13.5 右键上下文菜单

在视口中右键点击可弹出上下文菜单。菜单内容根据当前选择状态动态变化：

#### 空白区域右键

### 13.6 单位切换

SuperCAD 支持以下图纸单位：

| 单位 | 缩写 | 说明 |

## 第十五章 常见问题与解答

### Q1：打开 SuperCAD 后视口是空白的？

**原因**：场景中没有灯具 Actor。

**解决**：
1. 确认已在 UE 关卡中放置了 SuperDMX 灯具 Actor（ASuperDmxActorBase 及其子类）
2. 按 **F** 键适应全部，让视口聚焦到灯具所在区域
3. 如果灯具存在但不可见，检查图层面板中的图层是否处于隐藏状态

### Q2：灯具旁边没有显示地址码和编号？

**原因**：
- 缩放太远时标签会自动隐藏 — 尝试放大视图
- 灯具未配置 DMX 地址或灯库 — 检查灯具 Actor 的属性设置
- 信息显示可能被关闭 — 检查文档显示设置

### Q3：标注的距离值不正确？

**原因**：标注距离是在当前视图方向对应的平面上计算的（投影距离），而非 3D 真实距离。

**解决**：
- 俯视图中的标注测量的是水平面（XY）上的距离
- 如果需要测量高度差，切换到正立面或侧立面视图

### Q4：如何修改已放置的标注？

目前版本的标注放置后不可直接编辑文字内容。如需修改：
1. 使用选择工具选中标注
2. 按 **Delete** 键删除
3. 重新使用标注工具创建

### Q5：导出的 DXF 文件在 AutoCAD 中打不开？

**解决**：
- SuperCAD 导出的是 DXF R12 格式，兼容所有 CAD 软件
- 确认文件扩展名为 `.dxf`
- 尝试使用「打开」而非「导入」来加载 DXF 文件
- 确认文件编码为 UTF-8

### Q6：关闭 SuperCAD 后数据会丢失吗？

**回答**：
- 如果已保存文档（Ctrl+S），数据不会丢失
- 关闭窗口时如果有未保存修改，系统会弹出保存提示
- 下次打开 SuperCAD 时会自动加载上次打开的文档

### Q7：如何在 Content Browser 中管理 CAD 文档？

SuperCAD 文档是标准的 UE 资产：
- 在 Content Browser 中可以看到 SuperCAD Document 类型的资产
- 双击即可打开
- 右键 → 杂项 → SuperCAD Document 可创建新文档
- 支持重命名、移动、复制等标准资产操作

### Q8：视口中出现额外的黄色/红色线框？

**原因**：UE 编辑器的 Builder Brush 或 BSP 几何体的辅助线。

**解决**：SuperCAD 已内置过滤机制自动隐藏这些辅助线。如果仍然出现，尝试右键菜单 → 刷新场景数据。

### Q9：如何刷新场景数据？

当 3D 场景发生大量变化后，可以手动强制刷新：
- 右键菜单 → **刷新场景数据**

正常情况下，场景爬虫会自动检测灯具的增删和位置变化，无需手动刷新。

### Q10：线框模式下灯具颜色如何控制？

线框模式下灯具的颜色由其所属图层的颜色决定：
1. 在图层面板中点击图层的颜色色块
2. 在颜色选择器中选择新颜色
3. 该图层上所有灯具的线框颜色立即更新
