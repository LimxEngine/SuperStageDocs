# Super Truss 桁架龙门架 — 用户手册

## 1. 概述

**Super Truss Gantry**（桁架龙门架）是 SuperStage 插件提供的程序化桁架生成工具。它可以在 Unreal Engine 编辑器中**实时**生成各种规格的桁架龙门架结构，用于舞台灯光、音响设备的悬挂搭建可视化。

将该 Actor 拖入场景后，您可以在"细节面板"（Details Panel）中调整所有参数，场景中的桁架模型会**即时更新**预览效果，无需手动触发重建。

### 适用场景

- 演出舞台灯光桁架搭建预演
- 展览/会议活动的龙门架布局规划
- 户外大型演出的桁架结构预可视化
- 载荷安全计算与工程评估

---

## 2. 如何添加到场景

1. 在 UE 编辑器的"放置 Actor"面板中搜索 **"Super Truss Gantry"**
2. 将其拖入视口场景中
3. 选中该 Actor，在右侧"细节面板"中查看和修改参数

---

## 3. 参数详解

### 3.1 桁架规格参数（TrussProfile）

这一组参数决定了桁架本身的截面形状、尺寸规格和斜撑排布方式。

#### 3.1.1 截面类型（Section Type）

控制桁架的横截面形状，直接影响桁架的承载能力和外观。

| 选项 | 说明 | 适用场景 |
|------|------|----------|
| **Box (4-Chord)** | 方形截面，4根主弦杆 + 4面斜撑。承载能力最强，是最常见的演出桁架截面 | 大型演出、重型灯光设备悬挂 |
| **Triangle (3-Chord)** | 三角截面，3根主弦杆 + 3面斜撑。比方形轻，但承载能力略低 | 中小型演出、展览 |
| **Flat / Ladder (2-Chord)** | 平面截面，仅有2根弦杆 + 单面斜撑。最轻量，外形如梯子 | 轻型安装、小型展览、背景装饰 |

> **设计提示**：如果您需要悬挂大量重型灯具（如染色灯、光束灯），建议选择 **Box** 截面。如果仅做装饰性展示或悬挂少量轻型设备，可选择 **Triangle** 或 **Flat**。

#### 3.1.2 标准规格（Truss Size）

选择桁架的标准尺寸系列。这些规格对应行业标准产品（如 Prolyte、Global Truss）的实际产品线。

| 选项 | 截面尺寸 | 自重 | 适用场景 |
|------|----------|------|----------|
| **290mm (Light)** | 29cm | 约 6 kg/m | 展览、小型演出、室内活动。对应 Prolyte H30V 系列 |
| **400mm (Medium)** | 40cm | 约 9 kg/m | 常规演出、中型活动。对应 Prolyte H40V 系列。**默认选项** |
| **520mm (Heavy)** | 52cm | 约 16 kg/m | 大型户外演出、巡回演出。对应 Prolyte S52V 系列 |

每种规格包含以下物理参数（只读，自动填充）：

- **主弦管外径** — 290/400 为 50mm，520 为 50mm
- **主弦管壁厚** — 290/400 为 2mm，520 为 3mm
- **斜撑管外径** — 290 为 20mm，400/520 为 25mm
- **斜撑管壁厚** — 均为 2mm
- **标准节间距** — 290/400 为 50cm，520 为 60cm
- **截面惯性矩** — 用于载荷计算，290=2500cm⁴, 400=5200cm⁴, 520=12800cm⁴

> **设计提示**：规格选择应根据**悬挂设备总重**和**跨度**来决定。跨度超过 12m 且悬挂载荷较重时，建议使用 520mm 重型桁架。

#### 3.1.3 斜撑模式（Brace Pattern）

控制桁架各面上斜撑管件的排布方式。

| 选项 | 说明 | 视觉效果 |
|------|------|----------|
| **Warren (V-Pattern)** | 交替 V 形排布，相邻节间的斜撑方向交替。最常见的演出桁架斜撑模式 | 简洁、对称 |
| **Cross (X-Pattern)** | 每个节间内两根斜撑交叉成 X 形，结构刚度更高 | 密集、坚固感 |

### 3.2 龙门架造型参数（GantryShape）

这一组参数控制龙门架的整体外形结构——立柱数量、横梁排布方式等。

#### 3.2.1 造型（Shape）

| 选项 | 结构描述 | 示意 |
|------|----------|------|
| **Goal Post** | 门形：2根立柱 + 1根横梁。最基本的龙门架造型 | `┌──┐` |
| **T-Shape** | T形：横梁两端超出立柱，形成悬挑。适合需要横梁覆盖更宽范围的场景 | `├──┤` 带悬挑 |
| **Portal (with Outriggers)** | 门式加固：在 Goal Post 基础上增加底部外支撑管和配重底座，提高稳定性 | `┌──┐` + 底部支撑 |
| **Double Span** | 双跨：3根立柱 + 2根横梁，覆盖更宽的跨度 | `┌──┬──┐` |

#### 3.2.2 横梁跨度（Span Length）

- **含义**：两根立柱之间的内侧距离（水平横梁的有效长度）
- **单位**：厘米（cm）
- **范围**：100 ~ 3000 cm（即 1m ~ 30m）
- **默认值**：800 cm（8m）

> **使用建议**：对于标准舞台，常用跨度在 600~1200cm 之间。超过 1500cm 的跨度需要注意载荷安全，建议查看统计面板中的挠度和最大载荷数据。

#### 3.2.3 立柱高度（Upright Height）

- **含义**：从底脚板到立柱顶端的垂直高度
- **单位**：厘米（cm）
- **范围**：100 ~ 2000 cm（即 1m ~ 20m）
- **默认值**：600 cm（6m）

#### 3.2.4 横梁超出量（Overhang）— 仅 T-Shape

- **含义**：横梁超出立柱两端的距离。仅在造型选择 **T-Shape** 时可编辑
- **单位**：厘米（cm）
- **范围**：0 ~ 500 cm
- **默认值**：100 cm（1m）

> **说明**：横梁总长度 = 跨度 + 两侧超出量×2。例如跨度 800cm + 超出量 100cm = 横梁总长 1000cm。

#### 3.2.5 第二跨跨度（Second Span）— 仅 Double Span

- **含义**：双跨造型中第二跨的跨度长度。仅在造型选择 **Double Span** 时可编辑
- **单位**：厘米（cm）
- **范围**：100 ~ 3000 cm
- **默认值**：800 cm（8m）

> **说明**：双跨总宽度 = 第一跨度 + 第二跨度。中间立柱为共用支点。

---

### 3.3 升降参数（Lift）

#### 3.3.1 升降高度（Lift Height）

- **含义**：横梁从立柱顶部**向下**降低的距离。模拟链条葫芦将横梁降低的效果
- **单位**：厘米（cm）
- **范围**：0 ~ （立柱高度 - 截面尺寸）
- **默认值**：0 cm（横梁在柱顶位置）

> **工作原理**：横梁的实际高度 = 立柱高度 - 升降高度。例如立柱 600cm，升降 100cm → 横梁位于 500cm 高度。系统会自动限制升降高度不超过安全范围。

#### 3.3.2 悬挂载荷（Suspended Load）

- **含义**：用户预计悬挂在横梁上的设备总重量
- **单位**：千克（kg）
- **范围**：0 ~ 不限
- **默认值**：0 kg

> **重要**：此参数不影响桁架的外观，但会影响统计面板中的**载荷安全计算**。系统会根据此值计算横梁挠度、最大允许载荷等安全指标，帮助您评估结构安全性。

---

### 3.4 可视化开关（Visibility）

这些开关控制龙门架各附属构件的显示/隐藏。

#### 3.4.1 显示底脚板（Show Base Plates）

- **默认**：开启 ✅
- **说明**：在每根立柱底部显示一块 60×60cm 的铝合金底脚板。关闭后底脚板不显示。

#### 3.4.2 显示外支撑（Show Outriggers）— 仅 Portal 造型

- **默认**：开启 ✅
- **说明**：在每根立柱底部向四个方向（前/后/左/右）延伸支撑管（Ø48.3mm，长度 120cm）。仅在造型选择 **Portal** 时可见并可编辑
- **作用**：外支撑用于增加龙门架的底部稳定性，防止倾倒

#### 3.4.3 显示链条葫芦（Show Chain Hoists）

- **默认**：开启 ✅
- **说明**：在横梁下方的立柱内侧位置显示链条葫芦（电动葫芦）模型。葫芦尺寸为 50×20×30cm（标准 1 吨电动葫芦）
- **数量规则**：
  - Goal Post / Portal / T-Shape：2 个（左右各一）
  - Double Span：4 个（每个跨度两侧各一）

#### 3.4.4 居中放置（Centered）

- **默认**：开启 ✅
- **说明**：
  - **开启**：Actor 的原点位于横梁正中心位置，方便对齐和放置
  - **关闭**：Actor 的原点位于左侧立柱底部，龙门架向右侧延伸

### 3.5 材质（Materials）

为龙门架的不同构件分别指定材质。如果不指定（留空），将使用引擎默认材质。

| 参数 | 影响范围 |
|------|----------|
| **Chord Material** | 主弦杆（粗管件） |
| **Brace Material** | 斜撑管、水平撑管、外支撑管 |
| **Base Plate Material** | 底脚板 |
| **Chain Hoist Material** | 链条葫芦 |

> **技巧**：建议使用金属质感的材质（如铝合金银色、黑色烤漆等）以获得最佳视觉效果。

---

### 3.6 统计信息（Statistics）— 只读

统计面板显示当前配置下龙门架的各项工程数据，所有数据**自动计算**，不可手动编辑。

#### 3.6.1 构件数量统计

| 字段 | 说明 |
|------|------|
| **Chords** | 主弦杆数量 |
| **HorizontalBraces** | 水平撑数量（每个节点处连接相邻弦杆的短管件） |
| **DiagonalBraces** | 斜撑数量 |
| **BasePlates** | 底脚板数量 |
| **Outriggers** | 外支撑数量（仅 Portal 造型） |
| **ChainHoists** | 链条葫芦数量 |
| **TotalInstances** | 所有构件的总数量 |

#### 3.6.2 重量与载荷统计

| 字段 | 说明 |
|------|------|
| **SelfWeight (kg)** | 桁架自重，包括管件、底脚板、链条葫芦等所有构件的总重量 |
| **MaxPointLoad (kg)** | 最大跨中集中载荷。在横梁正中间位置可以安全悬挂的最大重量（基于 DIN 4113 标准，许用挠度 L/300，安全系数 1.5） |
| **MaxDistributedLoad (kg/m)** | 最大均布载荷。沿横梁全长均匀分布时可承受的最大线载荷 |
| **MaxDeflection (cm)** | 当前悬挂载荷下横梁跨中的最大挠度（下垂量）。如果此值接近 跨度/300，说明载荷已接近安全极限 |
| **SuspendedLoad (kg)** | 您设定的悬挂载荷值（对应升降参数中的"悬挂载荷"） |
| **ReactionPerUpright (kg)** | 每根立柱底部的反力（地面需要承受的重量），= (自重 + 悬挂载荷) ÷ 立柱数 |

#### 3.6.3 当前规格参数

显示当前选择的桁架规格（如 S400）的详细物理参数，包括截面尺寸、管径、壁厚、节间距、自重、惯性矩等。这些数据来源于行业标准产品手册（Prolyte / Global Truss），供参考验证。

## 4. 安全计算说明

Super Truss 内置了基于 **DIN 4113**（德国铝合金结构设计标准）的载荷安全计算模型：

### 计算原理

- **结构模型**：简支梁（横梁两端支撑在立柱上）
- **材料**：铝合金 6082-T6（弹性模量 70 GPa，屈服强度 260 MPa）
- **安全系数**：可变荷载 1.5，永久荷载 1.35
- **许用挠度**：跨度/300（演出行业惯例）

### 如何判断安全性

1. 查看 **MaxPointLoad**：您的悬挂载荷应**远小于**此值
2. 查看 **MaxDeflection**：挠度应小于 跨度/300
3. 查看 **ReactionPerUpright**：确保地面/支撑点能承受此重量

> ⚠️ **免责声明**：此计算仅供预可视化参考，不可替代正式的结构工程计算。实际施工必须由有资质的结构工程师进行专业计算和审核。

## 6. 注意事项

1. **参数修改即时生效** — 所有参数修改后桁架模型会立即重建，无需额外操作
2. **仅当参数实际变化时重建** — 单纯拖动或旋转 Actor 不会触发重建，性能友好
3. **升降高度自动限制** — 系统会确保横梁不低于截面尺寸的高度，防止穿模
4. **条件编辑** — 部分参数只在特定造型下可编辑（如"超出量"仅 T-Shape，"外支撑"仅 Portal）
