# Super Projector 投影仪 — 用户手册

## 1. 概述

**Super Projector**（投影仪 / Mapping 投影）是 SuperStage 插件提供的投影映射模拟工具。它利用 Unreal Engine 的**光照函数**（Light Function）技术，通过三个独立的聚光灯组件（红/绿/蓝通道）将用户指定的贴图投射到场景中的物体表面，实现舞台投影映射（Projection Mapping）的预可视化效果。

### 工作原理

Super Projector 模拟了真实投影仪的工作方式：

1. 将投影贴图分解为 **R（红）、G（绿）、B（蓝）** 三个颜色通道
2. 每个通道通过一个独立的**聚光灯 + 光照函数材质**投射
3. 三个颜色通道在物体表面**叠加混合**，还原出完整的彩色投影画面
4. 支持**梯形校正**，模拟真实投影仪的四角调整功能

### 适用场景

- 舞台投影映射（Projection Mapping）效果预演
- 建筑外墙投影方案设计
- 演出灯光投影贴图预可视化
- 投影安装位置和角度规划
- 梯形失真校正预览

---

## 2. 如何添加到场景

1. 在 UE 编辑器的"放置 Actor"面板中搜索 **"Super Projector"**
2. 将其拖入视口场景中
3. 选中该 Actor，在"细节面板"中设置投影贴图和各项参数
4. 将投影仪对准目标物体表面

> **朝向说明**：Super Projector 的默认投射方向为 Actor 的**负 X 轴方向**（即 Actor 正面朝向的反方向）。调整 Actor 的旋转角度来对准投影目标。

## 3. 参数详解

### 3.1 投影内容（Mapping Content）

#### 3.1.1 映射贴图（Mapping Texture）

- **含义**：投影仪投射的**图像内容**。可以是任何 UE 项目中的 2D 纹理资源
- **类型**：UTexture（纹理资源引用）
- **默认值**：无（空）

> **设置方法**：
> 1. 将图片素材（PNG/JPG/TGA 等）导入 UE 项目
> 2. 在细节面板的 Mapping Texture 属性中，点击下拉箭头选择已导入的纹理
> 3. 投影仪会立即将该图像投射到场景中

> **支持的纹理格式**：支持 UE 项目中的任何标准纹理资源。建议使用 **正方形或 16:9 比例**的纹理以获得最佳效果。

#### 3.1.2 映射缩放（Mapping Scale）

- **含义**：控制投影画面的 **Light Function Scale**（光照函数缩放），值越大投影画面越小（单位为渲染缩放值，通常对应分辨率）
- **类型**：二维向量（X, Y）
- **范围**：128 ~ 4096
- **默认值**：(1920, 1080)

| 参数 | 说明 |
|------|------|
| **X** | 水平方向缩放。默认 1920（对应 1080p 宽度） |
| **Y** | 垂直方向缩放。默认 1080（对应 1080p 高度） |

> **工作原理**：Mapping Scale 实际控制的是聚光灯的 **Light Function Scale** 属性。系统还会自动计算对角线长度（√(X² + Y²)）作为 Z 方向的缩放值，确保投影比例正确。

> **技巧**：如果投影画面太大或太小，调整 Mapping Scale 的 X 和 Y 值。通常保持 X 和 Y 的比例与贴图的宽高比一致可获得最佳效果。

---

### 3.2 光照参数（Light Settings）

#### 3.2.1 亮度（Dimmer）

- **含义**：投影仪的**光照强度**，直接设置聚光灯的 Intensity 值
- **单位**：坎德拉（Candelas）
- **范围**：0 ~ 不限
- **默认值**：100

> **说明**：默认值 100 为较低亮度，适合室内调试。根据场景需要调高。
>
> **建议值**：
> - 室内小型投影：5,000 ~ 20,000 cd
> - 舞台投影：50,000 ~ 200,000 cd
> - 户外大型投影：200,000+ cd

#### 3.2.2 最大投射距离（Max Light Distance）

- **含义**：投影光线可以到达的**最远距离**。超出此距离的表面不会接收到投影
- **单位**：厘米（cm）
- **范围**：100 ~ 不限
- **默认值**：3000 cm（30m）

> **说明**：对应聚光灯的 Attenuation Radius（衰减半径）。将此值设为略大于投影仪到目标表面距离的值即可。设得太大会增加不必要的光照计算。

#### 3.2.3 投射角度（Zoom）

- **含义**：投影锥体的**外角**（半角），控制投影画面的张角大小
- **单位**：度（°）
- **范围**：10 ~ 100°
- **默认值**：30°

> **工作原理**：Zoom 对应聚光灯的 Outer Cone Angle。角度越大，投影画面在相同距离上覆盖的面积越大（但亮度会相应降低）。
>
> **真实投影仪对照**：
> - 短焦投影仪：约 50~70°
> - 标准投影仪：约 25~40°
> - 长焦投影仪：约 10~20°

#### 3.2.4 边缘柔化系数（Dilution Factor）

- **含义**：控制投影画面**边缘到中心的过渡柔和程度**
- **范围**：0.0 ~ 1.0
- **默认值**：0.0

| 值 | 效果 |
|----|------|
| **0.0** | 投影边缘锐利，无渐变（Inner Cone = Outer Cone） |
| **0.5** | 边缘有中等程度的柔和过渡 |
| **1.0** | 从中心到边缘完全渐变（Inner Cone = 0°） |

> **工作原理**：系统将 Inner Cone Angle 设为 Lerp(Zoom, 0, DilutionFactor)。DilutionFactor = 0 时，内外锥角相同（硬边缘）；DilutionFactor = 1 时，内锥角为 0°（最大柔化）。

### 3.3 梯形校正（Mapping Deformation）

梯形校正用于**补偿投影仪与投射面之间角度偏差**导致的画面变形。通过调整四个角点的位置，将变形的梯形画面校正为矩形。

每个角点有两个调整轴（X 和 Y），共 **8 个可调参数**。

#### 3.3.1 左上角（Upper Left Corner）

- **X 分量**：控制左上角在**垂直方向**上的偏移
  - 0.0 = 角点在原始位置（最上方）
  - 1.0 = 角点移动到画面中心高度
- **Y 分量**：控制左上角在**水平方向**上的偏移
  - 0.0 = 角点在原始位置（最左方）
  - 1.0 = 角点移动到画面中心宽度

#### 3.3.2 左下角（Lower Left Corner）

- **X 分量**：控制左下角的垂直偏移（0=最下方，1=中心）
- **Y 分量**：控制左下角的水平偏移（0=最左方，1=中心）

#### 3.3.3 右上角（Upper Right Corner）

- **X 分量**：控制右上角的垂直偏移（0=最上方，1=中心）
- **Y 分量**：控制右上角的水平偏移（0=最右方，1=中心）

#### 3.3.4 右下角（Lower Right Corner）

- **X 分量**：控制右下角的垂直偏移（0=最下方，1=中心）
- **Y 分量**：控制右下角的水平偏移（0=最右方，1=中心）

> **使用方法**：
> 1. 先将投影仪对准目标面，不做任何校正（所有角点值保持 0.0）
> 2. 如果画面呈梯形变形（例如上宽下窄），调整对应角点将其"拉回"矩形
> 3. 通常从一个角开始微调，逐步校正
> 4. 所有值保持 0.0 = 无校正（原始投影）

> **常见校正场景**：
> - 投影仪**仰角投射**（从下往上投）：需要收缩上方两个角点的 Y 值
> - 投影仪**侧向偏移**：需要调整同一侧两个角点的 X 值
> - **枕形/桶形失真**：可以通过四个角点综合调整来近似校正

## 4. 使用流程

### 4.1 基本设置步骤

1. **放置投影仪** — 将 Super Projector 拖入场景
2. **调整位置和朝向** — 将投影仪移动到合适位置，旋转使其对准投影目标面
3. **设置投影贴图** — 在 Mapping Texture 中选择要投射的图像
4. **开启亮度** — 将 Dimmer 从 0 调高（建议从 50000 开始测试）
5. **调整投射距离** — 设置 Max Light Distance 使其覆盖到目标面
6. **调整投射角度** — 通过 Zoom 控制投影画面大小
7. **微调校正** — 如有需要，使用梯形校正修正画面变形

### 4.2 多台投影仪拼接

如果需要用多台投影仪覆盖大面积区域：

1. 放置多个 Super Projector Actor
2. 调整各自的位置和角度使投影区域相邻
3. 适当增加 Dilution Factor（边缘柔化），使重叠区域过渡自然
4. 分别为每台投影仪设置对应区域的贴图

## 5. 常见用法示例

### 示例 1：舞台正面投影

- 位置：观众区上方，正对舞台背景幕
- Dimmer：100,000 cd
- Max Light Distance：2000cm (20m)
- Zoom：30°
- Dilution Factor：0.1
- 梯形校正：所有值 0（正对投射，无变形）

### 示例 2：建筑外墙投影

- 位置：建筑正前方地面
- Dimmer：500,000 cd
- Max Light Distance：5000cm (50m)
- Zoom：45°
- Dilution Factor：0.0
- 梯形校正：因仰角投射，需调整上方两角点

### 示例 3：小型装饰投影

- 位置：近距离投射到布景上
- Dimmer：10,000 cd
- Max Light Distance：500cm (5m)
- Zoom：60°
- Dilution Factor：0.3
- Mapping Scale：(0.5, 0.5)

## 6. 注意事项

1. **默认亮度为 0** — 新放置的投影仪不会发光，必须手动调高 Dimmer 值
2. **三通道叠加** — 投影使用 RGB 三个独立聚光灯，理论上光照计算量是普通聚光灯的 3 倍
3. **需要光照函数材质** — 投影依赖插件内置的光照函数材质。如果材质资源缺失，投影将不工作
4. **投射方向** — 投影方向为 Actor 的负 X 轴方向（朝向 Actor 的"前方"）
5. **阴影** — 默认开启阴影投射，场景中的物体会遮挡投影光线
6. **权限验证** — 投影功能需要有效的插件授权。未授权时，Dimmer 调节和梯形校正将不生效
7. **梯形校正范围** — 每个角点的调整范围为 0~1，其中 1 表示将角点拉到画面中心。通常微调值在 0~0.3 之间即可满足大多数校正需求
