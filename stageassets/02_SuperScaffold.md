# Super Scaffold 直线脚手架 — 用户手册

## 1. 概述

**Super Scaffold**（直线脚手架）是 SuperStage 插件提供的程序化脚手架生成工具。它可以在 Unreal Engine 编辑器中**实时**生成矩形网格结构的脚手架系统，包括立杆、横杆、斜撑、底座板、顶部平台和配重块等完整构件。

将该 Actor 拖入场景后，在"细节面板"中调整参数即可**即时预览**不同尺寸和配置的脚手架结构。内置 **EN 12811**（欧洲脚手架标准）载荷计算系统，可自动计算自重、承载能力和所需配重。

### 适用场景

- 演出舞台升降台/观众台搭建预演
- 户外活动临时看台规划
- 灯光/音响平台搭建可视化
- 脚手架安全评估与配重计算

---

## 2. 如何添加到场景

1. 在 UE 编辑器的"放置 Actor"面板中搜索 **"Super Scaffold"**
2. 将其拖入视口场景中
3. 选中该 Actor，在右侧"细节面板"中查看和修改参数

---

## 3. 参数详解

### 3.1 结构尺寸参数（Structure）

这一组参数定义脚手架的整体框架布局。

#### 3.1.1 X 方向跨数（Span Count X）

- **含义**：脚手架在 X 轴方向（通常为舞台正面方向）的**跨数**
- **范围**：1 ~ 50
- **默认值**：3

> **说明**："跨数"是指相邻两根立杆之间的间隔数量。如果设置为 3，则 X 方向会有 4 根立杆（跨数 + 1 = 节点数）。

#### 3.1.2 Y 方向跨数（Span Count Y）

- **含义**：脚手架在 Y 轴方向（通常为舞台深度方向）的**跨数**
- **范围**：1 ~ 50
- **默认值**：2

#### 3.1.3 层数（Layer Count）

- **含义**：脚手架的**层数**（垂直方向上横杆的间隔数）
- **范围**：1 ~ 20
- **默认值**：3

> **说明**：每"层"由一组水平横杆构成。如果设置为 3 层，则从底部基础高度开始，向上有 4 个横杆层面（层数 + 1 = 横杆层面数）。

#### 3.1.4 X 方向跨距（Bay Size X）

- **含义**：X 方向上相邻两根立杆之间的**距离**
- **单位**：厘米（cm）
- **范围**：50 ~ 500 cm
- **默认值**：207 cm（Layher Allround 标准跨距）

> **设计提示**：行业标准脚手架常用跨距为 73/109/140/157/207/257/307cm（Layher Allround 系列）。

#### 3.1.5 Y 方向跨距（Bay Size Y）

- **含义**：Y 方向上相邻两根立杆之间的**距离**
- **单位**：厘米（cm）
- **范围**：50 ~ 500 cm
- **默认值**：207 cm（Layher Allround 标准跨距）

#### 3.1.6 层高（Layer Height）

- **含义**：每层横杆之间的**垂直间距**
- **单位**：厘米（cm）
- **范围**：50 ~ 400 cm
- **默认值**：200 cm（2m）

> **说明**：脚手架总高度 = 基础高度 + 层数 × 层高。例如基础高度 30cm，3 层，每层 200cm → 总高度 = 30 + 600 = 630cm。

---

### 3.2 管件尺寸参数（Tube Dimensions）

#### 3.2.1 主管外径（Main Tube Diameter）

- **含义**：立杆和横杆使用的钢管**外径**
- **单位**：厘米（cm）
- **范围**：3 ~ 10 cm
- **默认值**：4.83 cm（即 48.3mm，EN 12811 标准管径）

> **行业参考**：48.3mm（Ø48.3）是全球脚手架行业最常用的管径标准（对应 BS 1139 / EN 12811 标准）。

#### 3.2.2 斜撑管外径（Brace Tube Diameter）

- **含义**：对角斜撑管件的**外径**
- **单位**：厘米（cm）
- **范围**：1 ~ 20 cm
- **默认值**：4.83 cm（即 48.3mm，Layher Allround 标准）

> **说明**：默认斜撑管径与主管相同（Layher Allround 标准），可根据需要调整。

#### 3.2.3 底座板直径（Base Plate Size）

- **含义**：每根立杆底部圆形底座板的**直径**
- **单位**：厘米（cm）
- **范围**：10 ~ 40 cm
- **默认值**：15 cm

#### 3.2.4 基础高度（Base Height）

- **含义**：从地面到第一层横杆的**垂直距离**。模拟脚手架底座高度或调节腿高度
- **单位**：厘米（cm）
- **范围**：0 ~ 200 cm
- **默认值**：30 cm

> **说明**：这个值相当于脚手架可调底座（Screw Jack）的高度。实际施工中通常设置 20~40cm 以适应地面不平。

#### 3.2.5 甲板厚度（Deck Thickness）

- **含义**：顶层平台板的**厚度**
- **单位**：厘米（cm）
- **范围**：1 ~ 10 cm
- **默认值**：3.2 cm（32mm 标准合板）

---

### 3.3 可视化开关（Visibility）

#### 3.3.1 斜撑模式（Diagonal Mode）

控制脚手架上斜撑管件的显示方式。

| 选项 | 说明 |
|------|------|
| **None** | 不显示任何斜撑管件 |
| **Exterior Only** | 仅在脚手架的**外围四面**显示斜撑（X 形交叉斜撑），内部跨间不设斜撑 |
| **All Faces** | 在**所有面**上都显示斜撑，包括内部每个跨间的每个面 |

> **设计提示**：
> - 仅作视觉展示用途时，选择 **Exterior Only** 即可，效果最接近实际脚手架外观
> - 选择 **All Faces** 时构件数量会显著增加，适合展示高刚度结构
> - 选择 **None** 可获得最简洁的外观

#### 3.3.2 显示底座板（Show Base Plates）

- **默认**：开启 ✅
- **说明**：在每根立杆底部显示圆形底座板。底座板为 1cm 厚的扁平圆柱体，直径由"底座板直径"参数控制

#### 3.3.3 显示顶部平台（Show Top Deck）

- **默认**：关闭 ❌
- **说明**：在脚手架最顶层横杆上方铺设平台板。每个跨间（Bay）铺设一块独立的面板，面板尺寸自动匹配跨距

#### 3.3.4 显示配重块（Show Counterweights）

- **默认**：开启 ✅
- **说明**：在每个底座板旁边堆叠混凝土配重块。配重块按照 EN 12811 标准计算所需数量，自动在 X 方向偏移放置
- **配重块规格**：标准混凝土块 40×20×12cm，每块 25kg
- **堆叠方式**：垂直堆叠，每块高度 12cm

> **重要**：配重块的数量由系统自动计算（基于 EN 12811 安全系数 1.5），不可手动设置。如果计算结果为 0 块，则即使开启此开关也不会显示任何配重块。

#### 3.3.5 居中放置（Centered）

- **默认**：开启 ✅
- **说明**：
  - **开启**：Actor 原点位于脚手架 XY 平面的几何中心，Z 方向仍从地面起
  - **关闭**：Actor 原点位于左前角底部（X=0, Y=0, Z=0），脚手架向正 X 和正 Y 方向延伸

---

### 3.4 载荷参数（Load）

#### 3.4.1 荷载等级（Load Class）

根据 **EN 12811-1** 标准，选择脚手架的**设计荷载等级**。荷载等级决定了平台面板可以承受的最大均布面荷载。

| 选项 | 均布面荷载 | 适用场景 |
|------|-----------|----------|
| **Class 1** | 0.75 kN/m² (≈76 kg/m²) | 仅检修用途 |
| **Class 2** | 1.50 kN/m² (≈153 kg/m²) | 轻型工作平台 |
| **Class 3** | 2.00 kN/m² (≈204 kg/m²) | 标准工作平台。**默认选项** |
| **Class 4** | 3.00 kN/m² (≈306 kg/m²) | 重型存放平台 |
| **Class 5** | 4.50 kN/m² (≈459 kg/m²) | 石砌工作平台 |
| **Class 6** | 6.00 kN/m² (≈612 kg/m²) | 重型石砌平台 |

> **演出行业建议**：演出舞台平台通常选择 **Class 3** 或 **Class 4**。观众看台建议至少 **Class 4**。

#### 3.4.2 额外载荷（Additional Load）

- **含义**：除脚手架自重外，额外施加在结构上的**附加载荷**（如设备、人员等）
- **单位**：千克（kg）
- **范围**：0 ~ 不限
- **默认值**：0 kg

> **说明**：此值参与配重计算。配重需求 = (安全系数 - 1) × (自重 + 额外载荷)。增加额外载荷会直接增加所需配重量。

---

### 3.5 材质（Materials）

为脚手架的不同构件分别指定材质。留空则使用引擎默认材质。

| 参数 | 影响范围 |
|------|----------|
| **Tube Material** | 所有钢管件（立杆、横杆、斜撑、底座板柱） |
| **Deck Material** | 顶部平台面板 |
| **Base Plate Material** | 底座板 |
| **Counterweight Material** | 配重块 |

> **技巧**：建议钢管使用灰色/银色金属材质，平台板使用木纹或防滑板材质，配重块使用混凝土材质。

### 3.6 统计信息（Statistics）— 只读

#### 3.6.1 构件数量统计

| 字段 | 说明 |
|------|------|
| **Verticals** | 立杆数量 = (X跨数+1) × (Y跨数+1) |
| **HorizontalsPrimary** | X 方向横杆数量 = X跨数 × (Y跨数+1) × (层数+1) |
| **HorizontalsSecondary** | Y 方向横杆数量 = (X跨数+1) × Y跨数 × (层数+1) |
| **Diagonals** | 斜撑数量（取决于斜撑模式） |
| **BasePlates** | 底座板数量 = 立杆数量 |
| **DeckPanels** | 平台面板数量 = X跨数 × Y跨数 |
| **CounterweightBlocks** | 配重块总数 |
| **TotalInstances** | 所有构件的总数量 |

#### 3.6.2 重量与载荷统计

| 字段 | 说明 |
|------|------|
| **SelfWeight (kg)** | 脚手架总自重，包括所有钢管件、底座板和平台板的重量 |
| **MaxLoadCapacity (kg)** | 根据荷载等级和平台面积计算的最大承载能力 |
| **RequiredCounterweight (kg)** | EN 12811 标准要求的总配重量（安全系数 1.5） |
| **CounterweightPerBase (kg)** | 每个底座分配的配重量 = 总配重 ÷ 立杆数 |
| **BlocksPerBase** | 每个底座需要堆叠的配重块数量（向上取整，每块 25kg） |

## 4. 重量计算说明

### 4.1 自重计算

系统基于 **EN 12811** 标准参数计算脚手架自重：

- **钢管材料**：标准建筑用钢，密度 7850 kg/m³（即 0.00785 g/mm³）
- **壁厚**：3.2mm（EN 10210 标准热轧空心截面）
- **管件截面积**：π × (外径/2² - 内径/2²)，内径 = 外径 - 2×壁厚
- **每根管件重量**：长度 × 截面积 × 钢密度
- **底座板**：每块 2.5 kg（Layher Allround 标准底座）
- **平台板**：15 kg/m²（铝合金标准平台板）

### 4.2 配重计算

配重计算遵循 EN 12811 标准：

```
所需配重 = (安全系数 - 1) × (自重 + 额外载荷)
         = (1.5 - 1) × (自重 + 额外载荷)
         = 0.5 × (自重 + 额外载荷)
```

配重均匀分配到每个底座，然后按每块 25kg 向上取整确定配重块数量。

## 5. 常见用法示例

### 示例 1：标准舞台平台 (6m × 4m × 2m)

- X 方向跨数：3
- Y 方向跨数：2
- 层数：1
- X/Y 跨距：200cm
- 层高：200cm
- 荷载等级：Class 3

- X 方向跨数：4
- Y 方向跨数：3
- 层数：2
- X 跨距：250cm
- Y 跨距：200cm
- 层高：200cm
- 荷载等级：Class 4
- 显示配重块：开启

- X 方向跨数：1
- Y 方向跨数：1
- 层数：4
- X/Y 跨距：200cm
- 层高：200cm
- 斜撑模式：All Faces

## 6. 注意事项

1. **参数即时生效** — 所有参数修改后脚手架模型会立即重建预览
2. **大型结构注意性能** — 跨数和层数过大时（如超过 20×20×10），构件总数可能达到数万，建议关注编辑器性能
3. **配重块自动计算** — 无法手动指定配重块数量，完全由 EN 12811 标准公式决定
4. **钢管壁厚固定** — 壁厚统一按 3.2mm 计算，不可手动修改（EN 10210 标准）
5. **斜撑为 X 形交叉** — 每个面上的斜撑总是两根管件交叉排列
6. **居中模式影响坐标** — 开启/关闭居中模式会改变脚手架相对于 Actor 原点的位置
